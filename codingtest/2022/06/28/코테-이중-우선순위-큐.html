코딩테스트 | 이중우선순위큐

힙 (우선순위 큐)

Jun 28, 2022 About 5 mins

#CodingTest#C++

문제

이중 우선순위 큐는 다음 연산을 할 수 있는 자료구조를 말합니다.

명령어 수신 탑(높이)
I 숫자 큐에 주어진 숫자를 삽입합니다.
D 1 큐에서 최댓값을 삭제합니다.
D -1 큐에서 최솟값을 삭제합니다.

이중 우선순위 큐가 할 연산 operations가 매개변수로 주어질 때, 모든 연산을 처리한 후 큐가 비어있으면 [0,0] 비어있지 않으면 [최댓값, 최솟값]을 return 하도록 solution 함수를 구현해주세요.

조건

  • operations는 길이가 1 이상 1,000,000 이하인 문자열 배열입니다.
  • operations의 원소는 큐가 수행할 연산을 나타냅니다.
  • 원소는 “명령어 데이터” 형식으로 주어집니다.
    • 최댓값/최솟값을 삭제하는 연산에서 최댓값/최솟값이 둘 이상인 경우, 하나만 삭제합니다.
  • 빈 큐에 데이터를 삭제하라는 연산이 주어질 경우, 해당 연산은 무시합니다.

입출력 예시

operations return
{“I 16”,”D 1”} [0,0]
{“I 7”,”I 5”,”I -5”,”D -1”} [7,5]

풀이방식

나의 풀이방식

2개의 우선순위 큐를 지정하고, 각각 내림차순, 오름차순으로 정렬한다. Input시 2개의 힙에 모두 삽입하며, cnt를 통해서 개수를 카운트한다. 만약 cnt가 0이 된다면, 힙의 내용을 모두 pop해버린다. cnt가 0개 이상일때 2가지 힙에 관하여 Max, Min값을 도출해낸다.

  #include <queue>

  vector<int> solution(vector<string> operations) {
    vector<int> answer(2,0);
    priority_queue<int, vector<int>> MaxQ;
    priority_queue<int, vector<int>, greater<int>> MinQ;
    int cnt = 0;

    for (string op : operations) {
      if (!cnt) {
        while (!MaxQ.empty()) MaxQ.pop();
        while (!MinQ.empty()) MinQ.pop();
      }

      if (op[0] == 'I') {
        int tmp = stoi(op.substr(2));
        MaxQ.push(tmp);
        MinQ.push(tmp);
        cnt++;
      }
      else {
        if(!cnt) continue;
        
        if (op[2] == '1') MaxQ.pop();
        else MinQ.pop();

        cnt--;
      }
    }

    if (cnt) {
      answer[0] = MaxQ.top();
      answer[1] = MinQ.top();
    }
    return answer;
  }

다른 이의 풀이방식

이는 multiset을 활용한 방식으로 힙을 사용한 방식이 아니다. 먼저 multiset이란 set과 비슷하지만 key값이 중복된다는 특징을 지니고 있다. set을 사용해도 지장이 없으며, queue는 iterartor를 사용한 탐색이 불가능하지만, set은 노드 탐색이 가능하기 때문에 어느 지점이든 바로 접근이 가능하다.

  #include <set>

  vector<int> solution(vector<string> arguments) {
    vector<int> answer;
    multiset<int> que;
    multiset<int>::iterator iter;
    string sub;

    for(auto s : arguments) {
      sub = s.substr(0, 2);

      //삽입
      if(sub=="I ") que.insert(stoi(s.substr(2,s.length()-2))); 
      else if(s.substr(2,1) == "1" && que.size() > 0) { que.erase(--que.end()); } //최대값 삭제
      else if(que.size() > 0) { que.erase(que.begin()); } //최솟값 삭제
    }

    if(que.size() == 0) { 
      answer.push_back(0); 
      answer.push_back(0);
    }
    else { 
      iter = --que.end(); 
      answer.push_back(*iter); 

      iter = que.begin(); 
      answer.push_back(*iter);
    }

    return answer;
  }
