코딩테스트 | N-Queen

DFS

Jul 05, 2022 About 3 mins

#CodingTest#C++

문제

가로, 세로 길이가 n인 정사각형으로된 체스판이 있습니다. 체스판 위의 n개의 퀸이 서로를 공격할 수 없도록 배치하고 싶습니다.

예를 들어서 n이 4인경우 다음과 같이 퀸을 배치하면 n개의 퀸은 서로를 한번에 공격 할 수 없습니다.

체스판의 가로 세로의 세로의 길이 n이 매개변수로 주어질 때, n개의 퀸이 조건에 만족 하도록 배치할 수 있는 방법의 수를 return하는 solution함수를 완성해주세요.

조건

  • 퀸(Queen)은 가로, 세로, 대각선으로 이동할 수 있습니다.
  • n은 12이하의 자연수 입니다.

입출력 예시

n return
4 2

풀이방식

먼저 DFS를 사용하였고, Queen의 특징을 정리하자면 아래와 같다.

  1. 같은 행과 열에 위치할 수 없다.
  2. 대각선으로 이동이 가능하기 때문에 다른 Queen의 대각선에 위치할 수 없다.
2번 같은 경우의 조건은 현재 행에서부터의 상단 행들에 대해서 기울기 공식으로 판단할 수 있는데 이 공식은 __” 상단 Queen X좌표 - 현재 X좌표 == 상단 Queen Y좌표 - 현재 Y좌표 “이다.__ </span> 만약 이를 만족한다면 상단 Queen과 현재 선택하려는 위치가 같은 대각선 상에 위치함을 알 수 있다.

즉, 상단의 행에서 아래로 점점내려오는 형식의 DFS탐색이며, 행과 열은 “ch[행] = 열”과 같이 표현한다. 행에 대해서 탐색하지 않았을때는 -1을 저장한다.

  vector<int> ch(13,-1);
  int answer = 0;

  void DFS(int cnt, int Q) {
    if (cnt == Q) answer++;
    else {
      //상단 Q의 자리를 선정할때만 사용
      if (cnt == 0) {
        for (int i = 0; i < Q; i++) {
          if (ch[cnt] == -1) {
            ch[cnt] = i;
            DFS(cnt + 1, Q);
            ch[cnt] = -1;
          }
        }
      }
      else {
        // i : 선택하려는 열(Y축)
        for (int i = 0; i < Q; i++) {    
          bool isTrue = true;
          
          /* 같은 행렬이거나 기울기에 위치하는지 확인
          * j : 이전 Queen들이며, ch[j] : 그 Queen이 위치한 열
          */
          for (int j = 0; j < cnt; j++){
            if(ch[j] == i || abs(j - cnt) == abs(ch[j] - i)){
              isTrue = false;
              break;
            }
          }
          
          // 만약 위치하지 않다면 재귀.. 
          if (isTrue) {
            ch[cnt] = i;
            DFS(cnt + 1, Q);
            ch[cnt] = -1;
          }
        }
      }
    }
  }
  int solution(int n) {
      DFS(0,n);
      return answer;
  }

개선점

  • 초기에는 바로 위 상단에 대해서만 DFS를 적용하였기에 테스트를 통과하지 못했다.
  • 가끔 수학 공식을 사용할 수 있는 문제가 나오니 유념하자.
