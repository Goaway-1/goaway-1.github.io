Game Algorithm

Feb 15, 2023 About 1 min

#Algorithm#C++

map

map은 각 노드가 Key, Value 쌍으로 이루어지고 중복을 허용하지 않는 트리구조의 컨테이너이다. 검색, 삽입, 삭제가 O(logn)인 “레드블랙트리”로 구성되어 있다. (균형 이진 트리)

기본적인 형태는 “map<key,value>”로 구성되며, 키를 기준으로 오름차순으로 정렬된다. 이 과정은 자동을 진행되며, 내림차순으로 정렬하고 싶다면, “map<int,int,greater>”처럼 선언하거나 key값이 int형인 경우 음(-)의 기호를 붙여 저장할 수 있다.

unorderd_map

map보다 빠른 탐색을 위한 자료구조로 이는 “해쉬테이블”로 구현했으며, 시간 복잡도는 O(1)이다. 이 또한 중복된 데이터를 허용하지 않지만, key가 유사한 데이터가 많을 시 해시 충돌로 인해 성능이 저하될 수도 있다. 정렬되어 있지 않다.

list

“더블 링크 리스트”의 구조로 이루어진 “이중 연결 리스트”이며, “노드 기반” 컨테이너이다. 시퀀스의 일종으로 순서를 유지하는 구조이다. vector와는 다르게 정렬, 이어붙이기가 가능하며, 임의접근은 불가능하고, 양방향 반복자를 이용하여 탐색할 수 있다.

vector

자동으로 메모리가 할당되는 배열로 동적할당이기 때문에 힙에 생성된다. 일반 배열과 같은 형태이지만 동적으로 크기가 변하고 메모리가 연속적이다. 맨 끝에서만 삽입과 삭제가 발생하며, 중간 삽입과 삭제가 빈번히 일어나는 경우 링크 리스트를 쓰는 것이 효율적이다.

array

배열은 영속된 메모리 영역을 차지하는 동일한 형식의 시퀀스이다. 단일 식별자를 통해 같은 자료형의 여러 변수에 접근할 수 있게 해주는 집계 데이터 유형이다. vector와 유사하지만 메모리가 정적으로 선언되기에 스택에 저장된다.
