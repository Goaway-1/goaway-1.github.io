## 문제
  
  두 쌍의 괄호로 만들 수 있는 모든 괄호문자열의 경우는 (()), ()() 두가지입니다. 자연수 N이 입력되면 N쌍의 괄호로 만들 수 있는 모든 괄호문자열을 출력하는 프로그램을 작성하세요.


## 조건
  - 자연수의 개수 N(3<=N<=10)이 주어집니다.

## 입출력 예시

  |N|result|
  |:--:|:--:|
  |3|["((()))"],["(()())"],["(())()"],["()(())"],["()()()"]],5|
 
## 풀이방식

  > DFS 풀이방식

  DFS를 사용하여 좌우 가로를 삽입하는 풀이로 아래 조건에 따라 삽입합니다. 너무 쉬우니 생략
 
  ```cpp
  int answer = 0;

  void DFS(int n,string str,int left, int right){
    if(left ==  n && right == n){
      cout << str << endl;
      answer++;
    }
    else{
      if(left < n) DFS(n,str+'(',left+1,right);
      if(right < n && left > right) DFS(n,str+')',left,right+1);
    }
  }

  int solution(int n){		
    DFS(n,"",0,0);
    return answer;
  }
  ```
