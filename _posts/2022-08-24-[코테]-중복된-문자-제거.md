## 문제
  
  대소문자가 섞여있는 문자열이 주어지면 해당 문자열을 모두 대문자로 통일한 후, 그 순서는 유지하면서 중복된 문자는 제거한 문자열을 만듭니다. 그 문자열은 사전순으로 가장 빠른 단 어가 되게 만들어야 합니다. 만약 주어진 문자열이 "CbDBEaCEb" 라면 일단 대문자로 통일 합니다. “CBDBEACEB" 그 다음 중복을 제거하되 순서는 유지되면서 사전순으로 가장 빠른 단어를 만듭니다. 그 결과는 “CBDBEACEB"에거 뽑아낸 "BDACE"입니다

## 조건
  - 문자열 str이 입력되며, 이때 문자열의 길이는 50을 넘지 않습니다.

## 입출력 예시

  |str|result|
  |:--:|:--:|
  |CbDBEaCEb|BDACE|
 
## 풀이방식

### 1번 풀이방식 (일부 타임리미트)

  > map을 사용하여 문자를 차레대로 정렬하고, 그 문자가 맨 앞이라는 가정하에 DFS를 진행

  틀린 답이기에 간단하게 설명하자면, 2개의 map을 통해서 "각 문자의 개수", "각 문자가 있는 인덱스 번호"를 추가하였고, 이를 활용하여 첫번째 글자에 대해서만 DFS를 진행했습니다. 이때 정답이 나오면 바로 DFS를 중지했는데, 굳이 왜 첫번째 글자에 대해서만 비교를 했을까요?
  
  ```cpp
  #include <unordered_map>
  #include <map>

  string answer = "";
  void DFS(map<char,int> arr,string str ,int max,string tmpStr, int idx){
    if(idx > str.size()) return;
    
    if(max == tmpStr.size()){
      if(answer.size() == 0) answer = tmpStr;
      else{
        for(int i=0;i<answer.size();i++){
          if(answer[i] < tmpStr[i]) break;
          if(answer[i] > tmpStr[i]){
            answer = tmpStr;
            break;
          }
        }
      }
    }
    else{
      if(tmpStr == ""){
        arr[str[idx]]--;
        DFS(arr,str,max,tmpStr + str[idx], idx+1);
        return;
      }
            
      if(arr[str[idx]] > 0){
        arr[str[idx]]--;
        DFS(arr,str,max,tmpStr + str[idx], idx+1);
        arr[str[idx]]++;
      }
      DFS(arr,str,max,tmpStr, idx+1);
    }
  }

  string solution(string str){	
    map<char, int> M;               //각 char의 개수가 저장된 Mapㄴ
    map<char, vector<int>> Midx;    //각 char의 인덱스가 저장된 Map
       
    // 1번 : 대문자화 및 M, Midx 채움
    for(int i=0;i<str.size();i++){
      if(str[i] >= 'a' && str[i] <= 'z') str[i] = str[i] - 32;
      M[str[i]]++;
      Midx[str[i]].push_back(i);		
    } 
    
    // 2번 : DFS를 위한 배열 제작! -> char의 사용 여부 (중복 방지)
    map<char,int> arr;
    for(auto x = M.begin();x!=M.end();x++) arr.insert(make_pair(x->first,1));
    
    // 3번 : DFS
    for(auto x = M.begin();x!=M.end();x++){
      // 해당 문자가 가능하다면 DFS 
      for(auto y = 0;y < Midx[x->first].size();y++){
        if(Midx[x->first][y] + M.size() - 1 < str.size()){
          DFS(arr,str, M.size(),"",Midx[x->first][y]);
        }
      }
    }
    return answer;
  }
  ```

### 2번 풀이방식

  > 첫번째가 아닌 문자를 하나씩 삽입하면서 기존 문자의 마지막보다 새로 넣으려는 문자가 더 작다면 바꿔치기 합니다.

  먼저 map이 아닌 unordered_map을 2개 사용하여 __"각 문자의 총 개수(sH)", "문자를 사용했는지 여부(cH)"를__ 만듭니다. 그 정보를 사용하여, str의 앞에서부터 answer에 문자를 넣어갑니다. 이때 answer의 마지막 문자가 이제 들어가려는 문자보다 값이 크다면(사전 상 뒤라면) 교체해줍니다. 그럼 그 마지막이였던 문자는 다시 사용할 수 있습니다.

  1. 대문자화 & 각 문자의 총 개수를 sH에 저장
  2. answer에 str을 앞에서부터 삽입
      - 만약 <span style="color:yellow;">현재 answer의 마지막 문자가 새로 들어오려는 문자보다 크고, 나중에 또 출현할 수 있다면 교체</span>

  ```cpp
  #include <unordered_map>
  string solution(string str){	
    string answer = "";
    unordered_map<char, int> sH, ch;
    
    // 1번 : 대문자화 & 각 문자의 총 개수를 sH에 저장
    transform(str.begin(), str.end(), str.begin(), ::toupper);
    for(char x : str) sH[x]++;

    // 2번 : answer에 삽입
    for(char x : str){
      sH[x]--;
      if(ch[x]==1) continue;    // 이미 사용했다면 무시

      // 2-1번 : answer의 마지막 문자가 새로 들어오려는 문자보다 크고, 마지막 문자가 나중에 또 출현할 수 있다면 교체
      while(answer.size() != 0 && answer.back() > x && sH[answer.back()] > 0){
        ch[answer.back()] = 0;
        answer.pop_back();	
      }
      answer.push_back(x);
      ch[x] = 1;
    }
    return answer;
  }
  ```
