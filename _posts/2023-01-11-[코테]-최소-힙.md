## 문제

  널리 잘 알려진 자료구조 중 최소 힙이 있다. 최소 힙을 이용하여 다음과 같은 연산을 지원하는 프로그램을 작성하시오.

    1. 배열에 자연수 x를 넣는다.
    2. 가장 작은 값을 출력하고, 그 값을 배열에서 제거한다.
  
  프로그램은 처음에 비어있는 배열에서 시작하게 된다.

  [링크](https://www.acmicpc.net/problem/1927)
  
## 조건

  - 첫째 줄에 연산의 개수 N(1 ≤ N ≤ 100,000)이 주어진다. 
  - 다음 N개의 줄에는 연산에 대한 정보를 나타내는 정수 x가 주어진다. 
    - 만약 x가 자연수라면 배열에 x라는 값을 넣는(추가하는) 연산이고,
    - x가 0이라면 배열에서 가장 작은 값을 출력하고 그 값을 배열에서 제거하는 경우이다.
    - x는 231보다 작은 자연수 또는 0이고, 음의 정수는 입력으로 주어지지 않는다.

## 입출력 예시
  
  |N|arr|answer|
  |:--:|:--:|:--:|
  |9|[0,12345678,1,2,0,0,0,0,32]|[0,1,2,12345678,0]|

## 풀이방식

  > priority_queue
  
  단순 우선순위 큐를 역순으로 사용할줄 아는가의 문제.. 
  
  '-'로 넣고, '-'더해서 빼면됌

  ```cpp
  #include<iostream>
  #include<queue>
  using namespace std;

  int main() {
    int N = 0, tmp = 0;
    priority_queue<int> pQ;
    cin >> N;

    for (int i = 0; i < N; i++){
      cin >> tmp;
      if (tmp == 0) {
        if (pQ.size() > 0) {
          cout << -pQ.top() << '\n';
          pQ.pop();
        }
        else cout << "0\n";
      }
      else pQ.push(-tmp);
    }
    return 0;
  }
  ```
