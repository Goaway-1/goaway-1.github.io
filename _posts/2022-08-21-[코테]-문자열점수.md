## 문제

  알파벳 소문자로 구성된 길이 1 이상의 두 문자열 A, B가 있습니다. 이 문자열들의 임의의 위 치에 공백을 삽입하여 두 문자열의 길이를 같게 만든 다음, 앞에서부터 한 글자씩 살펴보면서, 같은 위치에 있는 두 문자 A[i], B[i]에 대해서 다음과 같이 점수를 계산합니다. 
  
  1. 두 문자가 같은 경우에는 n1(n1>0)점을 받게 됩니다. 단, 두 문자가 공백으로 같은 경우는 허용하지 않습니다. 
  2. 두 문자 중 하나가 공백인 경우에는 n2(n2<0)점을 받게 됩니다.
  3. 두 문자가 모두 공백이 아니고 서로 다른 경우에는 n3(n3<0)점을 받게 된다.

  예를 들어 n1=5, n2=-1, n3=-3이고 두 문자열이 abcb, acb 인 경우,

  ||||
  |--|--|--|
  |a|b|c|d|
  |a| |c|b|

  이 경우 앞에서부터 점수를 계산하면 각각 5, -1, 5, 5점이 되고 따라서 총점은 14점이 된다. 두 문자열이 주어지면 공백을 적절히 추가하여 얻을 수 있는 최대 총점을 구하는 프로그램을 작성하세요.

## 조건
  - n1,n2,n3의 정보가 담긴 n[3]의 값이 주어집니다.
  - 문자열 A, B가 (문자열의 길이 <= 3000) 주어집니다. 

## 입출력 예시

  |num1|num2|result|
  |:--:|:--:|:--:|
  |acbehf|abefc||
  
## 풀이방식

### 1번 풀이방식 (오답)

  > 공백을 문자 사이에 실제로 삽입하여 DFS로 비교하는 방법

  공백의 개수에는 한계가 없기 때문에 문제가 발생하는 듯 하며, 정답과 일치하지도 않는다. __"공백"이라는__ 단어에 집착한 결과다.
  
  ```cpp
  string getStr(string str, int idx){
    string tmp = str.substr(0,idx);
    tmp += " ";
    tmp += str.substr(idx,str.size());
    
    return tmp;
  } 

  void DFS(string fixStr, string Str,int n[3],int idx){
    if(fixStr.size() == Str.size()){
      int val = 0;
      for(int i=0;i<fixStr.size();i++){
        if(fixStr[i] == Str[i]) val += n[0];
        else if(Str[i] == ' ') val += n[1];
        else val += n[2];
      }
      answer = max(answer, val);
    }
    else{
      if(idx >= Str.size()) return;
      
      DFS(fixStr,getStr(Str,idx),n,idx+1);		//넣음 
      DFS(fixStr,Str,n,idx+1);		            //않넣음 
    }
  }

  int solution(int n[3],string num1,string num2){		
    if(num1.size() > num2.size()) DFS(num1,num2,n,0);
    else if (num1.size() < num2.size()) DFS(num2,num1,n,0);
    else{	
      for(int i=0;i<num1.size();i++){
        if(num1[i] == num2[i]) answer += n[0];
        else answer += n[2];
      }
    }
      
    return answer;
  }
  ```

### 2번 풀이방식 (정답)

  > 동적 계획법을 사용하여, 이전 값을 활용하는 방법

  <img src="https://raw.githubusercontent.com/Goaway-1/goaway-1.github.io/master/_posts/images/StringAndScore.png" height="350" title="StringAndScore">

  1. 두 문자열 A,B에 대해서 앞에 공백을 추가한다. (동적 계획법을 진행하며 공백에 대해 경우의 수를 추가하기 위함)
  2. 공백일때의 경우를 먼저 계산 (0행 n열), (n행 0열)
  3. 메인 로직으로 dy배열을 채워간다.
      - 3-1 : A[i]와 B[j]가 일치하는 경우에는 dy[i-1][j-1]에 n1점을 더한다.
      - 3-2 : 두 값이 다른 경우에는 가장 큰 값이 만들어지는 경우를 사용한다.
        - 3-2-1 : 두 개 중 하나의 문자만 사용하는 경우에서 가장 큰 값을 도출 (i-1,j) 과 (i,j-1)의 비교. (+ 공백 값(n2))
        - 3-2-2 : 두 문자를 모두 사용하는 경우로 "두 문자가 다른 경우", "두 문자가 각각 공백에 대응하는 경우"에 따라 나눈다.
  4. 이 과정을 반복하여 마지막 배열의 값이 가장 큰 값이 들어온다.

  ```cpp
  int solution(int n[3],string A,string B){		
    // 1번 : A, B에 공백 추가
    A = ' ' + A;
    B = ' ' + B;
    int x = A.size();
    int y = B.size();
    vector<vector<int> > dy(x, vector<int>(y, 0));

    // 2번 : 공백일때의 경우 처리
    for(int i=1; i<x; i++) dy[i][0]=n[1]*i;
    for(int i=1; i<y; i++) dy[0][i]=n[1]*i;
    
    // 3번 : 메인 로직
    for(int i=1; i<x; i++){
      for(int j=1; j<y; j++){
        // 3-1번 : A[i]와 B[j]가 일치하는 경우
        if(A[i] == B[j]) dy[i][j] = dy[i-1][j-1] + n[0];
        // 3-2번 : 다른 경우
        else{
          int max;
          
          /** 3-2-1 : 두개 중 하나의 문자만 사용하는 경우 */
          // 하나의 문자만 사용했을 때 값이 커지는 경우 
          if(dy[i-1][j] + n[1] > dy[i][j-1] + n[1]) max = dy[i-1][j] + n[1];
          else max = dy[i][j-1] + n[1];				
          
          /** 3-3 : 두 문자를 모두 사용하는 경우 */
          // 두 문자가 서로 다름 (a,b) 
          if(dy[i-1][j-1] + n[2] > max) max = dy[i-1][j-1] + n[2];
          
          // 두 문자 모두 공백 처리 (a "", "" b)
          if(dy[i-1][j-1] + (n[1]*2)>max) max = dy[i-1][j-1] + (n[1]*2);
          dy[i][j] = max;
        }
      }
    }
    
    /*for(int i=0; i<x; i++){
      for(int j=0; j<y; j++) cout<<dy[i][j]<<"\t ";
      cout<<endl;
    }*/
      
    return dy[x-1][y-1];
  }
  ```
