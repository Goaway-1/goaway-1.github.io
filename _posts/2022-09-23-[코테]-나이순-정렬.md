## 문제
  
  온라인 저지에 가입한 사람들의 나이와 이름이 가입한 순서대로 주어진다. 이때, 회원들을 나이가 증가하는 순으로, 나이가 같으면 먼저 가입한 사람이 앞에 오는 순서로 정렬하는 프로그램을 작성하시오.

## 조건

  - 첫째 줄에 온라인 저지 회원의 수 N이 주어진다. (1 ≤ N ≤ 100,000)
  - 둘째 줄부터 N개의 줄에는 각 회원의 나이와 이름이 공백으로 구분되어 주어진다. 나이는 1보다 크거나 같으며, 200보다 작거나 같은 정수이고, 이름은 알파벳 대소문자로 이루어져 있고, 길이가 100보다 작거나 같은 문자열이다. 입력은 가입한 순서로 주어진다.

## 입출력 예시

  |N|Arr|return|
  |:--:|:--:|:--:|
  |3|[[21, Junkyu], [21, Dohyun], [20, Sunyoung]]|[[20, Sunyoung], [21, Junkyu], [21, Dohyun]]|

## 풀이방식

  > 구조체와 sort알고리즘을 활용한 풀이

  자세한 설명은 주석으로 대체합니다.

  ```cpp
  #include <iostream> 
  #include <vector> 
  #include <string>
  #include <algorithm>

  using namespace std;

  // 회원의 구조체
  struct st {    
    int index, age;
    string name;

    st(int i, int a, string n) : index(i), age(a), name(n) {}
    void getData() { cout << age << " " << name << "\n"; }
    
    // sort시 사용되는 조건
    bool operator <(const st& b) {
      if(age != b.age) return age < b.age;
      else return index < b.index;
    }
  };

  int main(){
    int n, age;
    string name;
    vector<st> arr;
    cin >> n;

    for (int i = 0; i < n; i++){
      cin >> age >> name;
      arr.push_back(st(i,age,name));
    }

    // 조건에 따른 배열의 순서 조율
    sort(arr.begin(), arr.end());
    for (int i = 0; i < n; i++) arr[i].getData();
    return 0;
  }
  ```

## 개선점
  - 구조체를 만들필요는 없지만 습관상 만들어버림.
  - operator 조건에서 2번째 조건은 필요없는데 만들어버림.
