## 문제

  두 마리의 백조가 호수에서 살고 있었다. 그렇지만 두 마리는 호수를 덮고 있는 빙판으로 만나지 못한다.

  호수는 행이 R개, 열이 C개인 직사각형 모양이다. 어떤 칸은 얼음으로 덮여있다.

  호수는 차례로 녹는데, 매일 물 공간과 접촉한 모든 빙판 공간은 녹는다. 두 개의 공간이 접촉하려면 가로나 세로로 닿아 있는 것만 (대각선은 고려하지 않는다) 생각한다.

  아래에는 세 가지 예가 있다.

  ```
    ...XXXXXX..XX.XXX ....XXXX.......XX .....XX.......... 
    ....XXXXXXXXX.XXX .....XXXX..X..... ......X.......... 
    ...XXXXXXXXXXXX.. ....XXX..XXXX.... .....X.....X..... 
    ..XXXXX..XXXXXX.. ...XXX....XXXX... ....X......XX.... 
    .XXXXXX..XXXXXX.. ..XXXX....XXXX... ...XX......XX.... 
    XXXXXXX...XXXX... ..XXXX.....XX.... ....X............ 
    ..XXXXX...XXX.... ....XX.....X..... ................. 
    ....XXXXX.XXX.... .....XX....X..... ................. 
          처음               첫째 날             둘째 날
  ```

  백조는 오직 물 공간에서 세로나 가로로만(대각선은 제외한다) 움직일 수 있다.

  며칠이 지나야 백조들이 만날 수 있는 지 계산하는 프로그램을 작성하시오.

## 조건

  - 입력의 첫째 줄에는 R과 C가 주어진다. 단, 1 ≤ R, C ≤ 1500.
  - 다음 R개의 줄에는 각각 길이 C의 문자열이 하나씩 주어진다. '.'은 물 공간, 'X'는 빙판 공간, 'L'은 백조가 있는 공간으로 나타낸다.

## 입출력 예시
  
  |R|C|arr|result|
  |:--:|:--:|:--:|:--:|
  |10|2|[".L","..","XX","XX","XX","XX","XX","XX","..",".L"]|3|
  |4|11|["..XXX...X..",".X.XXX...L.","....XXX..X.","X.L..XXX..."]|2|
  |8|17|["...XXXXXX..XX.XXX","....XXXXXXXXX.XXX","...XXXXXXXXXXXX..","..XXXXX.LXXXXXX..",".XXXXXX..XXXXXX..","XXXXXXX...XXXX...","..XXXXX...XXX....","....XXXXX.XXXL..."]|2|

## 풀이방식

  > BFS와 4개의 Queue를 이용한 풀이

  BFS를 활용한 문제인 것은 알았기에, 얼음이 녹는 과정은 쉽게 진행하였으나 백조가 만나는 날을 계산하는 것에 어려움을 겪었음

  백조는 물위에 존재하기 때문에 특정 백조에서 당일 탐색할 수 있는 곳을 __"duckQ와 BFS"를__ 사용하여 탐색하고, 얼음인 곳은 다음날 탐색이 가능하기 때문에 __"duckNQ를"__ 사용하여 저장했다.
  이후 물이 존재하는 모든 곳에서도 탐색을 진행하고, __"iceQ와 iceNQ, BFS를"__ 사용했다.

  위 과정을 반복하여 물의 길이 열렸을때, 백조가 또 다른 백조를 만나게되면 반복문을 종료하게 된다.

  이때 추가적으로 백조가 진행했던 길을 파악하기 위해서 INT형 2차원 배열(chmap)을 사용하였다.
  
  ---

  1. 특정 백조의 위치와 모든 물의 위치를 각 Queue에 저장
  2. 백조, 물의 순서대로 오늘날 진행가능한 모든 곳을 BFS로 탐색한다.
    - 얼음을 만난 경우 다음날 진행 가능하기 때문에 다음날의 Queue에 저장한다.
    - 이때 백조는 탐색했던 좌표를 판단한 새로운 2차원 배열을 추가적으로 사용한다.
  3. 위 과정을 반복하며, 백조가 BFS를 진행할때 또 다른 백조를 마주친다면 while()문을 종료한다. 

  ```cpp
  #include <iostream>
  #include <vector>
  #include <string>
  #include <algorithm>
  #include <queue>
  using namespace std;

  // 상하좌우 이동을 위한 POS
  int xx[] = { -1,0,1,0 };
  int yy[] = { 0,1,0,-1 };

  int main() {
    int r, c;
    cin >> r >> c;
    vector<vector<char>> map(r, vector<char>(c));       // 현재 물, 얼음, 백조의 상태를 표현
    vector<vector<int>> chmap(r, vector<int>(c,0));     // 백조가 진행한 길을 표현

    /** 현재 이동 가능한 물과 백조의 좌표를 저장 */
    queue<pair<int, int>> iceQ;		  
    queue<pair<int, int>> duckQ; 

    /** 입력 */  
    for (int i = 0; i < r; i++) {
      for (int j = 0; j < c; j++){
        cin >> map[i][j];

        if (map[i][j] != 'X')  iceQ.push(make_pair(i, j));		
        if (map[i][j] == 'L' && duckQ.size() == 0) duckQ.push(make_pair(i, j));
      }
    }

    bool isDone = false;  // 백조끼리의 만남 여부
    int cnt = 0;          // 날의 흐름
    while (!isDone) {
      /** 물과 백조가 다음에 진행할 수 있는 좌표 */
      queue<pair<int, int>> iceNQ;		
      queue<pair<int, int>> duckNQ;

      /** DUCK */
      chmap[duckQ.front().first][duckQ.front().second] = 1;
      while (!duckQ.empty()) {
        int x = duckQ.front().first;
        int y = duckQ.front().second;
        duckQ.pop();

        for (int i = 0; i < 4; i++) {
          int tmpX = x + xx[i];
          int tmpY = y + yy[i];

          /** Duck's BFS
            X : 다음 날 진행 가능하기에 다음큐(duckNQ)에 삽입 
            . : 현재 진행 가능하기에 현재큐(duckQ)에 삽입 
            L : 다른 백조와의 만남을 의미하며 종료
          */
          if (tmpX >= 0 && tmpX < r && tmpY >= 0 && tmpY < c && chmap[tmpX][tmpY] == 0) {
            if (map[tmpX][tmpY] == 'X') duckNQ.push(make_pair(tmpX, tmpY));
            else if (map[tmpX][tmpY] == '.') duckQ.push(make_pair(tmpX, tmpY));
            else {
              isDone = true;
              break;
            }

            chmap[tmpX][tmpY] = 1;
          }
        }
        if (isDone) break;
      }

      if (isDone) break;
      
      /** Water */
      while (!iceQ.empty()) {
        int x = iceQ.front().first;
        int y = iceQ.front().second;
        iceQ.pop();

        for (int i = 0; i < 4; i++) {
          int tmpX = x + xx[i];
          int tmpY = y + yy[i];

          /** Water's BFS
            X : 다음 날 진행 가능하기에 다음큐(iceNQ)에 삽입 
          */
          if (tmpX >= 0 && tmpX < r && tmpY >= 0 && tmpY < c) {
            if (map[tmpX][tmpY] == 'X') {
              map[tmpX][tmpY] = '.';
              iceNQ.push(make_pair(tmpX, tmpY));
            }
          }
        }
      }

      /** 다음 날 진행을 위한 초기화 */
      duckQ = duckNQ;
      iceQ = iceNQ;
      cnt++;

      /*for (int i = 0; i < r; i++) {
        for (int j = 0; j < c; j++)cout << map[i][j] << " ";
        cout << "\n";
      }
      cout << "##################\n";*/
    }
    cout << cnt;
  }
  ```

## 개선점
  - Queue와 BFS를 조합하여 사용하는 방법은 알고 있지만 정확히 응용하는 방법은 무지한 것 같다.
  - 백준에 있는 다양한 문제를 통해서 접근해보자.
