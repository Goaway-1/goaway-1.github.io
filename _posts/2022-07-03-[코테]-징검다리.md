## 문제

  출발지점부터 distance만큼 떨어진 곳에 도착지점이 있습니다. 그리고 그사이에는 바위들이 놓여있습니다. 바위 중 몇 개를 제거하려고 합니다.
  
  예를 들어, 도착지점이 25만큼 떨어져 있고, 바위가 [2, 14, 11, 21, 17] 지점에 놓여있을 때 바위 2개를 제거하면 출발지점, 도착지점, 바위 간의 거리가 아래와 같습니다.

  |제거한 바위의 위치|각 바위 사이의 거리|거리의 최솟값|
  |:--:|:--:|:--:|
  |[21, 17]|[2, 9, 3, 11]|2|
  |[2, 21]|[11, 3, 3, 8]|3|
  |[2, 11]|[14, 3, 4, 4]|3|
  |[11, 21]|[2, 12, 3, 8]|2|
  |[2, 14]|[11, 6, 4, 4]|4|

  위에서 구한 거리의 최솟값 중에 가장 큰 값은 4입니다.

  출발지점부터 도착지점까지의 거리 distance, 바위들이 있는 위치를 담은 배열 rocks, 제거할 바위의 수 n이 매개변수로 주어질 때, 바위를 n개 제거한 뒤 각 지점 사이의 거리의 최솟값 중에 가장 큰 값을 return 하도록 solution 함수를 작성해주세요.

## 조건
  
  - 도착지점까지의 거리 distance는 1 이상 1,000,000,000 이하입니다.
  - 바위는 1개 이상 50,000개 이하가 있습니다.
  - n 은 1 이상 바위의 개수 이하입니다.
 
## 입출력 예시

  |distance|rocks|n|return|
  |:--:|:--:|:--:|:--:|
  |25|[2, 14, 11, 21, 17]|2|4|

## 풀이방식

  바위 사이의 최솟값을 기준으로 이분탐색을 진행하였다. 즉, 최소한 둘 사이의 거리가 최솟값이상이여야 하고, 그렇지 않으면 바위를 제거하는 로직이다.

  좌측은 "시작지점(0)", 우측은 "도착지점(distance)"으로 잡고 이분탐색을 진행하며, isPos()메서드의 skip은 제거한 바위의 수, curPos는 현재 위치이다.

  이때 조건은 최솟값 cnt가 들어왔을때 "curPos + cnt < rocks[i]"를 만족한다면 그 돌은 삭제하지 않고, 그렇지 않다면 돌을 삭제한다. 그리고 이때 삭제된 돌의 개수와 n이 같은지 확인하고 bool값을 반환한다.

  그 후 이분탐색의 기존 방식대로 left와 right를 교체해주면서 진행해나간다.

  ```cpp
  #include <stack> 
  bool isPos(vector<int> &rocks,int cnt,int n) {
    int skip = 0;       //삭제한 돌의 수
    int curPos = 0;

    for (int i = 0; i < rocks.size(); i++) {
      if(curPos + cnt <= rocks[i]) curPos = rocks[i];
      else {
        skip++;
        if (skip > n) return false;
      }
    }
  return true;
  }
  int solution(int distance, vector<int> rocks, int n) {
    int answer = 0;

    int right = distance;
    int left = 0;
    int mid = 0;

    sort(rocks.begin(),rocks.end());

    while (left < right) {
      mid = (right + left) / 2;

      if (isPos(rocks,mid,n)) {
        left = mid + 1;
        answer = mid;
      }
      else right = mid;
    }
    return answer;
  }
  ```
