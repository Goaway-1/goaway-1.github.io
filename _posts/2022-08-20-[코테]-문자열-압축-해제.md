## 문제

  압축된 결과의 문자열이 주어지면 다시 원 상태로 압축을 해제하여 출력하는 프로그램을 작성하세요. 압축된 결과는 3(ab) 라고 주어지면 괄호안에 문자열이 3번 반복된 것을 압축했다는 의미입니다. 이걸 원상태로 해제한 것은 “ababab”입니다. 만약 2(ab)(k)3(bc) 를 압축해제 하면 "ababkbcbcbc"입니다. 2(ab)(k)3(bc)에서 (k)문자열과 같이 반복횟수가 1인 경우는 숫자를 생략하고 압축되어 있습니다.

## 조건
 
  - 압축된 결과인 문자열 zip이 주어집니다.

## 입출력 예시

  |zip|result|
  |:--:|:--:|
  |2(ab)(k)3(bc)|ababkbcbcbc|
  
## 풀이방식

### 나의 풀이방식
  >  string 문자열을 분해할 수 있냐는 문제

  너무 쉬워서 자세한 설명은 생략합니다.

  ```cpp
  string solution(string zip){	
    string answer;
    int cnt = 0;
    string tmp = "";
    
    for(int i=0;i<zip.size();i++){
      if(zip[i] == '(' || zip[i] == ')'){
        // answer에 추가
        if(zip[i] == ')'){				
          if(cnt == 0) answer += tmp;
          else for(int j=0;j<cnt;j++) answer += tmp;
          
          cnt = 0;
          tmp = "";
        }
      }
      else if(zip[i] >= 'a' && zip[i] <= 'z') tmp += zip[i];    //문자의 경우
      else cnt = (cnt * 10) + (zip[i]- 48);                     //숫자의 경우
    }
    
    return answer;	
  }
  ```

### 다른이의 풀이방식

  > stack을 사용하여 숫자, 문자부분을 따로 추출한 풀이이다.

  굳이 stack을 사용할 이유도 없다고 본다. 불필요한 과정의 반복임 특히 reverse()

  ```cpp
  string solution(string num){	
    string answer;
    stack<char> s;
    
    for(char x : num){
      if(x == ')'){
        // 문자 추출 
        string tmp;
        while(!s.empty() && s.top()!='('){
          tmp += s.top();
          s.pop();
        }
        s.pop();
        reverse(tmp.begin(), tmp.end());
        
        // 숫자 추출 
        string num;
        if(s.empty()) num="1";
        while(!s.empty()){
          num+=s.top();
          s.pop();
        }
        reverse(num.begin(), num.end());
        
        // answer에 추가 
        int n = stoi(num);
        for(int i=0; i<n; i++) answer += tmp;
      }
      else s.push(x);
    }
    
    return answer;	
  }
  ```
