## 문제

  철수는 롤케이크를 두 조각으로 잘라서 동생과 한 조각씩 나눠 먹으려고 합니다. 이 롤케이크에는 여러가지 토핑들이 일렬로 올려져 있습니다. 철수와 동생은 롤케이크를 공평하게 나눠먹으려 하는데, 그들은 롤케이크의 크기보다 롤케이크 위에 올려진 토핑들의 종류에 더 관심이 많습니다. 그래서 잘린 조각들의 크기와 올려진 토핑의 개수에 상관없이 각 조각에 동일한 가짓수의 토핑이 올라가면 공평하게 롤케이크가 나누어진 것으로 생각합니다.

  예를 들어, 롤케이크에 4가지 종류의 토핑이 올려져 있다고 합시다. 토핑들을 1, 2, 3, 4와 같이 번호로 표시했을 때, 케이크 위에 토핑들이 [1, 2, 1, 3, 1, 4, 1, 2] 순서로 올려져 있습니다. 만약 세 번째 토핑(1)과 네 번째 토핑(3) 사이를 자르면 롤케이크의 토핑은 [1, 2, 1], [3, 1, 4, 1, 2]로 나뉘게 됩니다. 철수가 [1, 2, 1]이 놓인 조각을, 동생이 [3, 1, 4, 1, 2]가 놓인 조각을 먹게 되면 철수는 두 가지 토핑(1, 2)을 맛볼 수 있지만, 동생은 네 가지 토핑(1, 2, 3, 4)을 맛볼 수 있으므로, 이는 공평하게 나누어진 것이 아닙니다. 만약 롤케이크의 네 번째 토핑(3)과 다섯 번째 토핑(1) 사이를 자르면 [1, 2, 1, 3], [1, 4, 1, 2]로 나뉘게 됩니다. 이 경우 철수는 세 가지 토핑(1, 2, 3)을, 동생도 세 가지 토핑(1, 2, 4)을 맛볼 수 있으므로, 이는 공평하게 나누어진 것입니다. 공평하게 롤케이크를 자르는 방법은 여러가지 일 수 있습니다. 위의 롤케이크를 [1, 2, 1, 3, 1], [4, 1, 2]으로 잘라도 공평하게 나뉩니다. 어떤 경우에는 롤케이크를 공평하게 나누지 못할 수도 있습니다.

  롤케이크에 올려진 토핑들의 번호를 저장한 정수 배열 topping이 매개변수로 주어질 때, 롤케이크를 공평하게 자르는 방법의 수를 return 하도록 solution 함수를 완성해주세요.

## 조건

  - 1 ≤ topping의 길이 ≤ 1,000,000
  - 1 ≤ topping의 원소 ≤ 10,000

## 입출력 예시
  
  |queue1|result|
  |:--:|:--:|
  |[1, 2, 1, 3, 1, 4, 1, 2]|2|
  |[1, 2, 3, 1, 4]|0|

## 풀이방식

### 느린 풀이방식
 
  > unordered_map을 사용한 풀이

  단순히 형과 동생의 map을 구성하고, map.size()로 토핑의 종류를 카운팅한다. 

  초기 동생이 모두 먹는다고 가정하고 map을 구성하며, 추후 형이 하나씩 더 먹어간다고 가정하여 map을 늘려간다.

  ※ 주석참조

  ```cpp
  #include <string>
  #include <vector>
  #include <unordered_map>
  using namespace std;

  int solution(vector<int> topping) {
    int answer = 0;
    unordered_map<int,int> M1, M2;

    // 동생이 모두 먹는다 가정
    for (int i = 0; i < topping.size(); i++) M2[topping[i]]++;

    // 형이 하나씩 먹기 시작
    for (int i = 0; i < topping.size(); i++){
      M1[topping[i]]++;
      if(--M2[topping[i]] == 0) M2.erase(topping[i]);

      if(M1.size() == M2.size()) answer++;
    }

    return answer;
  }
  ```

### 빠른 풀이방식

  > unordered_map 대신 vector를 사용한 풀이

  위 방법은 시간복잡도가 느리기 때문에 vector를 사용하기로 함

  ```cpp
  #include <string>
  #include <vector>
  using namespace std;

  int solution(vector<int> topping){
    int answer = 0;
    vector<int> OldVec(10001, 0), YngVec(10001, 0);       //Old는 형, Yng은 동생
    int OldCnt = 0, YngCnt = 0;                           //종류의 개수

    // 동생이 모두 먹는다 가정
    for (int i = 0; i < topping.size(); i++){
      if (YngVec[topping[i]] == 0) YngCnt++;
      YngVec[topping[i]]++;
    }

    // 형이 하나씩 먹기 시작
    for (int i = 0; i < topping.size(); i++){
      if (--YngVec[topping[i]] == 0) YngCnt--;
      if (OldVec[topping[i]]++ == 0) OldCnt++;

      if(OldCnt == YngCnt) answer++;
    }
    return answer;
  }
  ```

## 개선점
  - map과 같은 STL은 느리다!
