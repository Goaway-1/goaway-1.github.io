## 문제
  
  모두가 잘 알고 있듯이 톰과 제리는 별로 가까운 사이가 아니다. 항상 톰은 제리를 쫒아다녔 지만 그때마다 번번이 실패하고 말았다. 곰곰이 꾀를 내던 톰은 제리가 자고 있을 때 가장 빠 른 길로 잽싸게 달려가면 제리를 잡을 수 있겠다 싶었다. 그런데 그러기 위해서는 가장 적은 횟수로 방향을 바꾸어서 제리가 있는 곳까지 가는 방법을 알아야만 한다. 톰과 제리가 있는 정사각형 모양의 방의 상태와, 둘의 위치가 주어질 때 이와 같은 경로를 찾아내는 프로그램을 작성하세요.

## 조건

  - 방의 넓이인 N(N<=100)이 주어집니다.
  - N*N의 지도 정보 arr가 주어집니다.
  - 톰과 제리의 위치를 표현한 T와 J가 주어집니다.

## 입출력 예시

  |N|arr|T|J|result|
  |:--:|:--:|:--:|:--:|:--:|
  |10|[[0000000000],[0110000111],[0110110000],[0110010110],[0000011100],[0100000000],[0111100011],[0100000000],[0000011100],[0111100000]]|[1,1]|[10,9]|4|
 
## 풀이방식

  > BFS 풀이방식

  BFS를 사용하는 방식으로 한 좌표에 대해서 상하좌우 한칸씩만 체크하는 것이 아닌, __<ins>상하좌우 행렬에</ins>__ 대해서 체크를 진행한다. 이때 벽에 닿으면 멈춘다.

  ```cpp
  int x[4] = {0,1,0,-1};
  int y[4] = {1,0,-1,0};

  int solution(int n,vector<vector<int>> arr,	vector<pair<int,int>> T, vector<pair<int,int>> J){		
    queue<pair<int, int>> Q;
    
    // INIT
    Q.push(make_pair(T.front().first,T.front().second));
    arr[T.front().first][T.front().second] = 2;
    
    // BFS
    while(!Q.empty()){
      int sx = Q.front().first;
      int sy = Q.front().second;
      Q.pop();
      
      for(int i=0;i<4;i++){
        int tx = sx + x[i];
        int ty = sy + y[i];
        
        /** ☆ 상하좌우 행렬이 가능할때까지 과정을 반복한다. ★
        * 이때 벽을 마주한다면 중지한다.
        */
        while(tx > 0 && tx <= n && ty > 0 && ty <= n && arr[tx][ty] != 1){
          if(arr[tx][ty] == 0){
            Q.push(make_pair(tx,ty));
            arr[tx][ty] = arr[sx][sy] + 1;
          }
          tx += x[i];
          ty += y[i];
        }
      } 
    }

    if(arr[J.front().first][J.front().second] == 0) return -1;
    return arr[J.front().first][J.front().second] - 3;
  }  
  ```

## 개선점
  - 초기에는 한번에 한칸씩만을 진행했기 때문에 어려움을 겪었지만, 한번에 한 '행' 또는 '열'에서 가능한 모든 것을 진행하였다.
