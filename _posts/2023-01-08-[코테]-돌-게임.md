## 문제

  돌 게임은 두 명이서 즐기는 재밌는 게임이다.

  탁자 위에 돌 N개가 있다. 상근이와 창영이는 턴을 번갈아가면서 돌을 가져가며, 돌은 1개 또는 3개 가져갈 수 있다. 마지막 돌을 가져가는 사람이 게임을 이기게 된다.

  두 사람이 완벽하게 게임을 했을 때, 이기는 사람을 구하는 프로그램을 작성하시오. 게임은 상근이가 먼저 시작한다.

  [링크](https://www.acmicpc.net/problem/9655)
  
## 조건

  - 첫째 줄에 N이 주어진다. (1 ≤ N ≤ 1000)

## 입출력 예시
  
  |N|answer|
  |:--:|:--:|
  |5|SK|  

## 풀이방식

  > 돌의 개수를 3으로 나눈 값과 나머지를 구하여 계산하는 문제

  돌을 가져갈 수 있는 개수는 항상 홀수개인 것을 확인할 수 있다. 그렇기 때문에 최대로 가져갈 수 있는 돌의 개수인 3으로 현재 돌을 나눈다.

  그렇게 값과 나머지를 구하고, 값이 짝수라면 가장 마지막으로 창수가 가져간 것이고, 홀수라면 상근이가 가져갔다고 생각할 수 있다.

  나머지가 짝수라면 그대로 두고, 홀수(1)라면 기존 마지막으로 가져간 인원이 아닌 다른 인원이 마지막 돌을 가져가게 된다. 

  ```cpp
  #include<iostream>
  #include<string>
  using namespace std;

  int main(void){
    int N = 0, ans = 0;
    string player[2] = {"SK","CY"};
    cin >> N;

    //  값과 나머지 계산 
    ans = ((N / 3) % 2 == 0) ? 1 : 0;
    if (N % 3) ans = (ans == 1) ? 0 : 1;

    cout << player[ans];
    return 0;
  }
  ```

## 개선점
  - 수학적으로 계산했어야 했는데, 나 좀 잘한듯
