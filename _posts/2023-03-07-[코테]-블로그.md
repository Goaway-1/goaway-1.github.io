## 문제

  찬솔이는 블로그를 시작한 지 벌써 $N$일이 지났다.

  요즘 바빠서 관리를 못 했다가 방문 기록을 봤더니 벌써 누적 방문 수가 6만을 넘었다.

  ![이미지](https://upload.acmicpc.net/5f95a11c-b879-408b-b3be-dcaa915f36ab/-/preview/)

  찬솔이는 $X$일 동안 가장 많이 들어온 방문자 수와 그 기간들을 알고 싶다.

  찬솔이를 대신해서 $X$일 동안 가장 많이 들어온 방문자 수와 기간이 몇 개 있는지 구해주자.

  [링크](https://www.acmicpc.net/problem/21921)
  
## 조건

  - 첫째 줄에 블로그를 시작하고 지난 일수 $N$와 $X$가 공백으로 구분되어 주어진다.
  - 둘째 줄에는 블로그 시작 $1$일차부터 $N$일차까지 하루 방문자 수가 공백으로 구분되어 주어진다.

## 입출력 예시
  
  |N|X|arr|answer|
  |:--:|:--:|:--:|:--:|
  |5|2|[1,4,2,5,1]|[7,1]|
  |7|5|[1,1,1,1,1,5,1]|[9,2]|
  |5|3|[0,0,0,0,0]|["SAD"]|

## 풀이방식

  > 누적합과 슬라이딩 윈도우를 활용한 문제 풀이

  초기 X개의 누적합을 저장해두고, __"1번 idx의 값을 빼고, X + 1번 idx의 값을 더하여"__ 누적합을 지속적으로 갱신한다. 

  ```cpp
  #include <iostream>
  #include <vector>
  using namespace std;

  int arr[250001];

  int main() {
    cin.tie(NULL);
    ios::sync_with_stdio(false);
    
    int n = 0, x = 0;

    // INIT
    cin >> n >> x; 
    for (int i = 0; i < n; i++) cin >> arr[i];

    /** 계산 */
    int i = 0, sum = 0, ans = 0, cnt = 1;		  // 인덱스, 총합, 최대 방문자 수, 같은 기간의 개수..

    for (; i < x; i++) sum += arr[i];		      //초기 X개수 만큼만 누적합 진행

    ans = sum;

    // 배열을 순회하며, 누적합을 갱신
    for (; i < n; i++) {
      sum = sum - arr[i - x] + arr[i];

      if (ans < sum) {
        ans = sum;
        cnt = 1;
      }
      else if (ans == sum) {
        cnt++;
      }
    }

    // 결과값 출력
    if(ans > 0) cout << ans << '\n' << cnt;
    else cout << "SAD";

    return 0;
  }
  ```
