## 문제
 
  사전에 알파벳 모음 'A', 'E', 'I', 'O', 'U'만을 사용하여 만들 수 있는, 길이 5 이하의 모든 단어가 수록되어 있습니다. 사전에서 첫 번째 단어는 "A"이고, 그다음은 "AA"이며, 마지막 단어는 "UUUUU"입니다.

  단어 하나 word가 매개변수로 주어질 때, 이 단어가 사전에서 몇 번째 단어인지 return 하도록 solution 함수를 완성해주세요.
  
## 조건

  - word의 길이는 1 이상 5 이하입니다.
  - word는 알파벳 대문자 'A', 'E', 'I', 'O', 'U'로만 이루어져 있습니다.

## 입출력 예시

  |word|result|
  |:--:|:--:|
  |"AAAAE"|6|
  |"AAAE"|10|
  |"I"|1563|
  |"EIO"|1189|

## 풀이방식
 
  DFS 풀이의 정석이 아닌가 생각해본다. curStr이라는 문자열에 알파벳을 추가/삭제를 반복하여 answer의 값을 도출한다.

  ```c++
  vector<char> alpha = { 'A','E','I','O','U' };
  int answer = 0, cnt = 0;

  void DFS(const string &word, string curStr) {
    if (answer != 0) return;

    if (word == curStr) answer = cnt;
    else if (curStr.size() >= 5) return;
    else {
      for (int i = 0; i < alpha.size(); i++){ 
        if (answer != 0) return;
        cnt++;
        DFS(word, curStr + alpha[i]);
      }
    }
  }

  int solution(string word) {
    DFS(word,"");
    return answer;
  }
  ```

## 개선점
  - 만약 word가 배열로 주어지고, result의 결과 또한 배열로 출력되야 한다면, 많은 시간을 잡아먹을 것으로 예상된다. 이때 푸는 방법은 아래와 같다.

  1. word에서 사전상 가장 나중에 나오는 단어까지 DFS를 반복하며 이 과정에서 나오는 모든 단어는 unordered_map에 저장한다.
  2. 계산이 완료되면, map에서 맞는 단어를 찾아 결과를 반환한다.
