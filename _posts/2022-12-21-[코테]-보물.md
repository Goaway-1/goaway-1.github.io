## 문제

  옛날 옛적에 수학이 항상 큰 골칫거리였던 나라가 있었다. 이 나라의 국왕 김지민은 다음과 같은 문제를 내고 큰 상금을 걸었다.

  길이가 N인 정수 배열 A와 B가 있다. 다음과 같이 함수 S를 정의하자.

  ```
  S = A[0] × B[0] + ... + A[N-1] × B[N-1]
  ```

  S의 값을 가장 작게 만들기 위해 A의 수를 재배열하자. 단, B에 있는 수는 재배열하면 안 된다.

  S의 최솟값을 출력하는 프로그램을 작성하시오.

  [링크](https://www.acmicpc.net/problem/1026)
  
## 조건

  - 첫째 줄에 N이 주어진다. 
  - 둘째 줄에는 A에 있는 N개의 수가 순서대로 주어지고, 
  - 셋째 줄에는 B에 있는 수가 순서대로 주어진다. N은 50보다 작거나 같은 자연수이고, A와 B의 각 원소는 100보다 작거나 같은 음이 아닌 정수이다.

## 입출력 예시
  
  |N|A|B|result|
  |:--:|:--:|:--:|:--:|
  |5|[1,1,1,6,0]|[2,7,8,3,1]|18|
  |3|[1,1,3]|[10,30,20]|80|
  |9|[5,15,100,31,39,0,0,3,26]|[11,12,13,2,3,4,5,9,1]|528|

## 풀이방식

  > 최솟값 : 가장 작은 수와 가장 큰 수를 곱하는 경우 도출

  가장 작은 수와 가장 큰 수를 곱하는 경우가 최솟값을 도출할 수 있다. 그렇기에 A배열을 오름차순으로 정렬하고, B배열을 내림차순으로 정렬한다.

  그 후 인덱스에 맞게 서로 곱하고 더해 결과값을 저장한다.

  ```cpp
  #include <iostream>
  #include <vector>
  #include <algorithm>
  using namespace std;

  int main(){
    int N = 0, answer = 0;
    cin >> N;
    vector<int> A(N), B(N);

    /** INIT & SORT */
    for (int i = 0; i < N; i++) cin >> A[i];
    for (int i = 0; i < N; i++) cin >> B[i];
    sort(A.begin(),A.end());
    sort(B.begin(),B.end(),greater<int>());

    for (int i = 0; i < N; i++) answer += (A[i] * B[i]);

    cout << answer;
    return 0;
  }
  ```
