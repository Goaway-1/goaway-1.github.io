## 문제

  준규는 N×M 크기의 미로에 갇혀있다. 미로는 1×1크기의 방으로 나누어져 있고, 각 방에는 사탕이 놓여져 있다. 미로의 가장 왼쪽 윗 방은 (1, 1)이고, 가장 오른쪽 아랫 방은 (N, M)이다.

  준규는 현재 (1, 1)에 있고, (N, M)으로 이동하려고 한다. 준규가 (r, c)에 있으면, (r+1, c), (r, c+1), (r+1, c+1)로 이동할 수 있고, 각 방을 방문할 때마다 방에 놓여져있는 사탕을 모두 가져갈 수 있다. 또, 미로 밖으로 나갈 수는 없다.

  준규가 (N, M)으로 이동할 때, 가져올 수 있는 사탕 개수의 최댓값을 구하시오.

  [링크](https://www.acmicpc.net/problem/11048)
  
## 조건

  - 첫째 줄에 미로의 크기 N, M이 주어진다. (1 ≤ N, M ≤ 1,000)
  - 둘째 줄부터 N개 줄에는 총 M개의 숫자가 주어지며, r번째 줄의 c번째 수는 (r, c)에 놓여져 있는 사탕의 개수이다. 사탕의 개수는 0보다 크거나 같고, 100보다 작거나 같다.

## 입출력 예시
  
  |N|M|arr|result|
  |:--:|:--:|:--:|:--:|
  |3|4|[[1,2,3,4],[0,0,0,5],[9,8,7,6]]|31|
  |3|3|[[1,0,0],[0,1,0],[0,0,1]]|3|
  |4|3|[[1,2,3],[6,5,4],[7,8,9],[12,11,10]]|47|

## 풀이방식

  > DP를 사용한 풀이로 각 idx위치에서 [i-1][j], [i][j-1]을 비교하여 큰 값을 더함

  이전 값을 사용하는 대표적인 DP문제로 만약 인덱스가 [i][j]일때, "[i-1][j], [i][j-1]" 2개의 값을 비교하여 큰 값과 현재 인덱스 값을 더하여 계산한다. 

  이와 같은 과정을 반복하면 가장 많이 사탕을 얻으면서 이동할 수 있다. 단 이때 대각선의 이동은 제한하였는데, 대각선은 어차피 손해이기 때문이다.

  ```cpp
  #include <iostream>
  #include <vector>
  using namespace std;

  int main() {
    int n, m;
    cin >> n >> m;
    vector<vector<int>> Map(n, vector<int>(m));
    vector<vector<int>> MapVal(n, vector<int>(m, 0));

    // INIT
    for (int i = 0; i < n; i++) for (int j = 0; j < m; j++) cin >> Map[i][j];
    MapVal[0][0] = Map[0][0];
    for (int i = 1; i < m; i++) MapVal[0][i] = Map[0][i] + MapVal[0][i - 1];
    for (int i = 1; i < n; i++) MapVal[i][0] = Map[i][0] +  MapVal[i-1][0];

    // DP
    for (int i = 1; i < n; i++){
      for (int j = 1; j < m; j++) {
        MapVal[i][j] = (MapVal[i][j - 1] > MapVal[i - 1][j]) ? Map[i][j] + MapVal[i][j - 1]: Map[i][j] + MapVal[i-1][j];
      }
    }
    cout << MapVal[n - 1][m - 1];
    return 0;
  }
  ```
