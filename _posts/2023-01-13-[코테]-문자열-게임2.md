## 문제

  작년에 이어 새로운 문자열 게임이 있다. 게임의 진행 방식은 아래와 같다.

  1. 알파벳 소문자로 이루어진 문자열 W가 주어진다.
  2. 양의 정수 K가 주어진다.
  3. 어떤 문자를 정확히 K개를 포함하는 가장 짧은 연속 문자열의 길이를 구한다.
  4. 어떤 문자를 정확히 K개를 포함하고, 문자열의 첫 번째와 마지막 글자가 해당 문자로 같은 가장 긴 연속 문자열의 길이를 구한다.

  위와 같은 방식으로 게임을 T회 진행한다.

  [링크](https://www.acmicpc.net/problem/20437)
  
## 조건

  - 문자열 게임의 수 T가 주어진다. (1 ≤ T ≤ 100)
  - 다음 줄부터 2개의 줄 동안 문자열 W와 정수 K가 주어진다. (1 ≤ K ≤ |W| ≤ 10,000) 

## 입출력 예시
  
  |T|W|K|answer|
  |:--:|:--:|:--:|:--:|
  |2|["superaquatornado","abcdefghijklmnopqrstuvwxyz"]|[2,5]|[4,8],[-1]|
  |1|["abaaaba"]|[3]|[3,4]|

## 풀이방식

  > 어떤 문자를 K개 가지고 있으며, 양끝이 어떤 문자인 최대/최소 길이를 구하는 문제

  그래서 arr배열에 각 문자별로 인덱스 위치를 저장하였다. 이는 순서대로 저장되기 때문에 "arr[문자][인덱스 + (K + 1)] - arr[문자][인덱스]"의 계산을 통해 최대/최소 길이를 구한다.

  이때 최대길이가 0이라면, 해당 글자를 통해서 문자열을 구할 수 없다고 판단하고, -1을 반환한다.  

  ```cpp
  #include<iostream>
  #include<string>
  #include<vector>
  #include<limits.h>

  #define TMP 97
  using namespace std;

  int main() {
    int t, k;
    string w;
    cin >> t;

    for (int i = 0; i < t; i++){
      cin >> w >> k;
      vector<int> arr[26];

      // 각 문자배열의 인덱스 값을 저장
      for (int j = 0; j < w.size(); j++) arr[(w[j] - TMP)].push_back(j);

      // K개의 인덱스를 가지는 문자만 최대&최소 길이를 계산
      int minVal = INT_MAX , maxVal = 0;
      for (int j = 0; j < 26; j++) {
        if (arr[j].size() < k) continue;
        
        int lng = k - 1, val = 0;
        for (int h = 0; h < arr[j].size() - lng; h++) {
          val = (arr[j][h + lng] - arr[j][h]) + 1;
          minVal = min(minVal, val);
          maxVal = max(maxVal, val);
        }
      }

      // 출력 : 최대길이가 0이 아니라면...
      if(maxVal != 0) cout << minVal << " " << maxVal<<'\n';
      else cout << "-1\n";
    }
    
    return 0;
  }
  ```
