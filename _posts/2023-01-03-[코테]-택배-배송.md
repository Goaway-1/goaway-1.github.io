## 문제

  농부 현서는 농부 찬홍이에게 택배를 배달해줘야 합니다. 그리고 지금, 갈 준비를 하고 있습니다. 평화롭게 가려면 가는 길에 만나는 모든 소들에게 맛있는 여물을 줘야 합니다. 물론 현서는 구두쇠라서 최소한의 소들을 만나면서 지나가고 싶습니다.

  농부 현서에게는 지도가 있습니다. N (1 <= N <= 50,000) 개의 헛간과, 소들의 길인 M (1 <= M <= 50,000) 개의 양방향 길이 그려져 있고, 각각의 길은 C_i (0 <= C_i <= 1,000) 마리의 소가 있습니다. 소들의 길은 두 개의 떨어진 헛간인 A_i 와 B_i (1 <= A_i <= N; 1 <= B_i <= N; A_i != B_i)를 잇습니다. 두 개의 헛간은 하나 이상의 길로 연결되어 있을 수도 있습니다. 농부 현서는 헛간 1에 있고 농부 찬홍이는 헛간 N에 있습니다.

  다음 지도를 참고하세요.

  ```
           [2]---
          / |    \
         /1 |     \ 6
        /   |      \
     [1]   0|    --[3]
        \   |   /     \2
        4\  |  /4      [6]
          \ | /       /1
           [4]-----[5] 
                3  
  ```

  농부 현서가 선택할 수 있는 최선의 통로는 1 -> 2 -> 4 -> 5 -> 6 입니다. 왜냐하면 여물의 총합이 1 + 0 + 3 + 1 = 5 이기 때문입니다.

  농부 현서의 지도가 주어지고, 지나가는 길에 소를 만나면 줘야할 여물의 비용이 주어질 때 최소 여물은 얼마일까요? 농부 현서는 가는 길의 길이는 고려하지 않습니다.

  [링크](https://www.acmicpc.net/problem/20055)
  
## 조건

  - 첫째 줄에 N과 M이 공백을 사이에 두고 주어집니다.
  - 둘째 줄부터 M+1번째 줄까지 세 개의 정수 A_i, B_i, C_i가 주어집니다.

## 입출력 예시
  
  |N|M|arr|answer|
  |:--:|:--:|:--:|:--:|
  |6|8|[[4,5,3],[2,4,0],[4,1,4],[2,1,1],[5,6,1],[3,6,2],[3,2,6],[3,4,4]]|5|

## 풀이방식

### 1번 풀이방식 (타임아웃)

  > DFS를 활용한 풀이
  
  DFS로 풀었으나 역시 타임아웃 발생. 즉각적인 비교를 하지 않았기 때문..

  ```cpp
  #include<iostream>
  #include<vector>
  #include<limits.h>
  using namespace std;

  int N = 0, M = 0;
  vector<pair<int,int>> arr[50001];
  int ch[50001];
  int answer = INT_MAX;

  void DFS(int from,int val) {
    if(answer < val) return;

    if (from == N) answer = min(answer, val);
    else {
      // from에서부터 갈 수 있는 곳..
      for (int i = 0; i < arr[from].size(); i++){
        if (ch[arr[from][i].first] == 0) {
          ch[arr[from][i].first] = 1;
          DFS(arr[from][i].first, val + arr[from][i].second);
          ch[arr[from][i].first] = 0;
        }
      }
    }
  }

  int main(void){
    cin >> N >> M;

    // INIT 
    int a,b,c;
    for (int i = 0; i < M; i++){
      cin >> a >> b >> c;
      arr[a].push_back(make_pair(b,c));
      arr[b].push_back(make_pair(a,c));
    }

    ch[1] = 1;
    DFS(1,0);
   
    cout << answer;
    return 0;
  }
  ```

### 2번 풀이방식 (성공)

  > BFS를 활용한 풀이

  왔던 길을 지나갈 수 있으며, 대신 조건은 아래와 같다.
  
  - 처음 방문한 경우
  - 이미 방문한 경우 해당 노드에 적혀있는 경로의 값보다 현재 가지고 있는 경로의 값이 작으면 방문할 수 있다.

  ```cpp
  #include<iostream>
  #include<vector>
  #include<queue>
  using namespace std;

  int N = 0, M = 0;
  vector<pair<int,int>> arr[50001];
  int ch[50001];

  int main(void){
    cin >> N >> M;

    // INIT 
    int a,b,c;
    for (int i = 0; i < M; i++){
      cin >> a >> b >> c;
      arr[a].push_back(make_pair(b,c));
      arr[b].push_back(make_pair(a,c));
    }

    //BFS
    queue<int> Q;
    int x = 0, tmp = 0;
    Q.push(1);
    while (!Q.empty()) {
      x = Q.front();
      Q.pop();
      
      for (int i = 0; i < arr[x].size(); i++) {
        if(arr[x][i].first == 1) continue;
        
        tmp = ch[x] + arr[x][i].second;
        // 그 조건..
        if(ch[arr[x][i].first] == 0 || ch[arr[x][i].first] > tmp){
          ch[arr[x][i].first] = tmp;
          Q.push(arr[x][i].first);
        }
      }
    }
    cout << ch[N];
    return 0;
  }
  ```
