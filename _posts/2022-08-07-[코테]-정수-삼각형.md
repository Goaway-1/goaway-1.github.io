## 문제

  <img src="https://raw.githubusercontent.com/Goaway-1/goaway-1.github.io/master/_posts/images/inteagerTriangle.png" height="300" title="inteagerTriangle">

  위와 같은 삼각형의 꼭대기에서 바닥까지 이어지는 경로 중, 거쳐간 숫자의 합이 가장 큰 경우를 찾아보려고 합니다. 아래 칸으로 이동할 때는 대각선 방향으로 한 칸 오른쪽 또는 왼쪽으로만 이동 가능합니다. 예를 들어 3에서는 그 아래칸의 8 또는 1로만 이동이 가능합니다.

  삼각형의 정보가 담긴 배열 triangle이 매개변수로 주어질 때, 거쳐간 숫자의 최댓값을 return 하도록 solution 함수를 완성하세요.

## 조건

  - 삼각형의 높이는 1 이상 500 이하입니다.
  - 삼각형을 이루고 있는 숫자는 0 이상 9,999 이하의 정수입니다.

## 입출력 예시

  |triangle|result|
  |:--:|:--:|
  |[[7], [3, 8], [8, 1, 0], [2, 7, 4, 4], [4, 5, 2, 6, 5]]|30|
  
## 풀이방식

### 1번 풀이방식

  > 위에서부터 아래로 동적계획법을 적용한 풀이

  triangle[n][m]의 값은 __"triangle[n+1][m]와 triangle[n+1][m+1]"이다.__ 이를 활용하여 위의 값부터 아래로 이동해가며 계산. 하지만 불필요한 계산의 양이 많아 효율적이지 못함.
  
  ```cpp
  int solution(vector<vector<int>> triangle) {
    int answer = 0;
    vector<vector<int>> vec(triangle.size());

    for (int i = 0; i < triangle.size(); i++) {
      for (int j = 0; j < triangle[i].size(); j++) vec[i].push_back(0);
    }

    vec[0][0] = triangle[0][0];
    for (int i = 0; i < triangle.size() - 1; i++){
      for (int j = 0; j < triangle[i].size(); j++){
        int tmp = max(vec[i + 1][j], vec[i][j] + triangle[i + 1][j]);
        vec[i + 1][j] = tmp;

        tmp = max(vec[i + 1][j + 1], vec[i][j] + triangle[i + 1][j + 1]);
        vec[i + 1][j + 1] = tmp;
      }
    }
    for (int i = 0; i < vec[triangle.size() - 1].size(); i++) answer = max(answer, vec[triangle.size() - 1][i]);
    return answer;
  }
  ```

### 2번 풀이방식

  > 아래에서부터 위로 동적계획법을 적용한 풀이
  
  1번 풀이방법보다, 비교함수를 사용 빈도가 적기 때문에 시간복잡도에서 효율성을 보인다.

  triangle[n][m]의 값은 __"triangle[n+1][m]와 triangle[n+1][m+1]"를__ 비교하여 더 큰 값으로 지정하고, 최상단 [0][0]에 합산 중 가장 큰 값이 저장된다.

  ```cpp
  int solution(vector<vector<int>> triangle) {
    for (int i = triangle.size() - 2; i >= 0; i--){
      for (int j = 0; j < triangle[i + 1].size() - 1; j++){
        triangle[i][j] += (triangle[i + 1][j] > triangle[i + 1][j + 1]) ? triangle[i + 1][j] : triangle[i + 1][j + 1];
      }
    }

    return triangle[0][0];
  }
  ```
