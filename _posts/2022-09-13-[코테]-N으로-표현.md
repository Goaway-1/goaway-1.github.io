## 문제
  
아래와 같이 5와 사칙연산만으로 12를 표현할 수 있습니다.

12 = 5 + 5 + (5 / 5) + (5 / 5)
12 = 55 / 5 + 5 / 5
12 = (55 + 5) / 5

5를 사용한 횟수는 각각 6,5,4 입니다. 그리고 이중 가장 작은 경우는 4입니다.
이처럼 숫자 N과 number가 주어질 때, N과 사칙연산만 사용해서 표현 할 수 있는 방법 중 N 사용횟수의 최솟값을 return 하도록 solution 함수를 작성하세요.

## 조건

  - N은 1 이상 9 이하입니다.
  - number는 1 이상 32,000 이하입니다.
  - 수식에는 괄호와 사칙연산만 가능하며 나누기 연산에서 나머지는 무시합니다.
  - 최솟값이 8보다 크면 -1을 return 합니다.

## 입출력 예시

  |N|number|result|
  |:--:|:--:|:--:|
  |5|12|4|
  |2|11|3|
 
## 풀이방식

  ※ 본인의 풀이가 아닌 다른이의 풀이 방식이다. 본인은 아래 개선점에 적힌 사항때문에 막혀 풀지 못했다.

  > vector의 인덱스와 set을 활용한 풀이 (이때 인덱스는 N을 i개 써서 만들 수 있는 숫자를 의미한다.)

  숫자의 길이에 따라 다른 인덱스에 저장되며, 저장시 unordered_set을 활용하여 중복을 제거하였다. 그렇게 되면 vector에는 이전 값들을 사용하여 도출할 수 있는 값들이 저장된다. 한번의 과정을 마무리하고, find()메서드를 사용하여 number와 동일한 값이 있는지 검사 후 반복하거나 결과를 반환한다.

  1. vector타입의 DP는 여덟개의 인덱스를 갖으며 그 이상 넘어가면 -1을 반환한다.
  2. 첫 째자리 수부터 여덟번째 자리 수까지 반복하며, 만들 수 있는 경우의 수를 모두 찾아낸다.
      - 이때 사칙연산만을 사용하고, 중복은 제거된다.
  3. 현재 자리 수의 계산이 끝났을때, number가 존재한다면 반환한다.
 
  ```cpp
  #include <unordered_set>
  
  //N으로 연속된 수를 만드는 메서드
  int Make_Num(int N, int idx) {
    int result = N;
    for (int i = 1; i <= idx; i++) result = result * 10 + N;
    return result;
  }

  int solution(int N, int number) {
    if (N == number) return 1;

    // 1번 : i개의 N으로 만들 수 있는 연속된 숫자들 (set 활용)
    vector<unordered_set<int>> DP(8);

    DP[0].insert(N);

    // 2번 : 1 ~ 8번째 자리 수까지 반복
    for (int k = 1; k < 8; k++) {
      DP[k].insert(Make_Num(N, k)); 

      // 이전 값들에 대해 사칙 연산의 결과 삽입 (조건부)
      for (int i = 0; i < k; i++) {
        for (int j = 0; j < k; j++) {
          if (i + j + 1 != k) continue;

          // k로 만들 수 있는 경우를 구하기 위해서는 아래와 같다.
          // N1 N1 = k || N1 N2, N2 N1 = k || N1 N3, N2 N2, N3 N1 = k || ..... 
          for (int a : DP[i]) {
            for (int b : DP[j]) {
              DP[k].insert(a + b);
              DP[k].insert(a * b);
              if (a - b > 0) DP[k].insert(a - b);
              if (a / b > 0) DP[k].insert(a / b);
            }
          }
        }
      }
      if (DP[k].find(number) != DP[k].end()) return k + 1;
    }
    return -1;
  }
  ```

## 개선점
  - 문제에서 이상한 점을 발견했다. 주어지는 N과 사칙연산으로 number를 표현한다고 하였는데, N이 5일때 5를 이어붙어 55를 만들 수 있으나, 12일때 5를 이어붙일 수 없다.
