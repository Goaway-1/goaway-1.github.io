## 문제

  철수는 어떤 규칙에 의하여 숫자를 N개 썼습니다. 철수가 규칙에 의하여 쓴 숫자들의 총 합은 얼마인지 구하여 출력하는 프로그램을 작성하세요. 만약 철수가 다음과 같은 규칙으로 20개의 숫자를 썼다면 패턴은 32102이고 쓴 숫자의 총 합은 32입니다. 
  
  3210232102321……
  
  패턴은 처음 쓴 숫자부터 시작되며, 패턴의 길이는 최소 3이상이다. 그리고 반드시 존재한다.

## 조건
 
  - 패턴을 발견할 수 있는 num(? < num < 50)이 제공된다.
  - 어떤 규칙에 의하여 쓴 숫자 N(10<=N<=100,000,000)이 주어진다.

## 입출력 예시

  |num|N|result|
  |:--:|:--:|:--:|
  |3210232102321023|52|85|
  
## 풀이방식

  >  패턴을 구하고, N번 반복하녀 값을 구하면 된다.

  1. num에서 패턴을 파악하여 패턴을 구한다.
      - 동시에 패턴의 합을 구한다.
  2. N번 반복시 합을 구하기 위해서 ((N / 패턴 길이) * 패턴의 합)을 하고
      - 남은 N번 만큼 하나씩 더해준다.

  ```cpp
  int solution(string num,int n){	
    int i, sum = 0;
    
    // 1번 : num에서 패턴을 파악
    for(i=3; i<num.size(); i++){
      if(num.substr(0, i) == num.substr(i, i)){
        // 1-1번 : 합을 구함
        for(char x : num.substr(0, i)) {
          sum += (x - 48);
        }
        break;
      }
    }
    
    // 2번 : 합을 계산하기 위한 과정
    int answer = (sum * (n / i));
    int len = n % i;
    
    // 2-1번 : 남은 횟수 반복
    for(int i=0; i<len; i++) answer += (num[i] - 48);
    
    return answer;	
  }
  ```


## 개선점
  - 문제에 대해서 할말이 많다. 
    - 먼저 문제를 이해하기에 어렵다. 무슨 말인지 이해할 수 있겠는가?
    - 또한 넘겨오는 매개변수 또한 애매한 값이 첨부되어 있어있다. num으로 패턴을 구성할때 최소 3번은 반복되어 패턴으로 인정할 수 있다고 생각하는데, 만약 123412341이라면 1234로 끝나게 된다. 이 1234가 패턴이 아닐 수도 있다. 즉 조건으로 "num에서 패턴은 반드시 존재한다."고 했는데 이는 대충 예외를 막아버린 것이라고 생각한다.
    - 정답 풀이를 보면, 2번만 패턴이 반복되어도 인정으로 처리해버린다. 이게 맞을까?
  - 물론 아니라면 미안하다.
