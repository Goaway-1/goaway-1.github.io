## 문제
  
  자연수 N과 정수 K가 주어졌을 때 이항 계수를 구하는 프로그램을 작성하시오.

## 조건

  - 첫째 줄에 N과 K가 주어진다. (1 ≤ N ≤ 10, 0 ≤ K ≤ N)

## 입출력 예시

  |N|K|return|
  |:--:|:--:|:--:|
  |5|2|10|

## 풀이방식

  > 이전 값을 재사용할 수 있도록 배열 arr생성
  
  값의 범위가 10까지 밖에 안되기에 단순히 계산해도 되지만, 이전 값을 재사용할 수 있도록 배열(arr)를 참조하여 계산하도록 구현했다.

  cal()메서드는 val의 값에 대해 __"팩토리얼 값"을__ 계산한다. 이때의 값은 배열 arr에 저장되며, 추후 다시 사용할 수 있다.

  ```cpp
  #include <iostream> 
  using namespace std;

  // 인덱스에 맞는 팩토리얼의 값을 저장
  int arr[11];

  // 팩토리얼의 계산
  int cal(int val) {
    int answer = 1;
    for (int i = val; i > 1; i--) {
      if(arr[i] != 0) { 
        answer *= arr[i]; 
        break;
      }
      else answer *= i;
    }

    arr[val] = answer;
    return answer;
  }

  int main() {
    int n, k;
    cin >> n >> k;

    int answer = (cal(k) * cal(n-k));
    answer = cal(n) / answer;
    cout << answer;

    return 0;
  }
  ```
