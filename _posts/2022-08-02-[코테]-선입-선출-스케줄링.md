## 문제

  처리해야 할 동일한 작업이 n 개가 있고, 이를 처리하기 위한 CPU가 있습니다.

  이 CPU는 다음과 같은 특징이 있습니다.

    - CPU에는 여러 개의 코어가 있고, 코어별로 한 작업을 처리하는 시간이 다릅니다.
    - 한 코어에서 작업이 끝나면 작업이 없는 코어가 바로 다음 작업을 수행합니다.

  2개 이상의 코어가 남을 경우 앞의 코어부터 작업을 처리 합니다.
  처리해야 될 작업의 개수 n과, 각 코어의 처리시간이 담긴 배열 cores 가 매개변수로 주어질 때, 마지막 작업을 처리하는 코어의 번호를 return 하는 solution 함수를 완성해주세요.

## 조건

  - 코어의 수는 10,000 이하 2이상 입니다.
  - 코어당 작업을 처리하는 시간은 10,000이하 입니다.
  - 처리해야 하는 일의 개수는 50,000개를 넘기지 않습니다.


## 입출력 예시

  |n|cores|result|
  |:--:|:--:|:--:|
  |6|[1,2,3]|2|
  
## 풀이방식

  > 이분탐색을 사용하여 근사치까지 접근하고, 하나씩 반복

  말 그대로 이분탐색을 사용하여 최적의 mid값을 도출한다. 이때 mid값은 시간을 의미한다. (최소이기 때문에 left 사용)
  그 후 left를 cores의 각 인덱스 값으로 나누면 "그 시간(left)까지 가능한 횟수"가 도출된다. 그 후 처음부터 n과 같아질때까지 반복하면 원하는 결과를 얻을 수 있다.

  ```cpp
  int solution(int n, vector<int> cores) {
    int left = -1, right = 2e5, mid = 0;
    // 이분탐색을 사용하여 최적의 mid값을 도출
    while (left + 1 < right) {
      mid = (left + right) / 2;

      int cnt = cores.size();
      if (mid > 0) for (int i = 0; i < cores.size(); i++) cnt += mid / cores[i];

      if (cnt < n) left = mid;
      else right = mid;
    }

    if (left == -1) return n;

    // 현재 left에서 가능한 개수를 더하고, n과 같아질때까지 한개씩 반복.
    int cnt = cores.size();
    for (int i = 0; i < cores.size(); i++) cnt += left / cores[i];
    for (int i = 0; i < cores.size(); i++) {
      if ((left + 1) % cores[i] == 0) cnt++;
      if (cnt == n) return i + 1;
    }
    return 0;
  }
  ```

## 개선점
  - 초기에는 priority_queue를 사용했으나 시간초과 발생
