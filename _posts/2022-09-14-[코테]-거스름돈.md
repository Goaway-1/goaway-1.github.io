## 문제
  
  Finn은 편의점에서 야간 아르바이트를 하고 있습니다. 야간에 손님이 너무 없어 심심한 Finn은 손님들께 거스름돈을 n 원을 줄 때 방법의 경우의 수를 구하기로 하였습니다.

  예를 들어서 손님께 5원을 거슬러 줘야 하고 1원, 2원, 5원이 있다면 다음과 같이 4가지 방법으로 5원을 거슬러 줄 수 있습니다.

  - 1원을 5개 사용해서 거슬러 준다.
  - 1원을 3개 사용하고, 2원을 1개 사용해서 거슬러 준다.
  - 1원을 1개 사용하고, 2원을 2개 사용해서 거슬러 준다.
  - 5원을 1개 사용해서 거슬러 준다.
  
  거슬러 줘야 하는 금액 n과 Finn이 현재 보유하고 있는 돈의 종류 money가 매개변수로 주어질 때, Finn이 n 원을 거슬러 줄 방법의 수를 return 하도록 solution 함수를 완성해 주세요.

## 조건

  - n은 100,000 이하의 자연수입니다.
  - 화폐 단위는 100종류 이하입니다.
  - 모든 화폐는 무한하게 있다고 가정합니다.
  - 정답이 커질 수 있으니, 1,000,000,007로 나눈 나머지를 return 해주세요.

## 입출력 예시

  |N|money|result|
  |:--:|:--:|:--:|
  |5|[1,2,5]|4|
 
## 풀이방식

### 1번 풀이방식 (시간초과)

  > DFS를 사용한 풀이로 시간 초과 발생

  단순하기 때문에 설명은 생략
 
  ```cpp
  int answer = 0;
  int cache = 0;

  void DFS(vector<int> money, int cnt, int index) {
    if (cnt == cache) answer++;
    else {
      for (int i = index; i < money.size(); i++) {
        if (cnt + money[i] <= cache) DFS(money, cnt + money[i], i);
      }
    }
  }

  int solution(int n, vector<int> money) {
    cache = n;
    DFS(money, 0, 0);

    return answer % 1000000007;
  }
  ```

### 2번 풀이방식

  > 동적계획법을 사용한 풀이

  ch배열의 인덱스는 __"거스름돈이 인덱스의 크기 만큼있을때의 경우의 수를"__ 저장한다. 즉 이전의 값을 사용하여 다음의 값에 활용하는 동적계획법의 풀이이다. 즉, 가장 작은 단위의 동전을 사용했을 경우를 먼저 계산하고, 다음으로 큰 단위와 함께 사용했을때의 경우의 수를 계산한다. 지속적으로 잔돈의 크기를 키워가며 반복하면 결과를 도출할 수 있다.

  1. 먼저 0 ~ n까지 가장 작은 단위의 돈을 사용하여 만들 수 있는 경우의 수를 dp배열에 저장한다.
  2. 이중 for문을 사용하며, 현재 돈의 단위가 잔돈의 크기 보다 작을때, 실시한다.
      - i는 돈의 단위, j는 만들려는 잔돈의 크기를 의미한다.

  ```cpp
  int ch[100001];

  int solution(int n, vector<int> money) {
    ch[0] = 1;

    // 1번 : 가장 작은 단위 돈을 사용하여 만드는 경우의 수는 모두 1개
    for (int i = money[0]; i <= n; i += money[0]) ch[i] = 1;

    // 2번 : 동적계획법
    for (int i = 1; i < money.size(); i++) {
      for (int j = 0; j <= n; j++) {
        if (j >= money[i]) ch[j] += ch[j - money[i]] % 1000000007;
      }
    }
    return ch[n];
  }
  ```

## 개선점
  - 난 동적계획법을 더럽게 못 푼다는 단점이 있다.
