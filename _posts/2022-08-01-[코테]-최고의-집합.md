## 문제

  자연수 n 개로 이루어진 중복 집합(multi set, 편의상 이후에는 "집합"으로 통칭) 중에 다음 두 조건을 만족하는 집합을 최고의 집합이라고 합니다.

  1. 각 원소의 합이 S가 되는 수의 집합
  2. 위 조건을 만족하면서 각 원소의 곱 이 최대가 되는 집합

  예를 들어서 자연수 2개로 이루어진 집합 중 합이 9가 되는 집합은 다음과 같이 4개가 있습니다.
  { 1, 8 }, { 2, 7 }, { 3, 6 }, { 4, 5 }
  그 중 각 원소의 곱이 최대인 { 4, 5 }가 최고의 집합입니다.

  집합의 원소의 개수 n과 모든 원소들의 합 s가 매개변수로 주어질 때, 최고의 집합을 return 하는 solution 함수를 완성해주세요.

## 조건

  - 최고의 집합은 오름차순으로 정렬된 1차원 배열(list, vector) 로 return 해주세요.
  - 만약 최고의 집합이 존재하지 않는 경우에 크기가 1인 1차원 배열(list, vector) 에 -1 을 채워서 return 해주세요.
  - 자연수의 개수 n은 1 이상 10,000 이하의 자연수입니다.
  - 모든 원소들의 합 s는 1 이상, 100,000,000 이하의 자연수입니다.

## 입출력 예시

  |n|s|result|
  |:--:|:--:|:--:|
  |2|9|[4,5]|
  |2|1|[-1]|
  |2|8|[4,4]|

## 풀이방식

  > N개의 원소 간의 차이가 작아야 최대의 곱을 도출할 수 있다.

  본인은 "각 원소의 합이 S가 되는 수의 집합"보다는 "최대가 되는 집합"에 먼저 초점을 두고 풀이를 시작했다. 위와 같은 결과가 도출되었고 이에 따른 로직을 작성하였다.

  원소들은 중복이 가능하기 때문에 먼저 s를 n으로 나누어 최소 값을 도출한다. 만약 그 값(cnt)가 4라면 원소들은 모두 4로 구성된다. 그리고 나머지 값(RemainCnt)을 구하여 이 크기만큼 answer의 마지막 위치부터 1씩 더해준다.

  ```cpp
  vector<int> solution(int n, int s) {
    vector<int> answer;
    int cnt = s / n;            //나눌 수 있는 개수
    int RemainCnt = s % n;       //나머지 값

    if (cnt == 0) {
      answer.push_back(-1);
      return answer;
    }
    else {
      int tmp = n - RemainCnt;
      for (int i = 0; i < n; i++) {
        if (i >= tmp) answer.push_back(cnt+1);
        else answer.push_back(cnt);
      }
    }
    return answer;
  }
  ```

## 개선점
  - 처음에 봤을때는 어떤 방식으로 풀어야할지 답답했다. 하지만 딱봐도 DFS와 BFS를 통한 노가다는 아니라고 생각했다.
    - 그래서 수학적인 방법으로 접근했고, 내가 도출해낸 결과는 <span style = "color:orange;">"N개의 원소간의 값 차이가 작아야 곱이 최대가 된다는 점"</span>이였다.
  - 다른이의 풀이 또한 비슷하다.
