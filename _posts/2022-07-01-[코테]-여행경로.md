## 문제

  주어진 항공권을 모두 이용하여 여행경로를 짜려고 합니다. 항상 "ICN" 공항에서 출발합니다.

  항공권 정보가 담긴 2차원 배열 tickets가 매개변수로 주어질 때, 방문하는 공항 경로를 배열에 담아 return 하도록 solution 함수를 작성해주세요.

## 조건
  
  - 모든 공항은 알파벳 대문자 3글자로 이루어집니다.
  - 주어진 공항 수는 3개 이상 10,000개 이하입니다.
  - tickets의 각 행 [a, b]는 a 공항에서 b 공항으로 가는 항공권이 있다는 의미입니다.
  - 주어진 항공권은 모두 사용해야 합니다.
  - 만일 가능한 경로가 2개 이상일 경우 알파벳 순서가 앞서는 경로를 return 합니다.
  - 모든 도시를 방문할 수 없는 경우는 주어지지 않습니다.
 

## 입출력 예시

  |tickets|return|
  |:--:|:--:|
  |[["ICN", "JFK"], ["HND", "IAD"], ["JFK", "HND"]]|["ICN", "JFK", "HND", "IAD"]|
  |[["ICN", "SFO"], ["ICN", "ATL"], ["SFO", "ATL"], ["ATL", "ICN"], ["ATL","SFO"]]|["ICN", "ATL", "ICN", "SFO", "ATL", "SFO"]|  

## 풀이방식

### 나의 풀이방식

  DFS에 Stack을 활용한 풀이 방식으로, DFS의 구조는 __"현재 공항", "티켓", "현재 사용한 티켓의 수"__ 이다.

  <span style = "color:orange;">즉, 알고리즘은 아래와 같다.</span>

  1. 사용하지 않은 티켓(ch == 0)이고, 현재 위치(curPos)에서 출발 가능한 티켓을 찾는다.
  2. Stack에 도착 가능한 여행지를 넣는 과정을 반복한다.
  3. 티켓을 모두 사용했다면 stack의 내용을 모두 vector로 옮긴 후에 min()메서드를 사용하여 알파벳이 작은 순서를 사용한다.

  ```cpp
  #include <stack> 
  
  int ch[100000];
  stack<string> St;

  vector<string> answer;

  void DFS(string curPos, vector<vector<string>> tickets, int cnt) {
    if (cnt == tickets.size()) {
      stack<string> tmp = St;
      vector<string> arr(St.size());
      int si  = St.size() - 1;

      // stack to array
      while (!tmp.empty()){
        arr[si--] = tmp.top();
        tmp.pop();
      }
          
      // check
      if(answer.size() == 0) answer = arr;
      else answer = min(answer, arr);
    }
    else {
      for (int i = 0; i < tickets.size(); i++){
        if (ch[i] == 0 && curPos == tickets[i][0]) {
          ch[i] = 1;
          St.push(tickets[i][1]);

          DFS(tickets[i][1], tickets,cnt + 1);

          St.pop();
          ch[i] = 0;
        }
      }
    }
  }
  vector<string> solution(vector<vector<string>> tickets) {
    St.push("ICN");
    DFS("ICN", tickets, 0);
    return answer;
  }
  ```

### 다른이의 풀이 방식

  나의 방식과 동일하며 stack을 사용하냐, string을 사용하냐의 차이이다. 물론 이 방식이 더 효율적이다.

  이는 단순히 거처간 공항을 string에 이어붙히고, 이전 경로와 비교하여 알파벳에 따라 분류를 한것이다. string에 단순히 이어붙혔기 때문에 3자씩 끊어서 answer 배열에 추가하여 값을 내보낸다.

  ```c++
  int visited[1000000];
  string ans_string = "a";

  void dfs(vector<vector<string>> &tickets, string cur, string path, int depth) {
    if (depth == tickets.size()) {
      string p = path;
      if (path < ans_string) {
        ans_string = path;
      }
      return;
    }

    for (int i = 0; i < tickets.size(); i++) {
      if (cur == tickets[i][0] && !visited[i]) {
        visited[i] = 1;
        dfs(tickets, tickets[i][1], path + tickets[i][1], depth+1);
        visited[i] = 0;
      }
    }
  }

  vector<string> solution(vector<vector<string>> tickets) {
    vector<string> answer;
    dfs(tickets, "ICN", "ICN", 0);
    for (int i = 0; i < ans_string.size(); i += 3) {
      answer.push_back(ans_string.substr(i, 3));
    }
    return answer;
  }
  ```

## 개선점
  - 나의 풀이 방식은 먼저 stack을 사용하고 비우는 과정에서 많은 메모리를 소요한다는 단점이 있다. 다른 이가 푼 방식을 참조하면 좋을것같다.
