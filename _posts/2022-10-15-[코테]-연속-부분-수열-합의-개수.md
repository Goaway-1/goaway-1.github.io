## 문제

  철호는 수열을 가지고 놀기 좋아합니다. 어느 날 철호는 어떤 자연수로 이루어진 원형 수열의 연속하는 부분 수열의 합으로 만들 수 있는 수가 모두 몇 가지인지 알아보고 싶어졌습니다. 원형 수열이란 일반적인 수열에서 처음과 끝이 연결된 형태의 수열을 말합니다.

  원형 수열은 처음과 끝이 연결되어 끊기는 부분이 없기 때문에 연속하는 부분 수열도 일반적인 수열보다 많아집니다.
  
  원형 수열의 모든 원소 elements가 순서대로 주어질 때, 원형 수열의 연속 부분 수열 합으로 만들 수 있는 수의 개수를 return 하도록 solution 함수를 완성해주세요.

## 조건

  - 3 ≤ elements의 길이 ≤ 1,000
  - 1 ≤ elements의 원소 ≤ 1,000
  
## 입출력 예시
  
  |elements|result|
  |:--:|:--:|
  |[7,9,1,1,4]|18|

## 풀이방식

  > __"set"을__ 활용하여 중복을 제거한 풀이

  연속 부분의 크기가 1 ~ N일때까지의 값을 반복문을 통해서 계산하고, set을 활용하여 중복을 제거하여 set의 길이를 반환하는 로직이다.

  - 연속 부분의 크기가 1 ~ N일때를 계산하며, 순차적으로 0...N을 빼고, (0 + 연속부분길이)...(N + 연속부분의 길이)을 더한다.

  ```cpp
  #include <string>
  #include <vector>
  #include <set>
  using namespace std;

  int solution(vector<int> elements) {
    int answer = 0;
    set<int> S;

    // 1번 : 연속 부분의 크기가 1일때
    for (int i = 0; i < elements.size(); i++) S.insert(elements[i]);

    // 2번 : 연속 부분의 크기가 2 ~ (N-1)일때
    for (int i = 2; i < elements.size(); i++){
      int sum = 0;

      // INIT
      for (int j = 0; j < i; j++) sum += elements[j];
      S.insert(sum);

      for (int j = 0; j < elements.size() - 1; j++){
        sum -= elements[j];
        sum += elements[(j + i) % elements.size()];
        S.insert(sum);
      }
    }

    // 3번 : 연속 부분의 크기가 N일때
    int sum = 0;
    for (int i = 0; i < elements.size(); i++) sum += elements[i];
    S.insert(sum);

    return S.size();
  }
  ```

### 다른이의 풀이

  > 내 풀이와 비슷한 로직이나 훨씬 단순

  하지만 시간 복잡도에서는 불리하다.

  ```cpp
  int solution(vector<int> elements) {
    set<int> S;

    int n = elements.size();

    for (int i = 0 ; i < n ; ++i) {
      int sum = 0;
      for (int j = i ; j < i + n ; ++j) {
        sum += elements[j % n];
        S.insert(sum);
      }
    }

    return S.size();
  } 
  ```
