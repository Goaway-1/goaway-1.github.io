## 문제

  숫자와 문자가 섞여있는 문자열이 주어지면 숫자만을 추출하여 그 합을 구하는 프로그램을 작 성하세요. 문자열에서 연속된 숫자는 하나의 숫자로 간주합니다. 연속된 숫자는 5자리를 넘지 않으며 연속된 숫자의 제일 앞자리의 0은 무시합니다. 만약 문자열에서 연속된 숫자가 "050" 이라면 50입니다.

## 조건

  - 첫 번째 줄에 길이 100을 넘지 않는 문자열 S가 주어진다.
  - 문자열은 대소문자와 0-9의 숫자로 이루어져 있다.

## 풀이방식

### 1번 풀이방식 

  단순히 문자가 아닌 숫자를 만났을때 string에 한자리씩 추가하며, 문자를 만나면 answer에 int형으로 변환하여 더함.

    ```cpp
    #include <iostream>
    #include <string>

    int main(){
      string s;
      int answer = 0;
      cin >> s;
      
      string tmp = "";
      for(int i=0;i < s.size();i++){
        if(s[i] >='0' && s[i] <='9') {
          if(tmp.size() != 0 || s[i] != '0') tmp += s[i];
        }
        else if(tmp.size() > 0){
          answer += stoi(tmp);
          tmp = "";
        }
      }

      if(tmp.size() != 0) answer += stoi(tmp);
      
      cout << answer;
      return 0;
    }
    ```
