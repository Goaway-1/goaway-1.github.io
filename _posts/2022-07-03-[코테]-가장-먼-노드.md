## 문제

  n개의 노드가 있는 그래프가 있습니다. 각 노드는 1부터 n까지 번호가 적혀있습니다. 1번 노드에서 가장 멀리 떨어진 노드의 갯수를 구하려고 합니다. 가장 멀리 떨어진 노드란 최단경로로 이동했을 때 간선의 개수가 가장 많은 노드들을 의미합니다.

  노드의 개수 n, 간선에 대한 정보가 담긴 2차원 배열 vertex가 매개변수로 주어질 때, 1번 노드로부터 가장 멀리 떨어진 노드가 몇 개인지를 return 하도록 solution 함수를 작성해주세요.

## 조건
  
  - 노드의 개수 n은 2 이상 20,000 이하입니다.
  - 간선은 양방향이며 총 1개 이상 50,000개 이하의 간선이 있습니다.
  - vertex 배열 각 행 [a, b]는 a번 노드와 b번 노드 사이에 간선이 있다는 의미입니다.
 
## 입출력 예시

  |n|vertex|return|
  |:--:|:--:|:--:|
  |6|[[3, 6], [4, 3], [3, 2], [1, 3], [1, 2], [2, 4], [5, 2]]|3|

## 풀이방식

  BFS와 Queue를 혼합하여 사용한 방식으로 단순히 노드1을 0으로 하고, 한 칸씩 갈 수록 +1해주었다.

  ```cpp
  #include <queue> 
  #include <algorithm>

  int solution(int n, vector<vector<int>> edge) {
    int answer = 0;
    vector<int> distance(n + 1);
    queue<int> Q;
    Q.push(1);
    distance[1] = 1;

    while (!Q.empty()){
      int tmp = Q.front();
      Q.pop();

      for (auto& edge_at : edge) {
        if (tmp == edge_at[0] && tmp == edge_at[1]) {
          int other_node = 0;
          if (tmp == edge_at[0])  other_node = edge_at[1];
          else other_node = edge_at[0];

          if (distance[tmp] == 0 || distance[other_node] != 0) continue; 

          distance[other_node] = distance[tmp] + 1;
          Q.push(other_node);
        }
      }
    }

    int max_distance = *max_element(distance.begin(), distance.end());
    for (auto& dist : distance) if (max_distance == dist) answer++;

    return answer;
  }
  ```

## 개선점
  - 보자마자 큐를 사용한 BFS를 떠올렸지만, 그래프 문제이기에 __인접 리스트를__ 사용한 풀이 방식으로 진행했다. 테스트 케이스 10개는 모두 풀리는데 최종은 안풀려서 그냥 BFS를 사용했다.
