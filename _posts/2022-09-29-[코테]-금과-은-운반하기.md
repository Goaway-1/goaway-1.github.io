## 문제

  어느 왕국에 하나 이상의 도시들이 있습니다. 왕국의 왕은 새 도시를 짓기로 결정하였습니다. 해당 도시를 짓기 위해서는 도시를 짓는 장소에 금 a kg과 은 b kg이 전달되어야 합니다.

  각 도시에는 번호가 매겨져 있는데, i번 도시에는 금 g[i] kg, 은 s[i] kg, 그리고 트럭 한 대가 있습니다. i번 도시의 트럭은 오직 새 도시를 짓는 건설 장소와 i번 도시만을 왕복할 수 있으며, 편도로 이동하는 데 t[i] 시간이 걸리고, 최대 w[i] kg 광물을 운반할 수 있습니다. (광물은 금과 은입니다. 즉, 금과 은을 동시에 운반할 수 있습니다.) 모든 트럭은 같은 도로를 여러 번 왕복할 수 있으며 연료는 무한대라고 가정합니다.

  정수 a, b와 정수 배열 g, s, w, t가 매개변수로 주어집니다. 주어진 정보를 바탕으로 각 도시의 트럭을 최적으로 운행했을 때, 새로운 도시를 건설하기 위해 금 a kg과 은 b kg을 전달할 수 있는 가장 빠른 시간을 구해 return 하도록 solution 함수를 완성해주세요.

## 조건

  - 0 ≤ a, b ≤ 109
  - 1 ≤ g의 길이 = s의 길이 = w의 길이 = t의 길이 = 도시 개수 ≤ 105
    - 0 ≤ g[i], s[i] ≤ 109
    - 1 ≤ w[i] ≤ 102
    - 1 ≤ t[i] ≤ 105
    - a ≤ g의 모든 수의 합
    - b ≤ s의 모든 수의 합


## 입출력 예시

  |a|b|g|s|w|t|result|
  |:--:|:--:|:--:|:--:|:--:|:--:|:--:|
  |10|10|[100]|[100]|[7]|[10]|50|
  |90|500|[70,70,0]|[0,0,500]|[100,100,2]|[4,8,1]|499|

## 풀이방식

  > 시간에 대해 이분탐색을 활용했고, 그 시간 내에 총 얻을 수 있는 각각의 자원량을 모두 만족하는지 확인

  먼저 내 풀이는 아니다. 초기 나는 이분탐색과 DFS의 조합을 사용하여 풀생각이였다. 말도 안되지 무슨 ㅎㅋ

  무튼 위에서 설명한 것처럼 이분탐색을 사용한 풀이이며, 시간에 대한 이분탐색이다. 시간이 주어졌을때, gold와 silver를 모두 옮길 수 있냐 없냐로 구분된다. 여기서 주목해야할 점은 <ins>굳이 차량의 순서는 알 필요 없다는 점이다.</ins> 그냥 결과만 보면된다. 차량의 순서를 알기위해서라면 DFS를 사용했을듯싶다. 

  무튼 gold와 silver를 모두 옮길 수 있는지에 대한 확인은 isSucceed()메서드를 통해 진행되며, <ins>__"도시 별로 시간 내에 왕복할 수 있는 최대 횟수를" MoveCnt에 저장하고, 각 광물의 양을 합산하여 계산한다.__</ins>

  ```cpp
  #include <iostream> 
  #include <vector> 
  #include <algorithm>
  using namespace std;

  bool isSucceed(int a, int b, vector<int> g, vector<int> s, vector<int> w, vector<int> t, long long Mid) {
    long long Gold = 0, Silver = 0, Mineral = 0;

    for (int i = 0; i < g.size(); i++)	{
      long long Time = t[i];
      long long Round_Time = Time * 2;			    // 왕복 시간
      long long MoveCnt = Mid / Round_Time;		  // 한 차량이 광물을 옮길 수 있는 횟수
      if(Mid % Round_Time >= Time) MoveCnt++;		// 편도 가능 여부 확인

      long long MaxTake = w[i] * MoveCnt;			  // 최대 광물의 양

      Gold += min((long long)g[i], MaxTake);
      Silver += min((long long)s[i], MaxTake);
      Mineral += min((long long)g[i] + s[i], MaxTake);
    }

    // 모든 경우를 계산해 보았을때, 모두 만족한다면 MID라는 시간은 가능하다.
    if(Gold >= a && Silver >= b && Mineral >= a + b) return true;
    return false;
  }

  long long solution(int a, int b, vector<int> g, vector<int> s, vector<int> w, vector<int> t) {
    long long answer = 10e14 * 3, lt = 0, Mid = 0 rt = 10e14 * 3;

    // 이분 탐색 
    while(lt <= rt){
      Mid = (lt + rt) / 2;
      if(isSucceed(a,b,g,s,w,t,Mid)){
        answer = min(answer, Mid);
        end = Mid - 1;
      }
      else start = Mid + 1;
    }

    return answer;
  }
  ```
