## 문제

  N개의 수로 된 수열 A[1], A[2], …, A[N] 이 있다. 이 수열의 i번째 수부터 j번째 수까지의 합 A[i] + A[i+1] + … + A[j-1] + A[j]가 M이 되는 경우의 수를 구하는 프로그램을 작성하시오.

  [링크](https://www.acmicpc.net/problem/2003)
  
## 조건

  - 첫째 줄에 N(1 ≤ N ≤ 10,000), M(1 ≤ M ≤ 300,000,000)이 주어진다. 
  - 다음 줄에는 A[1], A[2], …, A[N]이 공백으로 분리되어 주어진다. 각각의 A[x]는 30,000을 넘지 않는 자연수이다.
  - 첫째 줄에 경우의 수를 출력한다.
    
## 입출력 예시
  
  |N|M|arr|answer|
  |:--:|:--:|:--:|:--:|
  |4|2|[1,1,1,1]|3|
  |10|5|[1,2,3,4,2,5,3,1,1,2]|3|
  

## 풀이방식

  나는 한 칸씩 앞으로 나아가며, 값을 더하고 뺏는데, 다른이는 각 위치에 모든 총합을 구해놓고 풀더라. 그 방식이 더 좋은듯

  ```cpp
  #include <iostream>
  #include <string>
  #include <algorithm>
  using namespace std;

  int arr[30001] = {0};

  int main()
  {
    int n,m, answer = 0;
    cin >> n >> m;

    for (int i = 1; i <= n; i++)
    {
      cin >> arr[i];
    }

    // Caluate
    int lt = 0, rt = 0, val = 0;
    while (true)
    {
      if (rt < n && val < m) val += arr[++rt];
      else if (lt < n) val -= arr[++lt];

      if (val == m) 
      {
        val -= arr[++lt];
        answer++;
      }

      if(lt >= n && rt >= n) break;
    }
    cout << answer;
  }
  ```

## 다른이의 풀이

  ```c++
  #include <iostream>

  int N, M;
  int sum[10001];

  int main() {
    std::cin >> N >> M;

    for (int i = 1; i < N + 1; i++) {
      std::cin >> sum[i];
      sum[i] += sum[i - 1];
    }

    int ans = 0;
    int left = 0, right = 1;
    while (left <= right) {
      int tmp = sum[right] - sum[left];
      if (tmp == M) {
        ans++;
      }
      if (tmp <= M) {
        if (right < N) {
          right++;
        } 
        else {
          left++;
        }
      }
      else {
        left++;
      }
    }
    std::cout << ans << "\n";
    return 0;
  }
  ```
