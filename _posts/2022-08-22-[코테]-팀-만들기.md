## 문제

  현수는 바둑팀 감독입니다. 현수의 팀에는 바둑선수가 9명 있습니다. 현수는 대회에 출전하기 위해 9명의 선수를 3명씩 3개의 팀으로 만들어서 대회에 출전하려고 합니다. 9명의 바둑선수 는 각각 정수로 표현되는 바둑실력을 가지고 있습니다. 현수가 최상의 팀을 만드는 방법은 9 명을 3팀으로 만들었을 때 가장 실력이 높은 팀과 가장 실력이 낮은 팀의 실력차가 가장 작도 록 해야 합니다. 각 팀의 실력은 해당 팀의 선수들의 실력을 모두 합한 값입니다. 9명의 바둑실력이 주어지면 현수가 최상의 팀을 만들었을 때 가장 실력이 높은 팀과 가장 낮 은 팀의 실력차를 출력하는 프로그램을 작성하세요.

## 조건
  - 9명의 실력(1~100)이 저장된 배열 score가 주어집니다. 

## 입출력 예시

  |score|result|
  |:--:|:--:|
  |[1,2,3,4,5,6,7,8,9]|0|
  
## 풀이방식

  > DFS를 사용하여 팀을 나누고, 조건 만족 시 합산을 구하는 방식

  __"arr배열"에는__ 각각 3개의 팀을 구성할때, 현재 배정할 수 있는 팀원의 자리이다. 즉, 0이면 팀원으로 구성할 수 없고, 1이면 1명을 더 팀원으로 구성할 수 있다는 뜻이다. __"ch배열"는__ 인덱스 번호별로 3개의 팀으로 나눠진 정보가 저장된다. 

  단순 DFS를 통해서 팀을 나누는 과정이기에 자세한 설명은 생략합니다.
  
  ```cpp
  // 팀을 구별하기 위한 배열 
  vector<int> ch(9,0);	

  // 팀에 속할 수 있는 카운트를 저장한 배열 
  vector<int> arr = {3,3,3};	

  int answer = INT_MAX;

  void DFS(vector<int> n, int idx){
    if(idx == 9){
      vector<int> sum(3,0);
      for(int i=0;i<9;i++) sum[ch[i]] += n[i];
      
      // max와 min의 차이 값 
      int tmp = (max(sum[0],max(sum[1],sum[2])) - min(sum[0],min(sum[1],sum[2])));
      answer = min(answer, tmp);
    }
    else{
      for(int i=0;i<3;i++){
        if(arr[i] > 0) {
          ch[idx] = i;
          
          arr[i]--;
          DFS(n, idx+1);
          arr[i]++;
        }
      }
    }
  }

  int solution(vector<int> n){	
    DFS(n,0);
    
    return answer;
  }
  ```
