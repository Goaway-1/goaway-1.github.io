## 문제

  한수는 크기가 2N × 2N인 2차원 배열을 Z모양으로 탐색하려고 한다. 예를 들어, 2×2배열을 왼쪽 위칸, 오른쪽 위칸, 왼쪽 아래칸, 오른쪽 아래칸 순서대로 방문하면 Z모양이다.

  ![n == 1](https://upload.acmicpc.net/21c73b56-5a91-43aa-b71f-9b74925c0adc/-/preview/)

  N > 1인 경우, 배열을 크기가 2N-1 × 2N-1로 4등분 한 후에 재귀적으로 순서대로 방문한다.

  다음 예는 22 × 22 크기의 배열을 방문한 순서이다.
  
  ![n == 2](https://upload.acmicpc.net/adc7cfae-e84d-4d5c-af8e-ee011f8fff8f/-/preview/)

  N이 주어졌을 때, r행 c열을 몇 번째로 방문하는지 출력하는 프로그램을 작성하시오.

  다음은 N=3일 때의 예이다.
  
  ![n == 3](https://upload.acmicpc.net/d3e84bb7-9424-4764-ad3a-811e7fcbd53f/-/preview/)

  [링크](https://www.acmicpc.net/problem/1074)
  
## 조건

  - 첫째 줄에 정수 N, r, c가 주어진다.
  - 1 ≤ N ≤ 15
  - 0 ≤ r, c < 2N

## 입출력 예시
  
  |N|r|c|result|
  |:--:|:--:|:--:|:--:|
  |2|3|1|11|
  |3|7|7|63|
  |1|0|0|0|
  |4|7|7|63|
  |10|511|511|262143|
  |10|512|512|786432|

## 풀이방식

  > 재귀를 통해서 해당 분면을 지속적으로 4등분하는 과정을 반복한다.

  ```cpp
  #include <iostream>
  using namespace std;

  int n, r, c;
  int ans;

  // size == 한변의 길이
  void Z(int y, int x, int size){
    if (y == r && x == c){
      cout << ans;
      return;
    }

    // r,c가 현재 사분면에 존재한다면
    if (r < y + size && r >= y && c < x + size && c >= x) {
      // 1,2,3,4 사분면 탐색
      Z(y, x, size / 2);
      Z(y, x + size / 2, size / 2);
      Z(y + size / 2, x, size / 2);
      Z(y + size / 2, x + size / 2, size / 2);
    }
    else ans += size * size;
  }
  int main(){
    cin >> n >> r >> c;
    Z(0, 0, (1 << n));
    return 0;
  }
  ```
