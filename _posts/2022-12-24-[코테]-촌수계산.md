## 문제

  우리 나라는 가족 혹은 친척들 사이의 관계를 촌수라는 단위로 표현하는 독특한 문화를 가지고 있다. 이러한 촌수는 다음과 같은 방식으로 계산된다. 기본적으로 부모와 자식 사이를 1촌으로 정의하고 이로부터 사람들 간의 촌수를 계산한다. 예를 들면 나와 아버지, 아버지와 할아버지는 각각 1촌으로 나와 할아버지는 2촌이 되고, 아버지 형제들과 할아버지는 1촌, 나와 아버지 형제들과는 3촌이 된다.

  여러 사람들에 대한 부모 자식들 간의 관계가 주어졌을 때, 주어진 두 사람의 촌수를 계산하는 프로그램을 작성하시오.

  [링크](https://www.acmicpc.net/problem/2644)
  
## 조건

  - 사람들은 1, 2, 3, …, n (1 ≤ n ≤ 100)의 연속된 번호로 각각 표시된다. 
    - 입력 파일의 첫째 줄에는 전체 사람의 수 n이 주어지고, 
  - 둘째 줄에는 촌수를 계산해야 하는 서로 다른 두 사람의 번호가 주어진다. 
  - 셋째 줄에는 부모 자식들 간의 관계의 개수 m이 주어진다. 
  - 넷째 줄부터는 부모 자식간의 관계를 나타내는 두 번호 x,y가 각 줄에 나온다. 
    - 이때 앞에 나오는 번호 x는 뒤에 나오는 정수 y의 부모 번호를 나타낸다.
  - 각 사람의 부모는 최대 한 명만 주어진다.

## 입출력 예시
  
  |N|N1|N2|M|arr|result|
  |:--:|:--:|:--:|:--:|:--:|:--:|
  |9|7|3|7|[[1,2],[1,3],[2,7],[2,8],[2,9],[4,5],[4,6]]|3|
  |9|8|6|7|[[1,2],[1,3],[2,7],[2,8],[2,9],[4,5],[4,6]]|-1|

## 풀이방식

  > 각 번호에서 갈 수 있는 번호를 2차원 배열을 사용하여 풀이

  가장 먼저 트리의 구조가 생각났는데, Map외에 다른 트리가 생각나지 않았고, 원하는 위치에 삽입하는 것이 불가능하다고 생각해서 각 배열에 추가로 배열을 Push하는 방법을 사용

  즉, 아래 그림과 같이 2차원 배열에 동적할당을 사용하여 풀이한다. N1에 해당하는 인덱스에서 출발하여 N2에 도찰할때까지 BFS한다.

  ![degree_of_kinship](https://raw.githubusercontent.com/Goaway-1/goaway-1.github.io/master/_posts/images/degree_of_kinship.png)

  ```cpp
  #include <iostream>
  #include <vector>
  #include <algorithm>
  #include <queue>

  using namespace std;

  int main(){
    queue<int> Q;
    bool isDone = false;
    int n,m, n1, n2,x,y;
    cin >> n >> n1 >> n2 >> m;
    vector<vector<int>> arr(n + 1);   // 
    vector<int> ch(n + 1,0);          // N1기준 촌수의 값을 저장

    // INIT : x(부모), y(자식)
    for (int i = 0; i < m; i++){
      cin >> x >> y; 
      arr[x].push_back(y);
      arr[y].push_back(x);
    }

    // BFS
    Q.push(n1);
    ch[n1] = 0; 
    while (!isDone && !Q.empty()) {
      int idx = Q.front();
      Q.pop();
        
      for (int i = 0; i < arr[idx].size(); i++){
        if(ch[arr[idx][i]] == 0) {
          Q.push(arr[idx][i]);
          ch[arr[idx][i]] = ch[idx] + 1;
        }
        if(arr[idx][i] == n2){
          isDone = true;
          break;
        }
      }
    }

    if(isDone) cout << ch[n2];
    else cout << -1;

    return 0;
  }
  ```
