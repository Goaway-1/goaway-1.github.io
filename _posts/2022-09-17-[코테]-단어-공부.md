## 문제
  
  ※ 백준 (solved.ac)의 Class 1의 문제로 그나마 헷갈렸던 문제. Class 10까지 풀것이다.

  알파벳 대소문자로 된 단어가 주어지면, 이 단어에서 가장 많이 사용된 알파벳이 무엇인지 알아내는 프로그램을 작성하시오. 단, 대문자와 소문자를 구분하지 않는다. 첫째 줄에 이 단어에서 가장 많이 사용된 알파벳을 대문자로 출력한다. 단, 가장 많이 사용된 알파벳이 여러 개 존재하는 경우에는 ?를 출력한다.

## 조건

  - 첫째 줄에 알파벳 대소문자로 이루어진 단어가 주어진다. 주어지는 단어의 길이는 1,000,000을 넘지 않는다.

## 입출력 예시

  |str|result|
  |:--:|:--:|
  |Mississipi|?|
  |zZa|Z|
  |z|Z|
  |baaa|A|
 
## 풀이방식

### 나의 풀이방식

  > map을 사용한 풀이 방식

  자세한 설명은 주석으로 대체한다.
 
  ```cpp
  #include <iostream>
  #include <string>
  #include <unordered_map>
  using namespace std;
  int main() {
    unordered_map<char,int> M;
    string str;
    
    // 문자에 해당하는 map에 개수를 증가시킨다.
    cin >> str;
    for(int i=0;i<str.size();i++){
      if(str[i] >= 'A' && str[i] <= 'Z') M[str[i]]++;
      else M[(str[i] - 32)]++;
    }
    
    int same = false;   //같은 문자가 있는지 확인
    int maxCnt = 0;
    char answer;

    // Map을 순환하며 최대 개수 확인
    for(auto it = M.begin();it!=M.end();it++){
      if(maxCnt < it->second){
        answer = it->first;
        maxCnt = it->second;
        same = false;
      }
      else if (maxCnt == it->second) same = true;
    }
    if(same) cout << "?";
    else cout << answer;
    
    return 0;
  }
  ```

### 다른이의 풀이방식

  > alpha라는 배열을 알파벳 개수만큼 만들어 놓고 사용한다.

  큰 차이는 없지만 이 방식이 더 빠르다!

  ```cpp
  #include <iostream>
  using namespace std;

  int alpha[26], cnt =0;

  int main() {
    string input;
    cin >> input;

    for(int i=0;i<input.length();i++) {
      if(input[i] < 97) alpha[input[i] - 65]++; //대문자
      else alpha[input[i] - 97]++;              //소문자
    }

    int max = 0, max_indx=0;

    for(int i=0;i<26;i++) {
      if(max<alpha[i]) {
        max=alpha[i];
        max_indx = i;
      }
    }
    
    for(int i=0;i<26;i++) if(max == alpha[i]) cnt++;
    
    if(cnt>1) cout << "?";
    else cout << (char)(max_indx+65);
  }
  ```
