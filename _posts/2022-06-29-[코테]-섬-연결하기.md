## 문제

  n개의 섬 사이에 다리를 건설하는 비용(costs)이 주어질 때, 최소의 비용으로 모든 섬이 서로 통행 가능하도록 만들 때 필요한 최소 비용을 return 하도록 solution을 완성하세요. 다리를 여러 번 건너더라도, 도달할 수만 있으면 통행 가능하다고 봅니다. 예를 들어 A 섬과 B 섬 사이에 다리가 있고, B 섬과 C 섬 사이에 다리가 있으면 A 섬과 C 섬은 서로 통행 가능합니다.
  
## 조건

  - 섬의 개수 n은 1 이상 100 이하입니다.
  - costs의 길이는 ((n-1) * n) / 2이하입니다.
  - 임의의 i에 대해, costs[i][0] 와 costs[i][1]에는 다리가 연결되는 두 섬의 번호가 들어있고, costs[i][2]에는 이 두 섬을 연결하는 다리를 건설할 때 드는 비용입니다.
  - 같은 연결은 두 번 주어지지 않습니다. 또한 순서가 바뀌더라도 같은 연결로 봅니다. 즉 0과 1 사이를 연결하는 비용이 주어졌을 때, 1과 0의 비용이 주어지지 않습니다.
  - 모든 섬 사이의 다리 건설 비용이 주어지지 않습니다. 이 경우, 두 섬 사이의 건설이 불가능한 것으로 봅니다.
  - 연결할 수 없는 섬은 주어지지 않습니다.

## 입출력 예시

  |n|costs|return|
  |:--:|:--:|:--:|
  |4|[[0, 1, 1], [0, 2, 2], [1, 2, 5], [1, 3, 1], [2, 3, 8]]|4|
  |5|[[0, 1, 5], [1, 2, 3], [2, 3, 3], [3, 1, 2], [3, 0, 4], [2, 4, 6], [4, 0, 7]]|15|
  |5|[[0, 1, 1], [3, 4, 1], [1, 2, 2], [2, 3, 4]]|8|
  

## 풀이방식

### 나의 풀이방식

  탐욕법 문제이기 때문에 __먼저 비용이 가장 작은 순서대로 정렬하여 진행한다.__ 그 후 a와 b를 union, find를 사용하여 서로 연관되어 있는지 확인하고 연관되어 있지않다면 서로 연결하는 로직이다.

  처음에는 union과 find를 사용하지 않고, 체크 배열을 만들어서 해당 섬을 거쳐갔는지 판단하였는데 이는 문제를 야기했다.

  ```cpp
  #include <algorithm>
  int unf[1001];

  bool sorting (vector<int> a, vector<int> b) {
    return a[2] < b[2];
  }

  int Find(int v) {
    if (v == unf[v]) return v;
    else return unf[v] = Find(unf[v]);
  }

  void Union(int a, int b) {
    a = Find(a);
    b = Find(b);
    if (a != b) unf[a] = b;
  }

  int solution(int n, vector<vector<int>> costs) {
    int answer = 0;
    int cnt = 0;            //check2

    for (int i = 0; i < n; i++) unf[i] = i;

    // 내림차순 정렬
    sort(costs.begin(), costs.end(), sorting);
    for (int i = 0; i < costs.size(); i++) {
      if (cnt == n - 1) break;
      if ((Find(costs[i][0]) != Find(costs[i][1]))) {
        Union(costs[i][0], costs[i][1]);
        cnt++;
        answer += costs[i][2];
      }
    }
    return answer;
  }
  ```

### 다른 이의 풀이방식

  나와 동일한 방식이지만 조금 더 단순하게 표현하였다.

  ```c++
  int d[101];

  // union & find
  int getParent(int node){
    if(node == d[node]) return node;
    else return d[node] = getParent(d[node]);
  }

  bool compare(vector<int> a, vector<int> b){
    return a[2] < b[2];
  }

  int solution(int n, vector<vector<int>> costs) {
    int answer = 0;
    for(int i = 0; i < n; i++) d[i] = i;
    
    // 내림차순 정렬
    sort(costs.begin(), costs.end(), compare);
    for(int i=0; i<costs.size(); i++){
      int start = getParent(costs[i][0]);
      int end = getParent(costs[i][1]);
      int cost = costs[i][2];

      if(start != end){
        d[end] = start;
        answer += cost;
      }
    }
    return answer;
  }
  ```
