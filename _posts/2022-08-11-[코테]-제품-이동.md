## 문제

  섬이 많은 나라로 유명한 인도네시아는 N(1≤N≤10,000)개의 섬으로 이루어진 나라이다. 이 섬들은 다리로 연결되어 있는데, 각 다리는 통과하려면 무게제한이 있다. 이 무게제한을 넘어 가면 다리는 무너지게 된다.

  엘리트 무역회사는 이들 섬 중 2개에 공장이 있는데 항상 두 공장에서 서로 제품을 이동하는 작업을 한다. 섬의 개수 N과 각 섬을 연결하는 다리 정보가 주어지면 한 번의 이동으로 옮길 수 있는 제품의 최대 무게를 구하세요. 단 다리의 무게제한은 제품의 무게제한으로만 가정한다.


## 조건

  - N, M(1≤M≤100,000)이 주어진다.
  - 출발지/도착지/가중치 A, B(1≤A, B≤N), C(1≤C≤ 1,000,000,000)를 담은 arr가 주어진다.
  - 출발 공장(start)와 도착 공장(end)를 제공한다.

## 입출력 예시

  |N|M|arr|start|end|result|
  |:--:|:--:|:--:|:--:|:--:|:--:|
  |5|6|[[1,2,5],[1,3,3],[1,4,2],[2,4,2],[3,4,4],[4,5,3]]|1|5|3|
  
## 풀이방식

  > 우선순위큐와 BFS를 활용한 풀이

  ch[]배열에는 모든 경로를 통해 도착한 섬의 가중치를 나타내는데, 이때 가중치는 최댓값을 저장한다. (문제가 최댓값을 요구하기에)

  1. 출발지/도착지/가중치를 저장할 isLand 구조체를 만든다.
  2. BFS를 통해 "이전 도착지"로 부터 "갈 수 있는 곳"을 기준에 따라 push한다.
      - "이전 출발지"와 "갈 수 있는 곳"동일하다면 취소한다.
      - 이전 가중치와 현재 가중치를 비교하여 더 작은 가중치를 넘긴다.
      - ch배열에 저장될때는 기존 값보다 큰 가중치를 저장한다.
  3. 다시 왔던길을 돌아가지 못하도록 갔던 길을 삭제한다. 
  4. 모든 경로를 완료할때까지, 즉, pQ가 비어있을때까지 반복한다.

  
  ```cpp
  #include <queue>
  vector<pair<int,int>> arr[100001];
  int ch[100001];
  struct isLand{
    // 출발지, 도착지, 가중치 
    int from, to, val;
    
    isLand(int a, int b,int c) : from(a),to(b), val(c) {}
    bool operator <(const isLand &rhs) const{
      return val < rhs.val; 
    }
  };

  int solution(int n,int m, int to, int go){	
    priority_queue<isLand> pQ;

    //BFS
    ch[to] = -1;
    pQ.push(isLand(-1,to,100000));
    while(!pQ.empty()){
      //이전 출발지/도착지/가중치 
      int tmpfrom = pQ.top().from;		 
      int tmpto = pQ.top().to;			  
      int tmpval = pQ.top().val;			
      pQ.pop();
      
      // 2번 
      for(int i=0;i<arr[tmpto].size();i++){
        int togo = arr[tmpto][i].first;	//갈 수 있는 곳 
        int val = arr[tmpto][i].second;	//그 곳의 가중치 
        
        // 2-1번 : "이전 출발지"가 "갈 수 있는 곳"과 동일하다면 취소
        if(tmpfrom == togo) continue;
        else {
          // 2-2번 : 최소값을 대입해야함 
          int insertVal = (val > tmpval) ? tmpval: val; 
          
          // 2-3번 : ch를 최대값으로 갱신 
          if(ch[togo] != 0) ch[togo] = max(ch[togo],insertVal);
          else ch[togo] = insertVal;
          
          pQ.push(isLand(tmpto, togo, insertVal));
        }
      }
      // 3번 : 사용한 배열은 모두 제거 
      arr[tmpto].clear();
    }
    
    return ch[go];	
  }
  ```

## 개선점
  - 다른이는 이분법을 사용하여 최대 값을 정하고, 그 값에 따른 결과를 정했다.
    - 이 방법이 더 효과적인듯 싶지만, 이분법 사용 후 길찾기를 다시 진행하기 때문에 내가 푼 방식과 비슷한 효율일것으로 예상된다.
