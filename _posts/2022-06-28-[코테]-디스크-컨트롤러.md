## 문제

  하드디스크는 한 번에 하나의 작업만 수행할 수 있습니다. 디스크 컨트롤러를 구현하는 방법은 여러 가지가 있습니다. 가장 일반적인 방법은 요청이 들어온 순서대로 처리하는 것입니다. 이때 작업이 요청된 시간부터 종료까지의 시간을 최소화할때 가장 효과적인 방법은 __현재 시간에서 가능한 가장 짧은 시간이 소요되는 작업을 우선적으로 처리하는 것__ 입니다. 가장 효율적인 알고리즘을 작성하고, 작업이 요청된 시간부터 종료까지의 시간의 평균을 구하세요.

## 조건

  - jobs의 길이는 1 이상 500 이하입니다.
  - jobs의 각 행은 하나의 작업에 대한 [작업이 요청되는 시점, 작업의 소요시간] 입니다.
  - 각 작업에 대해 작업이 요청되는 시간은 0 이상 1,000 이하입니다.
  - 각 작업에 대해 작업의 소요시간은 1 이상 1,000 이하입니다.
  - 하드디스크가 작업을 수행하고 있지 않을 때에는 먼저 요청이 들어온 작업부터 처리합니다.

## 입출력 예시

  |jobs|return|
  |:--:|:--:|
  |[[0, 3], [1, 9], [2, 6]]|9|

## 풀이방식

### 1번 풀이방식 

  먼저 2개의 구조체를 만들었다. 하나는 __"작업이 요청된 시간이 작은 순서에 따라 정렬되는 operator를 갖는 구조체"__, 다른 하나는 __"작업의 소요 시간이 작은 순서에 따라 정렬되는 operator를 갖는 구조체"__ 이다.

  이를 활용해서 priority_queue를 각각 하나씩 Q, Q2로 만들어주었고, 현재 시간을 저장할 int형 curTime변수를 생성해주었다.

  먼저 Q에 작업이 요청된 시간에 따라 낮은 순으로 정렬되도록하였고, 만약 curTime보다 Q.top이 작다면 Q2에 push해주었다. __즉 현재 시간에 가능한 모든 작업을 Q2에 넣어주었다.__

  Q2에 내용이 있다면 이 중 작업의 소요 시간이 가장 작은 Q2.top부터 처리해주었고, curTime += Q2.top().requiredTime을 진행하여 종료된 시간을 업데이트해주었다. __즉 Q2에서 가장 소요시간이 작은 작업부터 처리해주었다.__


  ```cpp
  #include <queue>

  struct DISK {
    int requestTime = 0, requiredTime = 0;

    DISK(int a, int b) {
      requestTime = a;
      requiredTime = b;
    }

    bool operator < (const DISK &b) const {
      return requestTime > b.requestTime;
    }
  };
  struct DISK2 {
    int requestTime = 0, requiredTime = 0;

    DISK2(int a, int b) {
      requestTime = a;
      requiredTime = b;
    }

    bool operator < (const DISK2& b) const {
      return requiredTime > b.requiredTime;
    }
  };

  int solution(vector<vector<int>> jobs) {
    int curTime = 0 ,answer = 0;
    priority_queue<DISK> Q;
    priority_queue<DISK2> Q2;

    for (int i = 0; i < jobs.size(); i++){
      Q.push(DISK(jobs[i][0], jobs[i][1]));
    }

    while (!Q.empty() || !Q2.empty()) {
      while (!Q.empty() && curTime >= Q.top().requestTime) {
        Q2.push(DISK2(Q.top().requestTime, Q.top().requiredTime));
        Q.pop();
      }
      if (!Q2.empty()) {
        answer += ((curTime - Q2.top().requestTime) + Q2.top().requiredTime);
        curTime += Q2.top().requiredTime;
        Q2.pop();
      }
      else curTime++;
    }

    return answer / jobs.size();
  }
  ```

### 2번 풀이방식 

  1번 풀이방식과 비슷하지만 훨신 더 단순한 로직으로 구성되어 있다. 1개의 우선순위 큐를 사용하며, 현재 시간 기준으로 가능한 작업들을 pq에 넣어주고, pq에 넣어지면 2번째 파라미터 값에 의해 오름차순으로 정렬되는 것이다. 이렇게 구현한다면 더 짧고 간결하게 구상할 수 있다.

  ```cpp
  #include <queue>

  struct cmp {
    bool operator()(vector<int> a, vector<int> b) {
      return a.at(1) > b.at(1);
    }
  };
  
  int solution(vector<vector<int>> jobs) {
    int answer = 0, j = 0, time = 0;
    sort(jobs.begin(), jobs.end());
    priority_queue<vector<int>, vector<vector<int>>, cmp> pq;

    while (j < jobs.size() || !pq.empty()) {
      if (jobs.size() > j && time >= jobs[j][0]) {
        pq.push(jobs[j++]);
        continue;
      }
      if (!pq.empty()) {
        time += pq.top()[1];
        answer += time - pq.top()[0];
        pq.pop();
      }
      else
        time = jobs[j][0];
    }
    return answer / jobs.size();
  }
  ```
