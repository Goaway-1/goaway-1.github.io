## 문제

  백준이는 동생에게 "가운데를 말해요" 게임을 가르쳐주고 있다. 백준이가 정수를 하나씩 외칠때마다 동생은 지금까지 백준이가 말한 수 중에서 중간값을 말해야 한다. 만약, 그동안 백준이가 외친 수의 개수가 짝수개라면 중간에 있는 두 수 중에서 작은 수를 말해야 한다.

  예를 들어 백준이가 동생에게 1, 5, 2, 10, -99, 7, 5를 순서대로 외쳤다고 하면, 동생은 1, 1, 2, 2, 2, 2, 5를 차례대로 말해야 한다. 백준이가 외치는 수가 주어졌을 때, 동생이 말해야 하는 수를 구하는 프로그램을 작성하시오.

  [링크](https://www.acmicpc.net/problem/1655)

## 조건

  - 첫째 줄에는 백준이가 외치는 정수의 개수 N이 주어진다. N은 1보다 크거나 같고, 100,000보다 작거나 같은 자연수이다. 
  - 그 다음 N줄에 걸쳐서 백준이가 외치는 정수가 차례대로 주어진다. 정수는 -10,000보다 크거나 같고, 10,000보다 작거나 같다.

## 입출력 예시
  
  |N|arr|result|
  |:--:|:--:|:--:|
  |7|[1,5,2,10,-99,7,5]|[1,1,2,2,2,2,5]|

## 풀이방식

### 1번 풀이방식
 
  > stack과 vector의 활용으로 해결

  모든 수를 입력받아 vector에 정렬한 후, 중심 idx에 접근하여 answer에 저장한 후, 가장 마지막에 입력받은 수를 vector에서 찾아 제거.

  나름 시간복잡도가 괜찮다고 생각했으나, 그렇지 않은듯 하다....

  ```cpp
  #include <iostream>
  #include <vector>
  #include <stack>
  #include <algorithm>
  using namespace std;

  int main(void) {
    stack<int> answer; 
    int n;
    cin >> n;

    vector<int> arr(n);       // 현재 백준이 말한 수의 저장
    stack<int> popSt;			    // 사용자가 입력한 수의 순서
    for(int i =0;i<n;i++) {
      cin >> arr[i];
      popSt.push(arr[i]);
    }
    sort(arr.begin(),arr.end());

    int arrSize = 0;
    while (!popSt.empty()){
      // 중심 수를 저장
      arrSize = arr.size();
      if (arrSize % 2 == 0) answer.push(arr[(arrSize / 2) - 1]);
      else answer.push(arr[(arrSize / 2)]);

      /* 가장 마지막으로 말했던 수를 찾아 제거
      *  아마 시간 초과는 여기서...
      */
      bool isPop = false;
      int popNum = popSt.top();
      for (int i = 0; i < arr.size(); i++) {
        if (popNum == arr[i] && !isPop) isPop = true;
        else if (isPop) arr[i - 1] = arr[i];
      }
      arr.resize(arrSize - 1);
      popSt.pop();
    }
    
    // 결과 출력
    while (!answer.empty()) {
      cout << answer.top() << "\n";
      answer.pop();
    }

    return 0;
  }
  ```

### 2번 풀이방식

  > 최대/최소 2개의 __"Priority Queue"를__ 사용한 풀이 

  최대/최소 PQ와 mid값을 저장하는 변수, 총 3개를 활용하는 풀이이다. 아래 그림과 같이 __mid보다 작은 수는 좌측의 최대힙을 사용하고, 큰 수는 우측의 최소힙을 사용하여 정렬한다.__ 이때 상황에 따라 각 힙에서 pop을 사용하면, 중간 값이 도출되며, 상황에 따른 로직은 아래와 같다.
  
  <img src="https://raw.githubusercontent.com/Goaway-1/goaway-1.github.io/master/_posts/images/Max&MinPQ.png" height="350" title="Max&MinPQ">

  __"K = 현재 노드의 개수, N = 새로운 값"__
  1. K가 홀수인 경우
    - __"N < MID"__ : N을 좌측, MID를 우측에 삽입
    - __"N > MID"__ : N을 우측, MID를 좌측에 삽입
  2. K가 짝수인 경우
    - __"N < 우측 TOP"__ : N을 좌측, 좌측 TOP을 MID에 삽입
    - __"N > 우측 TOP"__ : N을 우측, 우측 TOP을 MID에 삽입

  ```cpp
  #include <iostream>
  #include <vector>
  #include <queue>
  #include <algorithm>
  using namespace std;

  int main(void) {
    ios_base :: sync_with_stdio(false); 
    cin.tie(NULL); 
    cout.tie(NULL);
      
    int mid = 0, n = 0, tmp = 0;                          // mid : 가운데의 정수, n : 정수의 개수, tmp : 입력받는 정수
    priority_queue<int, vector<int>, less<int>> LPQ;      //좌측 PQ
    priority_queue<int, vector<int>, greater<int>> RPQ;   //우측 PQ

    cin >> n;
    for (int i = 0; i < n; i++) {
      cin >> tmp;

      // 1번 : 현재 개수가 홀수인 경우
      if (i % 2 != 0) {	
        if(tmp > mid) {   
          RPQ.push(tmp); 
          LPQ.push(mid);
        }
        else { 
          LPQ.push(tmp);
          RPQ.push(mid);
        }
        cout << LPQ.top() << "\n";
      }
      // 2번 : 현재 개수가 짝수인 경우
      else {	
        if(i == 0) mid = tmp;	
        else if (tmp > RPQ.top()) { 
          RPQ.push(tmp); 
          mid = RPQ.top(); 
          RPQ.pop();
        }
        else { 
          LPQ.push(tmp); 
          mid = LPQ.top(); 
          LPQ.pop();
        }
        cout << mid << "\n";
      }
    }
    return 0;
  }
  ```


## 개선점
  - 우선순위큐를 2개 사용하여 중간값을 도출할 수 있구나.
