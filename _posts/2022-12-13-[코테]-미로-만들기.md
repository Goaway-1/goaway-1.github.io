## 문제

  홍준이는 미로 안의 한 칸에 남쪽을 보며 서있다. 미로는 직사각형 격자모양이고, 각 칸은 이동할 수 있거나, 벽을 포함하고 있다. 모든 행과 열에는 적어도 하나의 이동할 수 있는 칸이 있다. 홍준이는 미로에서 모든 행과 열의 이동할 수 있는 칸을 걸어다녔다. 그러면서 자신의 움직임을 모두 노트에 쓰기로 했다. 홍준이는 미로의 지도를 자기 노트만을 이용해서 그리려고 한다.

  입력으로 홍준이가 적은 내용을 나타내는 문자열이 주어진다. 각 문자 하나는 한 번의 움직임을 말한다. ‘F’는 앞으로 한 칸 움직인 것이고, ‘L'과 ’R'은 방향을 왼쪽 또는 오른쪽으로 전환한 것이다. 즉, 90도를 회전하면서, 위치는 그대로인 것이다.

  [링크](https://www.acmicpc.net/problem/1347)
  
## 조건

  - 첫째 줄에 홍준이가 적은 내용의 길이가 주어진다. 길이는 0보다 크고, 50보다 작다. 둘째 줄에 홍준이가 적은 내용이 내용이 주어진다.

## 입출력 예시
  
  |N|M|arr|result|
  |:--:|:--:|:--:|:--:|
  |3|4|[[1,2,3,4],[0,0,0,5],[9,8,7,6]]|31|
  |3|3|[[1,0,0],[0,1,0],[0,0,1]]|3|
  |4|3|[[1,2,3],[6,5,4],[7,8,9],[12,11,10]]|47|

## 풀이방식

  > [50,50]에서 시작하며, 단순 이동
  
  단순 이동이기에 쉽지만, 시작점을 어디서 잡아야할지 어려웠던 문제였다. 주어지는 길이는 최대 50이기 때문에, 직진만 한다는 가정에 최대 map의 크기는 50이다. 하지만 상하좌우 어디로 갈지 모르기에 map의 크기를 101로 지정

  그리고 출력을 위해서 minX/Y, maxX/Y를 통해 최소/최대 좌표를 지정한다.

  ```cpp
  #include <iostream>
  #include <vector>
  #include <string>
  #include <algorithm>

  using namespace std;

  // 추후 char로 전환
  int Map[101][101];
  int y[4] = { 0, -1, 0, 1 };
  int x[4] = { 1, 0, -1, 0 };

  int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL); cout.tie(NULL);
    
    // way : 하(0),좌(1),상(2),우(3)
    int n = 0, way = 0;
    char m;

    // 현재 위치
    pair<int, int> pos = make_pair(50, 50);

    // 최소 & 최대 좌표
    int minX = 50, maxX = 50, minY = 50, maxY = 50; 
   
    // input
    cin >> n;
    Map[50][50] = 1;

    for (int i = 0; i < n; i++){
      cin >> m;
        
      if (m == 'R' || m == 'L') {
        way += (m == 'R') ?  1 : -1;
        if (way == -1) way = 3;
        else if (way == 4) way = 0;
      }
      else {
        int tmpX = pos.first + x[way];
        int tmpY = pos.second + y[way];
        pos.first = tmpX;
        pos.second = tmpY;

        // set min/max
        maxX = max(maxX, tmpX);
        minX = min(minX, tmpX);
        maxY = max(maxY, tmpY);
        minY = min(minY, tmpY);
            
        Map[tmpX][tmpY] = 1;
      }
    }
    for (int i = minX ; i <= maxX; i++){
      for (int j = minY ; j <= maxY; j++){
        if (Map[i][j] == 1) cout << ".";
        else cout << "#";
      }
      cout << "\n";
    }
    return 0;
  }
  ```
