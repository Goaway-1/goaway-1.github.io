## 문제

  사각형의 땅 중에서 현수가 다스릴 수 있는 땅의 크기(정사각형 한 변의 길이)를 정해주면 전체 땅 중에서 그 크기의 땅의 위치를 현수가 정하는 것으로, 전체 땅은 2차원 평면좌표 형태로 표현된다. 전체 땅 안에는 많은 오렌지 나무가 심겨져 있다. 각 오렌지 나무는 아래 그림처럼 x, y좌표로 표현됩니다. x, y 좌표는 0과 양의 정수로만 표현된다. __이때 오렌지 나무가 가장 많이 포함되는 지역을 선택했을때 얻을 수 있는 오렌지 나무의 개수를 구하라.__

## 조건

  - 첫 번째 줄에 땅의 크기를 나태내는 W(가로길이)와 H(세로길이), T, S가 입력된다.
  - T는 오렌지 나무의 개수이고, S는 선택할 수 있는 정사각형 한 변의 길이이다. (3 <= W, H <= 100,000) (1 <= T <= 100) (1 <= S <= W, H)
  - 두 번째 줄부터 T개의 오렌지 나무 위치 정보가 주어진다.

## 풀이방식

  가장 짧은 길이의 문자열을 찾고, 그 문자를 접두사로 가정했을때 다른 문자열들이 모두 포함하는지 확인, 만약 그렇지 않다면 접두사의 마지막 1자리를 줄여가면서 탐색하는 방법.

  ```cpp
  #include <vector>

  int main(){
    //freopen("input.txt", "rt", stdin);
    int w,h,t,s, res = 0;
    vector<int> x;
    vector<int> y;
    cin >> w >> h >> t >> s;
    
    // input
    int a, b;
    for(int i =0;i<t;i++){
      cin >> a >> b;
      x.push_back(a);
      y.push_back(b);
    }
    
    // Calculation
    for(int i=0;i<t;i++){
      for(int j=0;j<t;j++){
        int cnt = 0;
        for(int k=0;k<t;k++){
          if(x[k] >= x[i] && x[k] <= x[i] + s){
            if(y[k] >= y[j] && y[k] <= y[j] + s) cnt++;
          }
        }
        res = max(res,cnt);
      }
    }
    cout << res;
    
    return 0;
  }
  ```

## 개선점
  - 시간 복잡도가 n^2임을 확인할 수 있는데, 이를 해결하기 위한 다른 방안이 있을까 모색중이다.
