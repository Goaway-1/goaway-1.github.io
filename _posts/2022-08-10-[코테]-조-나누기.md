## 문제

  현수네 동아리는 수학동아리입니다. 동아리 회장은 수학성적을 올리기 위해 동아리 회원을 M 개의 조로 나누어 각 조별로 서로 도우며 공부하기로 했습니다. 동아리는 여러 학년이 섞여 있는데 동아리 회장은 조원들간의 친밀감을 높여주기 위해 비슷한 나이로 한 조가 되도록 동아리 회원들을 나이순으로 정렬한 다음 적당히 나누어 M개의 조를 만들려고 합니다. 다음 그림은 동아리 회원을 나이순으로 정렬한 각 회원의 수학성적입니다. M개의 조가 잘 짜졌다라고 하려면 각 조의 성적의 합 중 최대값이 최소가 되도록 하는 것입니다. 
  
  예를 들어 3개의 조로 나눌때 아래와 같이 나누면, (50+20), (30+70), (20+60+30) 로 각 조의 합은 각각 70, 100, 110이 되어 그 중 최대값은 110이 된다.

  다르게 아래와 같이 나누면, 합이 각각 100, 90, 90이 되어 그 중 최댓값은 100이 된다. 각 조의 합의 최댓값을 100보다 더 작게 만들 수 없다.

  <img src="https://raw.githubusercontent.com/Goaway-1/goaway-1.github.io/master/_posts/images/Group2.png" height="350" title="Group2">

  각 조의 합 중 최대값이 최소가 되도록 M개의 조로 나누었을 때, 그 최대값을 출력하는 프로그램을 작성하세요.

## 조건

  - 학생수 N과 조의 개수 M이 주어진다. (N <= 300, N <= M)
  - 나이순으로 정렬된 각 학생의 수학점수(arr)가 주어진다. (점수는 100이하의 자연수)

## 입출력 예시

  |N|M|arr|result|
  |:--:|:--:|:--:|:--:|
  |7|3|[50,20,30,70,20,60,30]|100|
  |25|3|[87, 4, 15, 72, 88, 64, 39, 40, 91, 33, 4, 44, 57, 87, 22, 24, 44, 28, 74, 32, 52, 73, 37, 42, 81]|419|
  
## 풀이방식

  > 이분탐색을 활용한 풀이로 기준은 "조의 최대 값"

  문제를 통해 배열의 위치는 변경할 수 없으며, 연속적으로 팀원을 구성해야 된다는 사실이 도출되었다. 이를 활용하여 앞에서부터 최대값에 따른 조의 개수를 구하면 된다고 생각했다.

  __"조의 최대 값"을 기준으로 몇개의 조를 구성할 수 있는지 이분탐색과 결정알고리즘을 활용하여 로직을 구성하였다.__ 그 후 최소 값이 최대 값에 수렴할때 로직을 종료하며, 원하는 조의 개수가 일치할때만 최대값을 answer에 삽입하는 방식이다.

  ※ 예외로 __N과 M이 같은 경우에는__ 한 명씩 조를 구성하는 것을 의미하기 때문에 수학 점수의 최대값을 반환한다.

  ```cpp
  int solution(int n,int m, vector<int> arr){
    int answer = 0;
    int minVal = 0, midVal = 0,maxVal = 0;
    
    // 예외 처리 (n과 m이 같은경우)
    if(n == m){
      for(int	i=0;i<n;i++) answer = max(answer, arr[i]);
      return answer;
    } 
    
    // 모든 값의 합 : 이분탐색을 위함
    for(int	i=0;i<n;i++) maxVal += arr[i];
    
    // 이분탐색
    while(minVal < maxVal){
      midVal = (minVal + maxVal) / 2;
      
      // 결과값(midVal)를 통해 얻을 수 있는 조의 개수
      int cnt = 0, tmp = 0; 
      for(int i=0;i<n;i++){		
        if(tmp + arr[i] < midVal) tmp += arr[i];
        else {
          tmp = (tmp + arr[i] == midVal) ? 0 : arr[i]; 
          cnt++;
        }
      }
      if(tmp > 0) cnt++;
      
      // 원하는 조의 개수와 일치하다면
      if(cnt == m) {
        answer = midVal;
        maxVal = midVal;
      }
      else if(cnt > m) minVal = midVal + 1;
      else maxVal = midVal - 1;
    }
    
    return answer;	
  }
  ```
