## 문제

  앞뒤를 뒤집어도 똑같은 문자열을 팰린드롬(palindrome)이라고 합니다.
  문자열 s가 주어질 때, s의 부분문자열(Substring)중 가장 긴 팰린드롬의 길이를 return 하는 solution 함수를 완성해 주세요.

  예를들면, 문자열 s가 "abcdcba"이면 7을 return하고 "abacde"이면 3을 return합니다.

## 조건

  - 문자열 s의 길이 : 2,500 이하의 자연수
  - 문자열 s는 알파벳 소문자로만 구성

## 입출력 예시

  |s|answer|
  |:--:|:--:|
  |"abcdcba"|7|
  |"abacde"|3|
  
## 풀이방식

  > map에 push_back()해서 알파벳별로 index값을 배열에 저장. 가능성이 있는 index만 체크

  <img src="https://raw.githubusercontent.com/Goaway-1/goaway-1.github.io/master/_posts/images/Palindrome.png" height="300" title="Palindrome">

  위 사진은 s가 각각 "abcdcba", "abacde"일때의 map을 나타낸 것이다. 알파벳별로 index의 값을 map에 저장했다. 이를 활용하여 각 알파벳의 index가 2개 이상이라면 그 index값을 left, right로 가정하여 팰린드롬인지를 체크한다.

  1. 알파벳별로 index의 값을 map에 저장했다. (map<char, vector<int>>의 구조)
  2. 각 알파벳의 index가 2개 이상이라면 그 index값을 left, right로 가정하여 팰린드롬인지를 체크한다.
      - 이때 left는 가장 처음 index를, right는 가장 마지막 index부터 시작한다.
      - 팰린드롬을 확인하기 전에 answer보다 길이가 길어야만 확인을 진행한다. (필수)
  3. 팰린드롬의 확인은 단순히 index를 줄여가며 확인한다.

  ```cpp
  #include <map> 
  #include <set>
  #include <vector>
  
  // left부터 right까지 팬린드롬이 맞는지 확인 
  bool IsPalindrome(int left, int right, const string &s) {
    while (left < right) {
      if (s[left++] != s[right--]) return false;
    }
    return true;
  }

  int solution(string s) {
    int answer = 1;
    map<char, vector<int>> m;       //key값에 맞는 index를 추가
    set<char> tmpS;                 //존재하는 key값을 모아둠

    if(s.size()==1) return 1;

    // key값에 맞는 index를 추가
    for (int i = 0; i < s.size(); i++) {
      m[s[i]].push_back(i);
      tmpS.insert(s[i]);
    }

    // 팰린드롬을 체크
    for (auto it = tmpS.begin(); it != tmpS.end(); it++) {
      vector<int> vec = m[*it];
      bool isEnd = false;

      // 존재하는 key값의 인덱스가 2개 이상인 경우에만 체크
      if (vec.size() >= 2) {
        for (int i = 0; i < vec.size(); i++) {
          int left = vec[i];
          for (int j = vec.size() - 1; j > i; j--) {
            int right = vec[j];
            
            // 거리가 짧다면 계산조차 하지 않음
            if (answer > (right - left + 1)) continue;

            // 실질적인 패린드롬의 계산
            if (IsPalindrome(left, right, s)) answer = (right - left + 1);
          }
        }
      }
    }
    return answer;
  }
  ```
