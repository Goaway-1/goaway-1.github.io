## 문제

  최대 공통 부분 문자열이란 두 문자열 acbehf와 abefc의 공통의 부분 문자열 중에서 가장 긴 것을 의미한다.  여기서 최대 공통 부분 문자열은 abef 이다.두 문자열이 주어지면 두 문자열의 최대공통부분문자열의 길이를 출력하는 프로그램을 작성하세요.

## 조건
 
  - 2개의 문자열 num1, num2가 주어진다. 이 문자열의 길이는 1000을 넘지 않는다.

## 입출력 예시

  |num1|num2|result|
  |:--:|:--:|:--:|
  |acbehf|abefc||
  
## 풀이방식

### 나의 풀이방식 (오답)

  > 두 문자열에 대해 2개의 인덱스(고정 인덱스, 비교 인덱스)를 저장하고, 그 값이 n1(고정 인덱스) == n2(비교 인덱스)와 일치하면서 공통부분을 추가해갔다. (반대의 경우도 마찬가지)
  
  하지만 가장 먼저 겹치는 알파벳에 대해서 무조건적으로 정답처리했기 때문에 4,5번에서 정답을 맞추지 못했다. 
  
  4,5번 문제는 "babbbbb" 와 "bbbbbbbbbbbba"를 비교했을때, 정답으로는 "ba"가 도출된다. 하지만 최대공통부분은 "bbbbb"가 될 수 있다. 즉 가장 우선적으로 나오는 것에 대해 정답처리를 진행했기 때문에 최대공통부분을 구할 수 없었다. 또한 코드가 더럽다.

  ```cpp
  int solution(string n1,string n2){	
    string answer;	
    
    // n1과 n2에 해당하는 인덱스 값 (고정 인덱스, 비교 인덱스)
    int idx1[2] = {0,0};
    int idx2[2] = {0,0};
    
    while(idx1[0] < n1.size() && idx2[0] < n2.size()){
      if(n1[idx1[0]] == n2[idx2[1]]){       // n1의 고정 인덱스와 n2의 비교인덱스가 일치하는 경우
        answer += n2[idx2[1]];
        idx2[0] = ++idx2[1];
        idx1[0]++;
        idx1[1]++;
      }
      else if(n2[idx2[0]] == n1[idx1[1]]){	// 그 반대의 경우
        answer += n1[idx1[1]];
        idx1[0] = ++idx1[1];
        idx2[0]++;
        idx2[1]++;
      }
      else if(idx1[1] >= n1.size() && idx2[1] >= n2.size()){    // 두개다 만족하지 못하는 경우
        idx1[1] = ++idx1[0];
        idx2[1] = ++idx2[0];			
      }
      else{
        if(idx1[1] < n1.size()) idx1[1]++;
        if(idx2[1] < n2.size()) idx2[1]++;
      }
    }
      
    return answer;	
  }
  ```

### 다른이의 풀이방식 (정답)

  > 동적계획법을 사용한 풀이로 어떤 위치에 있는 알파벳을 선택하느랴에 따라 최대값이 달라진다.

  예를들어 "abefc"와 "acbehf"일때, 순서대로 진행하면 다음과 같다. "'a'와 'a'", "'a'와 'ac'", "'a'와 'acb'", "'a'와 'acbe'"....

  <img src="https://raw.githubusercontent.com/Goaway-1/goaway-1.github.io/master/_posts/images/LCS.png" height="350" title="LCS">

  즉 이런과정을 반복하면 위와같은 그림의 형식이 완성된다. 동적 계획법이기 때문에 이전 값과 비교하여 가장 큰 값을 넘긴다.  

  ```cpp
  int solution(string num1,string num2){	
    vector<vector<int>> dy(num1.size(), vector<int>(num2.size(),0));
    int n = num1.size();
    int m = num2.size();
    num1 = ' ' + num1;	
    num2 = ' ' + num2;
    
    // 동적계획법
    for(int i=1;i<n;i++){
      for(int j=1;j<m;j++){
        if(num1[i] == num2[j]) dy[i][j] = dy[i-1][j-1] + 1;
        else dy[i][j] = max(dy[i-1][j], dy[i][j-1]);
      }	
    }
      
    return dy[n-1][m - 1];	
  }
  ```
