## 문제

  N개의 문자열이 주어지면 이 문자열들의 최대 공통 접두사를 출력하는 프로그램을 작성하세 요. 만약 문자열들이 {“long", "longtime", "longest"} 라면 세 단어의 최대 공통 접두사는 ”long"입니다.

## 조건

  - 첫 번째 줄에 자연수 N(3 <= N <= 30)이 주어진다.
  - 두 번째 줄부터 N개의 단어가 입력된다. (길이는 100이하)

## 풀이방식
 
### 1번 풀이방식

  먼저 어쩌구 저쩌구

### 2번 풀이방식
  
  먼저 어쩌구 저쩌구

## 코드

### 1번 코드

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
    cout << "Hello World!";
    return 0;
  }
  ```

### 2번 코드

  ```cpp
  #include <iostream>
  using namespace std;

  int main() {
    cout << "Hello World!";
    return 0;
  }
  ```
