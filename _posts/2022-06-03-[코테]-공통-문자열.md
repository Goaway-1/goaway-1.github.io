## 문제

  N개의 문자열이 주어지면 이 문자열들의 __최대 공통 접두사를__ 출력하는 프로그램을 작성하세요. 만약 문자열들이 {"long", "longtime", "longest"} 라면 세 단어의 최대 공통 접두사는 "long"입니다.

## 조건

  - 첫 번째 줄에 자연수 N(3 <= N <= 30)이 주어진다.
  - 두 번째 줄부터 N개의 단어가 입력된다. (길이는 100이하)
  - 모든 문자열은 소문자로 입력된다.

## 풀이방식
 
### 1번 풀이방식

  가장 짧은 길이의 문자열을 찾고, 그 문자를 접두사로 가정했을때 다른 문자열들이 모두 포함하는지 확인, 만약 그렇지 않다면 접두사의 마지막 1자리를 줄여가면서 탐색하는 방법.

  ```cpp
  int main(){
    //freopen("input.txt", "rt", stdin);

    int n, len=INT_MAX;
    string str;
    vector<string> words;

    // input
    cin >> n;
    for(int i=0; i<n; i++){
      cin >> str;
      words.push_back(str);
    }

    string res = words[0];
    for(int i=1; i<words.size(); i++){
      while(words[i].find(res) != 0){
        res=res.substr(0, res.size() - 1);
      }
    }
    cout<<res;
    return 0;
  }
  ```

### 2번 풀이방식
  
  문자열들의 앞에서부터 연속으로 한 자리 수의 문자에 대하여 비교하고, 만약 그 문자가 모두 같다면 결과에 덧붙이는 방식
    1. Set을 활용하여 개수가 1개일때 같다고 판단하는 방식. 
    2. 상수를 활용하는 방법.

  ```cpp
  #include <set>

  int main(){
    //freopen("input.txt", "rt", stdin);

    int n, len=INT_MAX;
    string str;
    vector<string> words;

    //input
    cin>>n;
    for(int i=0; i<n; i++){
      cin>>str;
      words.push_back(str);
      len = min(len, int(str.size()));
    }

    string res="";
    for(int i=0; i<len; i++){
      set<char> tmp;
      for(auto &s : words) tmp.insert(s[i]);
      if(tmp.size() == 1) res += words[0][i];
      else break;
    }
    cout<<res;
    return 0;
  }
  ```

## 개선점
  - 두 해결법 모두 시간 복잡도가 n^2임을 확인할 수 있는데, 이를 해결하기 위한 다른 방안이 있을까 모색중이다.
