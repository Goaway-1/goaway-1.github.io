## 문제

  Anagram이란 두 문자열이 알파벳의 나열 순서를 다르지만 그 구성이 일치하면 두 단어는 아나그램이라고 합니다. 예를 들면 AbaAeCe 와 baeeACA 는 알파벳을 나열 순서는 다르지만 그 구성을 살펴보면 A(2), a(1), b(1), C(1), e(2)로 알파벳과 그 개수가 모두 일치합니다. 즉 어느 한 단어를 재배열하면 상대편 단어가 될 수 있는 것을 아나그램이라 합니다. 하나의 문자열 S가 주어지면, S문자열에서 T문자열과 아나그램이 되는 연속된 부분문자열이 몇 개 있는지 구하는 프로그램을 작성하세요.

## 조건

  - 첫 번째 줄에 길이 100,000을 넘지 않는 문자열 S가 주어진다.
  - 두 번째 줄에 길이 50,000을 넘지 않는 문자열 T가 주어진다.(T길이<=S길이)
  - 두 문자열의 모두 소문자로 입력된다.

## 풀이방식

### 1번 풀이방식 

  입력 예시 : eabcbacade \n abc
  출력 예시 : 3

  ![Ana_init](https://github.com/Goaway-1/goaway-1.github.io/blob/master/_posts/images/Ana_init.png)

  먼저 문자열 S의 한 문자씩 Map에 추가하며, 위 그림와 같은 구조(<char, int>)를 이룬다. 이때 Map은 unordered를 사용하여 자동으로 정렬되지 않도록한다. 그 후 T문자열의 한 문자씩 Map의 내용과 비교해가며, 동일하다면 map의 int를 하나씩 빼준다. 이때 map의 모든 int값이 0이라면 "아나그램이 되는 연속된 부분 문자열"이기 때문에 cnt에 1을 더해준다.

  ![Ana_cal](https://github.com/Goaway-1/goaway-1.github.io/blob/master/_posts/images/Ana_cal.png)

  이때 속도향상을 위해서 __사전__ i-1번째의 문자가 map에 있다면 int에 더해주고, __추후__ i + s.size() - 1번째 문자가 map에 있다면 int에서 빼준다. 위 그림에서는 a,b,c가 0이 되었기에 아나그램을 만족한다.

    ```cpp
    #include <iostream>
    #include <string>
    #include <unordered_map>

    int main(){
      string s, t;
      int cnt = 0;
      unordered_map<char, int> M;
      cin >> s >> t;
      
      for(int i=0;i < t.size();i++) M[t[i]]++;
      
      //init
      bool isDone = true;
      for(int i=0;i<t.size();i++) if(M.find(s[i]) != M.end()) M[s[i]]--;
      for(auto it = M.begin();it != M.end();it++){
        if(it->second != 0){
          isDone = false;
          break;
        }
      }
      if(isDone) cnt++; 
      
      //calcuate
      for(int i=1;i < s.size()-1;i++){
        isDone = true;
        
        if(M.find(s[i-1]) != M.end()) M[s[i-1]]++;
        if(M.find(s[i+t.size()-1]) != M.end()) 	M[s[i+t.size()-1]]--;
        
        for(auto it = M.begin();it != M.end();it++){
          if(it->second != 0){
            isDone = false;
            break;
          }
        }
        
        if(isDone) cnt++;
      }
      
      cout << cnt;
      return 0;
    }
    ```

### 2번 풀이방식 

  이전 방식과 비슷하지만 이는 map을 2개 사용하여, 두 개의 맵이 동일하다면 처리. 이때도 역시 앞을 제거, 뒤를 추가해가며 진행

    ```cpp
    #include <iostream>
    #include <string>
    #include <unordered_map>

    int main(){
      string S, T;
      int cnt=0;
      unordered_map<char, int> sH, tH;
      cin >> S >> T;
      for(auto x : T) tH[x]++;

      int L=T.size()-1;
      for(int i=0; i<L; i++) sH[S[i]]++;

      int lt=0;
      for(int rt=L; rt<S.size(); rt++){
        sH[S[rt]]++;
        if(sH==tH) cnt++;
        sH[S[lt]]--;
        if(sH[S[lt]]==0) sH.erase(S[lt]);
        lt++;
      }
      cout<<cnt<<endl;
      return 0;
    }
    ```
