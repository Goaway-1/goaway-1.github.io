## 문제

  현수는 쿠웨이트에 있는 사막에 벼농사가 되도록 하기 위해 바다에 있는 물을 증류해서 사막 으로 보내려고 합니다. 거대한 송수관 공사를 해야 하는데 각 파이프의 길이와 용량이 각기 다릅니다. 바다에서 사막까지의 길이 L이 주어지고, N개의 파이프의 길이와 용량이 주어지면 단위시간에 송수관의 최대 용량을 구하는 프로그램을 작성하세요. 여러개의 파이프를 연결하여 송수관을 만들었을 때 단위시간 송수관의 용량은 송수관을 구성 하는 파이프 중 가장 작은 용량이 됩니다. 

  3 6
  2 7
  4 8
  1 5

  만약 바다에서 사막까지의 길이가 6이고, 현수가 가지고 있는 파이프의 개수가 4개이고 각기 다음과 같은 길이와 용량을 가지고 있다면 3+2+1=6 길이를 만든다면 최대용량은 5이고, 2+4=6 길이를 만든다면 최대용량은 7이 됩니 다. 즉 현수가 바다에서 사막으로 보낼 수 있는 송수관의 최대 용량은 7입니다.

## 조건

  - 바다에서 사막까지의 길이 L(5<=L<=300)과 파이프의 개수N(3<=N<=20)가 주어진다.
  - N개의 파이프의 길이와 용량이 배열로 주어진다.

## 입출력 예시

  |L|N|arr|result|
  |:--:|:--:|:--:|:--:|
  |6|4|[[3,6],[2,7],[4,8],[1,5]]|7|
  
## 풀이방식

  > DFS의 문제로 파이프를 사용하거나 사용하지 않는 경우에 대해서만 계산하면 된다.

  DFS()는 "lng, val, idx"로 구성되며, 각각 "현재 파이프의 길이, 용량, 인덱스 값"를 의미한다.

  DFS는 파이프를 사용하냐 사용하지 않느냐에 따라 2개로 구분되기 때문에 for()구문을 사용하지 않고 진행했다. 2가지 인덱스가 끝에 도달했을때, 원하는 파이프의 길이인지 확인하고 일치하다면 answer와 값을 비교하여 갱신한다.

  ```cpp
  vector<pair<int,int>> pipeArr;
  int pipL;
  int answer = 0;

  //lng : 현재 길이, val : 현재 용량, idx : 인덱스 값 
  void DFS(int lng, int val,int idx){
    if(pipL < lng) return;
    
    if(idx == pipeArr.size()){
      // 원하는 파이프의 길이를 확인
      if(pipL == lng) answer = max(answer,val);
    }
    else{
      DFS(lng + pipeArr[idx].first, min(val,pipeArr[idx].second),idx+1);    //파이프 사용
      DFS(lng,val,idx+1);                                                   //파이프 미사용
    }
  }

  int solution(int L,int N,vector<pair<int,int>> arr){	
    pipeArr = arr;
    pipL = L;
    
    DFS(0,INT_MAX,0);
    
    return answer;
  }
  ```


## 개선점
  - DFS에 있어 중요한 것은 시간을 줄이는 것, 즉 중복을 줄이는 것이다.
  - 어떤 경우에 대해서 계산이 필요한지 생각하고, 코드를 구성하자.
