## 문제

  길이가 같은 두 개의 큐가 주어집니다. 하나의 큐를 골라 원소를 추출(pop)하고, 추출된 원소를 다른 큐에 집어넣는(insert) 작업을 통해 각 큐의 원소 합이 같도록 만들려고 합니다. 이때 필요한 작업의 최소 횟수를 구하고자 합니다. 한 번의 pop과 한 번의 insert를 합쳐서 작업을 1회 수행한 것으로 간주합니다.

  큐는 먼저 집어넣은 원소가 먼저 나오는 구조입니다. 이 문제에서는 큐를 배열로 표현하며, 원소가 배열 앞쪽에 있을수록 먼저 집어넣은 원소임을 의미합니다. 즉, pop을 하면 배열의 첫 번째 원소가 추출되며, insert를 하면 배열의 끝에 원소가 추가됩니다. 예를 들어 큐 [1, 2, 3, 4]가 주어졌을 때, pop을 하면 맨 앞에 있는 원소 1이 추출되어 [2, 3, 4]가 되며, 이어서 5를 insert하면 [2, 3, 4, 5]가 됩니다.

  다음은 두 큐를 나타내는 예시입니다.

  ```cpp
  queue1 = [3, 2, 7, 2]
  queue2 = [4, 6, 5, 1]
  ```

  두 큐에 담긴 모든 원소의 합은 30입니다. 따라서, 각 큐의 합을 15로 만들어야 합니다. 예를 들어, 다음과 같이 2가지 방법이 있습니다.

  1. queue2의 4, 6, 5를 순서대로 추출하여 queue1에 추가한 뒤, queue1의 3, 2, 7, 2를 순서대로 추출하여 queue2에 추가합니다. 그 결과 queue1은 [4, 6, 5], queue2는 [1, 3, 2, 7, 2]가 되며, 각 큐의 원소 합은 15로 같습니다. 이 방법은 작업을 7번 수행합니다.
  2. queue1에서 3을 추출하여 queue2에 추가합니다. 그리고 queue2에서 4를 추출하여 queue1에 추가합니다. 그 결과 queue1은 [2, 7, 2, 4], queue2는 [6, 5, 1, 3]가 되며, 각 큐의 원소 합은 15로 같습니다. 이 방법은 작업을 2번만 수행하며, 이보다 적은 횟수로 목표를 달성할 수 없습니다.
  
  따라서 각 큐의 원소 합을 같게 만들기 위해 필요한 작업의 최소 횟수는 2입니다.

  길이가 같은 두 개의 큐를 나타내는 정수 배열 queue1, queue2가 매개변수로 주어집니다. 각 큐의 원소 합을 같게 만들기 위해 필요한 작업의 최소 횟수를 return 하도록 solution 함수를 완성해주세요. 단, 어떤 방법으로도 각 큐의 원소 합을 같게 만들 수 없는 경우, -1을 return 해주세요.

## 조건

  - 1 ≤ queue1의 길이 = queue2의 길이 ≤ 300,000
  - 1 ≤ queue1의 원소, queue2의 원소 ≤ 109
  - 주의: 언어에 따라 합 계산 과정 중 산술 오버플로우 발생 가능성이 있으므로 long type 고려가 필요합니다.

## 입출력 예시
  
  |queue1|queue2|result|
  |:--:|:--:|:--:|
  |[3, 2, 7, 2]|[4, 6, 5, 1]|2|
  |[1, 2, 1, 2]|[1, 10, 1, 2]|7|
  |[1, 1]|[1, 5]|-1|

## 풀이방식

### 틀린 풀이방식
 
  > DFS를 사용한 풀이. 시간복잡도도 높고, 정답률 또한 낮음

  초기 DFS가 아니면 모든 경우의 수를 파악할 수 없다고 생각했었음.... 

  ```cpp
  #include <string>
  #include <vector>
  #include <queue>
  using namespace std;

  int answer = -1;
  int maxCnt = 0;

  void DFS(queue<int> Q1, queue<int> Q2, int Q1Val, int Q2Val, const int tarVal, int cnt) {
    if (cnt > maxCnt || (answer != -1 && answer < cnt)) return;

    if (tarVal == Q1Val && tarVal == Q2Val) {
      if (answer == -1 || answer > cnt) answer = cnt;
    }
    else {
      if (!Q1.empty()) {
        int tmp = Q1.front();
        Q1.pop();
        Q2.push(tmp);
        DFS(Q1, Q2, Q1Val - tmp, Q2Val + tmp, tarVal, cnt + 1);
      }

      if (!Q2.empty()) {
        int tmp = Q2.front();
        Q2.pop();
        Q1.push(tmp);
        DFS(Q1, Q2, Q1Val + tmp, Q2Val - tmp, tarVal, cnt + 1);
      }
    }
  }

  int solution(vector<int> queue1, vector<int> queue2) {
    queue<int> Q1, Q2;
    int tolVal = 0, Q1Val = 0, Q2Val = 0;
    maxCnt = queue1.size() * 2;

    // Make Queue
    for (int i = 0; i < queue1.size(); i++) {
      Q1.push(queue1[i]);
      tolVal += queue1[i];
      Q1Val += queue1[i];

      Q2.push(queue2[i]);
      tolVal += queue2[i];
      Q2Val += queue2[i];
    }
    tolVal /= 2;
    DFS(Q1, Q2, Q1Val, Q2Val, tolVal, 0);
    return answer;
  }
  ```

### 맞은 풀이방식

  > 욕심쟁이 기법을 사용하여 항상 큰 큐를 pop하여, 작은 큐에 push

  위 설명처럼 욕심쟁이(그리드) 기법을 사용하며, 각 큐를 구성하는 값을 더한 값을 비교해 합이 더 큰 큐에서 pop하고, 작은 큐에 push한다.

  1. 큐 기능을 사용하기 위해서 __"vector타입"인__ queue1/2를 __"queue타입"으로__ 수정
      - 이때 Q가 queue이며, QVal는 각 큐의 합을 저장한 1차원 배열
  2. 양쪽 큐를 비교하여, pop & push 과정을 반복
      - 각 큐의 합이 동일할때까지 반복. 이때 idx배열은 코드의 길이를 줄이기 위함이며, 큐이 값이 더 큰지 구분

  ```cpp
  #include <string>
  #include <queue>
  #include <vector>
  using namespace std;

  int solution(vector<int> queue1, vector<int> queue2) {
    long long answer = 0;
    vector<queue<int>> Q(2);
    long long QVal[2] = { 0,0 };                                // 각 큐의 합을 저장
    long long tolVal = 0, maxCnt = (queue1.size() * 2) + 2;     // maxCnt는 최대 pop & push 횟수를 지정

    // 1번 : Make Queue || 총 합, 현재 큐의 합을 저장
    for (long long i = 0; i < queue1.size(); i++) {
      Q[0].push(queue1[i]);
      tolVal += queue1[i];
      QVal[0] += queue1[i];

      Q[1].push(queue2[i]);
      tolVal += queue2[i];
      QVal[1] += queue2[i];
    }
    tolVal /= 2;

    // 2번 : 양쪽 큐를 비교하여, pop & push과정을 반복
    while (true) {
      if (maxCnt < answer) return -1;

      if (QVal[0] == QVal[1]) break;
      else {
        int idx[2] = {0,1};             // 어떤 큐의 값이 더 큰지 구분 -> 초기는 queue1이 더 크다고 가정
        long long tmp = 0;              // pop()과 push()를 위해 값을 저장
  
        // 각 큐합의 크기 비교 -> queue2가 더 큰 경우..
        if (QVal[0] < QVal[1]) { 
          idx[0] = 1;
          idx[1] = 0;
        }

        // pop과 push 과정
        tmp = Q[idx[0]].front();
        Q[idx[0]].pop();
        Q[idx[1]].push(tmp);

        QVal[idx[0]] -= tmp;
        QVal[idx[1]] += tmp;
      }
      answer++;
    }
    return answer;
  }
  ```
