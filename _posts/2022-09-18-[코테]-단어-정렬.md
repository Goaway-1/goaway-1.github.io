## 문제
  
  알파벳 소문자로 이루어진 N개의 단어가 들어오면 아래와 같은 조건에 따라 정렬하는 프로그램을 작성하시오.

  1. 길이가 짧은 것부터
  2. 길이가 같으면 사전 순으로

  조건에 따라 정렬하여 단어들을 출력한다. 단, 같은 단어가 여러 번 입력된 경우에는 한 번씩만 출력한다.

## 조건

  - 첫째 줄에 단어의 개수 N이 주어진다. (1 ≤ N ≤ 20,000) 둘째 줄부터 N개의 줄에 걸쳐 알파벳 소문자로 이루어진 단어가 한 줄에 하나씩 주어진다.
  - 주어지는 문자열의 길이는 50을 넘지 않는다.

## 입출력 예시

  |n|arr|result|
  |:--:|:--:|:--:|
  |13|[but, i, wont, hesitate, no, more, no, more, it, cannot, wait, im, yours]|[i, im, it, no, but, more, wait, wont, yours, cannot, hesitate]|
 
## 풀이방식

### 1번 풀이방식

  > set을 통해 중복을 걸러내고, 새로운 sort 조건을 통해 정렬

  set을 활용하여 입력들 중에서의 중복을 걸러내고, vector로 문자들을 옮긴다. 그 후 정렬(sort)를 진행하는데, 이때 compare이라는 조건을 통해서 정렬한다.

  조건은 우선 길이를 우선으로 비교하고, 길이가 같다면 사전 순서대로 정렬한다.
 
  ```cpp
  #include <iostream>
  #include <string>
  #include <vector>
  #include <algorithm>
  #include <set>
  using namespace std;

  bool compare (string a, string b){
    if(a.size() != b.size()) return a.size() < b.size();
    else return a < b;
  }

  int main() {
    int n;
    string tmp = "";
    set<string> S;
    vector<string> arr;
    
    cin >> n;    
    for(int i=0;i<n;i++){
      cin >> tmp;
      S.insert(tmp);
    }
    
    for(auto x : S) arr.push_back(x);
    
    sort(arr.begin(),arr.end(),compare);
    
    for(int i=0;i<arr.size();i++) cout << arr[i] << "\n";
    
    return 0;
  }
  ```

### 2번 풀이방식

  > set을 사용하지 않고, __unique와 erase를__ 사용하여 중복을 제거

  위처럼 set을 사용하여 중복을 제거하여 vector에 넣지않고, 초기부터 vector에 넣고, 중복을 제거하는 방식이다. 내 생각이지만 set이라는 공간을 사용하지 않기 때문에 메모리 공간에서 더 효율적이지 않을까 싶다.

  ```cpp
  #include <iostream>
  #include <string>
  #include <vector>
  #include <algorithm>
  using namespace std;

  bool compare (string a, string b){
    if(a.size() != b.size()) return a.size() < b.size();
    else return a < b;
  }

  int main() {
    int n;
    string tmp = "";
    vector<string> arr;
    
    cin >> n;
    for(int i=0;i<n;i++){
      cin >> tmp;
      arr.push_back(tmp);
    }
      
    sort(arr.begin(),arr.end(),compare);
    arr.erase(unique(arr.begin(), arr.end()),arr.end());  // 수정된 부분
    
    for(int i=0;i<arr.size();i++) cout << arr[i] << "\n";
    
    return 0;
  }
  ```
## 개선점
  - 중복을 제거하기 위해서 set을 사용하지 않는 방법은 없을까해서 2번 방식으로 풀었다.
