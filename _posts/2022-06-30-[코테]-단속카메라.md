## 문제

  고속도로를 이동하는 모든 차량이 고속도로를 이용하면서 단속용 카메라를 한 번은 만나도록 카메라를 설치하려고 합니다. 
  
  고속도로를 이동하는 차량의 경로 routes가 매개변수로 주어질 때, 모든 차량이 한 번은 단속용 카메라를 만나도록 하려면 최소 몇 대의 카메라를 설치해야 하는지를 return 하도록 solution 함수를 완성하세요.
  
## 조건

  - 차량의 대수는 1대 이상 10,000대 이하입니다.
  - routes에는 차량의 이동 경로가 포함되어 있으며 routes[i][0]에는 i번째 차량이 고속도로에 진입한 지점, routes[i][1]에는 i번째 차량이 고속도로에서 나간 지점이 적혀 있습니다.
  - 차량의 진입/진출 지점에 카메라가 설치되어 있어도 카메라를 만난것으로 간주합니다.
  - 차량의 진입 지점, 진출 지점은 -30,000 이상 30,000 이하입니다.

## 입출력 예시

  |routes|costs|
  |:--:|:--:|
  |[[-20,-15], [-14,-5], [-18,-13], [-5,-3]]|2|
  

## 풀이방식

  먼저 카메라를 가장 먼저 들어온 차의 진입로에 설치하게 되면 손해다. 그 차보다 먼저들어오는 차는 없기 때문이다. 
  
  만약 n지점에 진입하고, m지점에서 진출했다면 이 사이에는 카메라가 무조건 존재해야한다. 이때 __**진출지점에 카메라가 존재한다고 가정한다면,**__ 그 사이에 들어오는 수많은 차들을 카메라로 찍을 수 있다. 그렇기에 진출지점을 기준으로 오름차순으로 정렬해야한다. 

  <span style = "color:orange;">즉, 알고리즘은 아래와 같다.</span>

  1. 배열의 진출지점을 기준으로 오름차순으로 정렬한다.
  2. 현재 카메라 설치 위치를 나타내는 변수 cur를 선언한다.
  3. 만약 진입구간이 현재 카메라 위치보다 크다면 answer를 1개 증가시켜, n ~ m 중 어디에 설치했는지는 모르겠지만 카메라를 하나 사용했다고 한다.

  ```cpp
  #include <algorithm>

  bool sorting (vector<int> a, vector<int> b) {
    if(a[1] != b[1]) return a[1] < b[1];
    else return a[0] < b[0];
  }

  int solution(vector<vector<int>> routes) {
    int answer = 0;
    int cur = -30001;

    sort(routes.begin(), routes.end(),sorting);

    for (int i = 0; i < routes.size(); i++){
      if(routes[i][0] > cur){
        answer++;
        cur = routes[i][1];
      }
    }
    return answer;
  }
  ```

## 개선점
  알고리즘은 생각하는 능력을 기르는 학습이다. 포기하지 않고, 계속 생각하자
