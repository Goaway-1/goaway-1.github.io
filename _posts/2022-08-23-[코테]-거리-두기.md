## 문제

  현수는 영화관에 도착했습니다. 영화상영 시간보다 약간 늦은 현수는 남은 좌석을 빨리 선택 하고 영화를 보려고 합니다. 현수에게 일렬로된 좌석정보가 주어지면, 이미 앉아 있는 사람들 중 가장 가까운 사람과 최대 한 멀리 떨어져 앉을 자석을 선택해야 합니다. 여러분이 도와주세요.

## 조건
  - 총 좌석의 N(3 <= N <= 100)이 주어진다.
  - N개의 좌석의 정보인 arr가 배열의 형태로 주어진다.

## 입출력 예시

  |N|Arr|result|
  |:--:|:--:|:--:|
  |10|1000100101|2|
 
## 풀이방식

### 나의 풀이방식

  > 조건을 만족하는 경우를 탐색한다. 시간 복잡도가 n이기 때문에 효과적이다.

  초기에는 이분탐색을 사용한 풀이를 원했지만, 맞는 않는 방식이여서 단순히 시뮬레이션을 통한 풀이를 사용하였다. 아래와 같은 조건을 만족하는 지가 중점이다. 시뮬레이션으로 풀었지만 효과적이지 못한 방법이다.

  1. 앞뒤에 1이 있는지 확인한다.
      - 앞/뒤 한 곳에만 1이 있는 경우에는 "현재 0의 개수"가 최대 거리가 된다.
      - 양쪽에 있는 경우 "현재 0의 개수"가 2로 나누어졌을때에 따라 달라진다.
        - 떨어지는 경우 "현재 0의 개수 / 2", 그렇지 않은 경우 "(현재 0의 개수 / 2) + 1"이다.
  
  ```cpp
  int solution(int n,vector<int> arr){	
    int answer = 0;
    int cnt = 0;      // 현재 0의 개수
    int posCnt = 0;   // 현재 1의 개수


    for(int i=0;i<arr.size();i++){
      if(arr[i] == 1) {
        ++posCnt;

        // 연속되서 1이 나오는 경우 & 1번을 만족하는 경우
        if(posCnt == 2 && cnt == 0) posCnt = 1;
        else if(posCnt == 2 || (posCnt == 1 && cnt > 0)){
          int tmp = 0;
          
          // 1번을 만족하되 양쪽에 1이 있는 경우 & 그렇지 않은 경우
          if(posCnt == 2) {
            posCnt = 0;
            tmp = (cnt % 2 == 0) ? cnt / 2 : (cnt /2) + 1;
          }
          else {
            tmp = cnt;
            posCnt = 1;
          }
          
          answer = max(answer, tmp);
          cnt = 0;
        }
      }
      else cnt++;
    }
    if(posCnt == 1 && cnt > 0) answer = max(answer, cnt);
    return answer;
  }
  ```

### 다른이의 풀이방식

  > 배열을 순/역방향으로 진행해가면서 연속으로 이루어지는 0의 개수를 파악한 후, 두 배열의 최소값을 반환

  <img src="https://raw.githubusercontent.com/Goaway-1/goaway-1.github.io/master/_posts/images/disatanceEach.png" height="350" title="disatanceEach">

  위에서 말한 것처럼, 배열 arr에 대해서 순/역방향으로 진행해가되, 연속으로 나타나는 0의 개수를 ch배열에 저장한다. 그럼 그림과 같이 배열이 표시되는데 이 두 배열을 비교하여 최소값을 대입하면 최대한의 거리를 알 수 있다. 

  ```cpp
  int solution(int n,vector<int> arr){	
    vector<int> ch(n);

    // 순방향
    int cnt = 101;
    for(int i=0;i<arr.size();i++){
      if(arr[i] == 1){
        cnt = 0;
        ch[i] = cnt;
      }
      else ch[i] = ++cnt;
    }
    
    // 역방향
    cnt = 101;
    for(int i=n-1;i>=0;i--){
      if(arr[i] == 1) cnt = 0;
      else ch[i] = min(++cnt, ch[i]);
    }
    
    //최솟값 찾기
    int answer = *max_element(ch.begin(),ch.end());
    return answer;
  }
  ```
