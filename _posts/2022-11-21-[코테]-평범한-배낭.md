## 문제

  이 문제는 아주 평범한 배낭에 관한 문제이다.

  한 달 후면 국가의 부름을 받게 되는 준서는 여행을 가려고 한다. 세상과의 단절을 슬퍼하며 최대한 즐기기 위한 여행이기 때문에, 가지고 다닐 배낭 또한 최대한 가치 있게 싸려고 한다.

  준서가 여행에 필요하다고 생각하는 N개의 물건이 있다. 각 물건은 무게 W와 가치 V를 가지는데, 해당 물건을 배낭에 넣어서 가면 준서가 V만큼 즐길 수 있다. 아직 행군을 해본 적이 없는 준서는 최대 K만큼의 무게만을 넣을 수 있는 배낭만 들고 다닐 수 있다. 준서가 최대한 즐거운 여행을 하기 위해 배낭에 넣을 수 있는 물건들의 가치의 최댓값을 알려주자.

  한 줄에 배낭에 넣을 수 있는 물건들의 가치합의 최댓값을 출력한다.

## 조건

  - 첫 줄에 물품의 수 N(1 ≤ N ≤ 100)과 준서가 버틸 수 있는 무게 K(1 ≤ K ≤ 100,000)가 주어진다. 
  - 두 번째 줄부터 N개의 줄에 거쳐 각 물건의 무게 W(1 ≤ W ≤ 100,000)와 해당 물건의 가치 V(0 ≤ V ≤ 1,000)가 주어진다.
  - 입력으로 주어지는 모든 수는 정수이다.

## 입출력 예시
  
  |N|K|arr|result|
  |:--:|:--:|:--:|:--:|
  |4|7|[[6,13],[4,8],[3,6],[5,12]]|14|

## 풀이방식

### 틀린 풀이방식
 
  > DFS를 사용한 풀이. 타임 아웃 발생

  ```cpp
  #include <iostream>
  #include <vector>
  #include <algorithm>
  using namespace std;
  int answer = 0;

  bool opper(pair<int,int> a, pair<int,int> b) {
    if(a.first != b.first) return a.first < b.first;
    else return a.second > b.second;
  }

  void DFS(vector<pair<int, int>>& arr, const int maxWight,int idx, int wight, int val) {
    for (int i = idx; i < arr.size(); i++) {
      if (wight + arr[i].first <= maxWight) {
        answer = max(answer, (val + arr[i].second));
        DFS(arr, maxWight, i + 1, wight + arr[i].first, val + arr[i].second);
        DFS(arr, maxWight, i + 1, wight, val);
      }
      else break;
    }
  }

  int main(){
    int n, k,w,v;
    cin >> n >> k;
    vector<pair<int, int>> arr(n);

    for (int i = 0; i < n; i++) {
      cin >> w >> v;
      arr[i] = make_pair(w, v);
    }

    sort(arr.begin(), arr.end(), opper);

    DFS(arr, k, 0,0, 0);

    cout << answer;

    return 0;
  }
  ```

### 맞은 풀이방식

  > DP 풀이방식으로 이전의 값을 활용한 방식

  물품의 개수 N과 가방이 버틸 수 있는 총 무게 K를 기준으로 2차원 배열(DP)를 생성한다. 이는 N번째 물건을 넣었을때, 0 ~ K 각각의 무게에 최대 가중치를 저장한다. 즉 물건을 하나씩 넣으면서 0 ~ K 까지의 무게에 대한 최대 값을 도출한다.

  이때 점화식은 __<ins>"DP[i][j] = max(DP[i - 1][j], DP[i - 1][j - W[i]] + V[i])"</ins>__ 이다. 이전 값이 큰지, 가방의 무게를 넘지 않는 선에서 합한 가중치 값이 큰지 비교한다.


  ```cpp
  #include <iostream>
  #include <algorithm>
  using namespace std;

  int n, k;
  int DP[101][100001];
  int w[101];
  int v[101];

  int main(){
    cin >> n >> k;

    for (int i = 1; i <= n; i++) cin >> w[i] >> v[i];

    for (int i = 1; i <= n; i++){
      for (int j = 1; j <= k; j++){
        if (j - w[i] >= 0) DP[i][j] = max(DP[i - 1][j], DP[i - 1][j - w[i]] + v[i]);
        else DP[i][j] = DP[i - 1][j];
      }
    }

    cout << DP[n][k];
  }
  ```

## 개선점
  - 난 DP와 DFS의 풀이를 헷갈려한다. 가장 먼저 DP 풀이가 생각났음에도 불구하고, DFS로 풀어버리는 난... DFS의 황제?
