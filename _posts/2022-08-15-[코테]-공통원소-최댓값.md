## 문제

  N개의 원소를 갖는 같은 크기의 A, B, C 세 개의 집합이 주어집니다.

  세 집합의 공통원소 중 가장 큰 숫자를 출력하는 프로그램을 작성하세요. 세 집합의 공통원소는 반드시 존재합니다.


## 조건

  - 세 집합의 크기인 자연수 N(3<=N<=20,000)이 주어집니다.
  - A, B, C집합의 원소가 주어집니다.

## 입출력 예시

  |N|A|B|C|result|
  |:--:|:--:|:--:|:--:|:--:|
  |10|[39, 31, 32, 75, 75, 7, 89, 21, 7, 67]|[4, 19, 52, 75, 35, 100, 4, 26, 24, 69]|[82, 53, 22, 64, 58, 80, 94, 75, 51, 69]|75|
  
## 풀이방식

  > A,B,C를 내림차순으로 정렬하고, 각각의 인덱스를 비교하는 방식

  1. 최대값을 구하기 때문에 A,B,C를 내림차순으로 정렬
  2. A,B,C의 처음 인덱스부터 비교
      - 이때 값이 모두 같다면 그 값을 반환
      - 그렇지 않다면 가장 큰 값에 해당하는 집합의 인덱스를 다음 인덱스로 넘김
  3. 2번을 반복

  ```cpp
  int solution(int n,vector<int> A,vector<int> B,vector<int> C){		
    // 1번 : 내림차순 정렬
    sort(A.begin(), A.end(), greater<int>());
    sort(B.begin(), B.end(), greater<int>());
    sort(C.begin(), C.end(), greater<int>());
    
    // 2번 : A,B,C를 앞에서부터 비교
    int idx[3] = {0,0,0};
    while(idx[0] < n && idx[1] < n && idx[2] < n){
      int tmpMax = max(A[idx[0]], max(B[idx[1]], C[idx[2]]));
      
      if(tmpMax == A[idx[0]] && tmpMax == B[idx[1]] && tmpMax == C[idx[2]]) return tmpMax;
      else{
        if(tmpMax == A[idx[0]]) idx[0]++;
        if(tmpMax == B[idx[1]]) idx[1]++;
        if(tmpMax == C[idx[2]]) idx[2]++;
      }
    }
    return -1;
  }
  ```
