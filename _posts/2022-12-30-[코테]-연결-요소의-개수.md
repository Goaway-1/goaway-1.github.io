## 문제

  방향 없는 그래프가 주어졌을 때, 연결 요소 (Connected Component)의 개수를 구하는 프로그램을 작성하시오.

  [링크](https://www.acmicpc.net/problem/11724)
  
## 조건

  - 첫째 줄에 정점의 개수 N과 간선의 개수 M이 주어진다. (1 ≤ N ≤ 1,000, 0 ≤ M ≤ N×(N-1)/2) 
  - 둘째 줄부터 M개의 줄에 간선의 양 끝점 u와 v가 주어진다. (1 ≤ u, v ≤ N, u ≠ v) 
    - 같은 간선은 한 번만 주어진다.

## 입출력 예시
  
  |N|M|arr|answer|
  |:--:|:--:|:--:|:--:|
  |6|5|[[1,2],[2,5],[5,1],[3,4],[4,6]]|2|
  |6|8|[[1,2],[2,5],[5,1],[3,4],[4,6],[5,4],[2,4],[2,3]]|1|

## 풀이방식

### 1번 풀이방식

  > Union과 Find를 사용한 풀이

  예전 인프런 강의 중 "친구인가?"라는 문제가 존재했는데, 이는 Union & Find를 사용하여 문제를 해결했다. 

  본인은 문제를 보고 DFS풀이인것을 알았지만 재미를 위해서 해당 방법을 사용하여 풀이하였다. 

  1. 먼저 크기(N)에 맞게 배열을 설정하고, 각 인덱스에 맞게 값을 지정한다.
      - (arr[1] = 1, arr[10] = 10)
  2. 그 후 연결된 간선들을 합치는 과정을 반복한다.
  3. 1 ~ N까지 각 노드별로 FIND하여 그룹의 장을 찾아 반환한다.
      - 이 그룹 대장의 개수가 그룹의 개수이다!

  설명이 조금 부족했는데, 해당 [링크](https://namu.wiki/w/Union%20Find)에서 Union & Find에 대해 학습하면 좋을 것 같다.

  ```cpp
  #include <iostream>
  #include <vector>
  #include <string>
  #include <set>
  using namespace std;

  int arr[1001];

  int Find(int v) {
    if(v == arr[v]) return v;
    else return arr[v] = Find(arr[v]);
  }

  void Union(int &a, int &b) {
    a = Find(a);
    b = Find(b);
    if(a != b) arr[a] = b;
  }

  int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);	cout.tie(NULL); 

    int N = 0, M = 0;
    cin >> N >> M;
    for (int i = 1; i <= N; i++) arr[i] = i;

    // UNION & FIND
    int a = 0, b = 0;
    for (int i = 0; i < M; i++){
      cin >> a >> b;
      Union(a,b);
    }

    // Get Count
    set<int> S;
    for (int i = 1; i <= N; i++) S.insert(Find(arr[i]));

    cout << S.size();
    return 0;
  }
  ```

### 2번 풀이방식

  > DFS를 활용한 풀이

  노드를 양방향으로 연결하고, 1 ~ N 까지 DFS를 통해서 해당 노드를 체크하며 진행한다.

  1회 진행할때마다 카운트를 증가시킨다.

  ```cpp
  #include <bits/stdc++.h>
  using namespace std;

  int d, s, x, y, ret;
  vector<int> v[1001];
  int visited[1001];

  void dfs(int x){
    visited[x] = 1;
    for(auto nx : v[x]){
      if(visited[nx]) continue;
      dfs(nx);
    }

    return;
  }

  int main(){
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);	cout.tie(NULL); 
    cin >> d >> s;
      
    while(s--){
      cin >> x >> y;
      v[x].push_back(y);
      v[y].push_back(x);
    }
    
    for(int i=1; i<=d; i++){
      if(visited[i] == 0){
        ret++;
        dfs(i);
      } 
    }	
    
    cout << ret << endl;
    return 0;
  }
  ```
