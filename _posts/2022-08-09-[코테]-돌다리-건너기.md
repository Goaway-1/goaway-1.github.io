## 문제

  철수는 학교에 가는데 개울을 만났습니다. 개울은 N개의 돌로 다리를 만들어 놓았습니다. 철 수는 돌 다리를 건널 때 한 번에 한 칸 또는 두 칸씩 건너뛰면서 돌다리를 건널 수 있습니다. 각 돌은 밟기 위한 비용이 있습니다. 철수가 개울을 건너기 위해 필요한 최소비용을 구하는 프로그램을 작성하세요.

## 조건

  - 돌의 개수는 자연수 N(3≤N≤45) 사이 값을 갖는다.
  - 각 돌을 밟기 위한 비용이 철수가 있는 곳부터 차례대로 주어집니다. 비용은 100이하의 자연수입니다.

## 입출력 예시

  |N|value|result|
  |:--:|:--:|:--:|
  |7|[2,2,50,2,3,50,5]|12|
  
## 풀이방식

  > 욕심쟁이 기법을 사용해서 현재 최고의 선택을 기준으로 진행

  현재 위치로부터 갈 수 있는 "N+1, N+2" 중 더 비용이 적은 곳으로 이동 

  ```cpp
  int solution(int num, vector<int> arr){
    int answer = 0;
    
    for(int i=0;i<num;){
      if(arr[i] >= arr[i+1]){
        answer += arr[i+1];
        i+=2;
      }
      else {
        answer += arr[i];
        i++;
      }
    }
    return answer;
  }
  ```
