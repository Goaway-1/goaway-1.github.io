## 문제

  효진이는 멀리 뛰기를 연습하고 있습니다. 효진이는 한번에 1칸, 또는 2칸을 뛸 수 있습니다. 칸이 총 4개 있을 때, 효진이는
  (1칸, 1칸, 1칸, 1칸)
  (1칸, 2칸, 1칸)
  (1칸, 1칸, 2칸)
  (2칸, 1칸, 1칸)
  (2칸, 2칸)
  의 5가지 방법으로 맨 끝 칸에 도달할 수 있습니다. 멀리뛰기에 사용될 칸의 수 n이 주어질 때, 효진이가 끝에 도달하는 방법이 몇 가지인지 알아내, 여기에 1234567를 나눈 나머지를 리턴하는 함수, solution을 완성하세요. 예를 들어 4가 입력된다면, 5를 return하면 됩니다.

## 조건

  - n은 1 이상, 2000 이하인 정수입니다.

## 입출력 예시

  |n|result|
  |:--:|:--:|
  |4|5|
  |3|3|

## 풀이방식

  딱 봐도 DFS로 풀었다가는 효율성문제가 발생할 것 같아서, 1과 2의 개수에 따른 규칙을 찾아보았다.

  총 길이와 1의 개수에 따라 팩토리얼 어쩌구 저쩌구를 찾아봤지만 "n자리가 있을때 k개를 배치하는 방법의 개수"에서 막혔다.

  결국 21세기의 힘을 빌렸고, 단순히 피보나치 수열을 이용하면 쉽게 풀 수 있었다. 아래는 개고생한 사진이다.

  <img src="https://raw.githubusercontent.com/Goaway-1/goaway-1.github.io/master/_posts/images/longjump/dogwork.jpg" height="300" title="dogwork">

  __피보나치 수열은__ 알다시피 이전의 값들이 현재의 값에 영향을 미치는 것이다. 구현은 아래와 같다.

  ```cpp
  int dp[2001];

  long long solution(int n) {
    long long answer = 0;
    dp[1] = 1; 
    dp[2] = 2;

    for (int i = 3; i <= n; i++) dp[i] = (dp[i - 1] + dp[i - 2]) % 1234567;

    answer = dp[n];
    return answer;
  }
  ```

## 개선점
  - 항상 말하지만 내 머리를 개선해야된다.
  - 하나의 방법에만 몰두하지 말고 다른 방법 또한 생각해보자...
