## 문제

  정수 n, left, right가 주어집니다. 다음 과정을 거쳐서 1차원 배열을 만들고자 합니다.

  1. n행 n열 크기의 비어있는 2차원 배열을 만듭니다.
  2. i = 1, 2, 3, ..., n에 대해서, 다음 과정을 반복합니다.
      - 1행 1열부터 i행 i열까지의 영역 내의 모든 빈 칸을 숫자 i로 채웁니다.
  3. 1행, 2행, ..., n행을 잘라내어 모두 이어붙인 새로운 1차원 배열을 만듭니다.
  4. 새로운 1차원 배열을 arr이라 할 때, arr[left], arr[left+1], ..., arr[right]만 남기고 나머지는 지웁니다.

  정수 n, left, right가 매개변수로 주어집니다. 주어진 과정대로 만들어진 1차원 배열을 return 하도록 solution 함수를 완성해주세요.

## 조건

  - 1 ≤ n ≤ 107
  - 0 ≤ left ≤ right < n2
  - right - left < 105

## 입출력 예시

  |n|left|right|result|
  |:--:|:--:|:--:|:--:|
  |3|2|5|[3,2,2,3]|
  |4|7|14|[4,3,3,3,4,4,4,4]|
  
## 풀이방식

### 나의 풀이방식

  > 인덱스 번호에 맞게 풀이

  |n이 3일때|||
  |:--:|:--:|:--:|
  |(0,0)|(0,1)|(0,2)|
  |(1,0)|(1,1)|(1,2)|
  |(2,0)|(2,1)|(2,2)|

  위 표는 n이 3일때의 인덱스 번호를 표현한 것이다. (x,y)라고 할때, 둘 중 더 큰 값에 대해 "+1"하면 그 인덱스에 저장되는 숫자가 도출된다. 이 번호를 단순 push_back하여 저장한다.
   
  ```cpp
  vector<int> solution(int n, long long left, long long right) {
    vector<int> answer;
    // 각각의 좌표
    int y = left / n;     //시작 높이
    int x = left % n;     //시작 인덱스
    int maxCnt = right - left;
    int cnt = 0;
    
    for (int i = y; i < n; i++){
      for (; x < n; x++){
        int tmp = (i > x) ? i + 1 : x + 1;
        answer.push_back(tmp);
        if(++cnt > maxCnt) break;
      }
      x = 0;
      if (cnt > maxCnt) break;
    }
    return answer;
  }
  ```

### 다른이의 풀이방식

  > left의 인덱스부터 n으로 나누었을때의 좌표값을 이용
  
  보다 직관적이제 left를 right까지 진행하되, 해당 인덱스를 n으로 나누고, 나머지를 계산하여 1을 더하는 전략

  ```cpp
  vector<int> solution(int n, long long left, long long right) {
    vector<int> answer;
    for(long long i = left; i <= right; i++){
      answer.push_back(max(i / n, i % n) + 1);
    }

    return answer;
  }
  ```
