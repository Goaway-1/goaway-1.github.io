## 문제

  눈금의 간격이 1인 M×N(M,N≤100)크기의 모눈종이가 있다. 이 모눈종이 위에 눈금에 맞추어 K개의 직사각형을 그릴 때, 이들 K개의 직사각형의 내부를 제외한 나머지 부분이 몇 개의 분리된 영역으로 나누어진다.

  예를 들어 M=5, N=7 인 모눈종이 위에 <그림 1>과 같이 직사각형 3개를 그렸다면, 그 나머지 영역은 <그림 2>와 같이 3개의 분리된 영역으로 나누어지게 된다.

  ![그림1과2](https://www.acmicpc.net/upload/images/zzJD2aQyF5Rm4IlOt.png)


  <그림 2>와 같이 분리된 세 영역의 넓이는 각각 1, 7, 13이 된다.

  M, N과 K 그리고 K개의 직사각형의 좌표가 주어질 때, K개의 직사각형 내부를 제외한 나머지 부분이 몇 개의 분리된 영역으로 나누어지는지, 그리고 분리된 각 영역의 넓이가 얼마인지를 구하여 이를 출력하는 프로그램을 작성하시오.

  [링크](https://www.acmicpc.net/problem/2583)
  
## 조건

  - 첫째 줄에 M과 N, 그리고 K가 빈칸을 사이에 두고 차례로 주어진다. M, N, K는 모두 100 이하의 자연수이다. 
  - 둘째 줄부터 K개의 줄에는 한 줄에 하나씩 직사각형의 왼쪽 아래 꼭짓점의 x, y좌표값과 오른쪽 위 꼭짓점의 x, y좌표값이 빈칸을 사이에 두고 차례로 주어진다. 
    - 모눈종이의 왼쪽 아래 꼭짓점의 좌표는 (0,0)이고, 오른쪽 위 꼭짓점의 좌표는(N,M)이다. 입력되는 K개의 직사각형들이 모눈종이 전체를 채우는 경우는 없다.

## 입출력 예시
  
  |M|N|K|Arr|result|
  |:--:|:--:|:--:|:--:|:--:|
  |5|7|3|[[0,2,4,4],[1,1,2,5],[4,0,6,2]]|3, [1, 7, 13]|

## 풀이방식

  > 직사각형 영역을 표시하고, 나머지 영역에 대해 BFS를 진행

  ```cpp
  #include <iostream>
  #include <vector>
  #include <string>
  #include <algorithm>
  #include <queue>

  using namespace std;

  int x[] = {0, 1, 0, -1};
  int y[] = {1, 0, -1, 0};
  int arr[100][100];

  // 직사각형 영역 1로 처리
  void CheckArea(int ltX, int ltY, int rtX, int rtY) {
    for (int i = ltY; i< rtY; i++) for (int j = ltX; j < rtX; j++) arr[i][j] = 1;
  }

  int main(){
    int M = 0, N = 0, K = 0;
    int answer = 0;
    vector<int> ans;

    cin >> M >> N >> K;

    // 직사각형 영역 처리
    int ltX, ltY, rtX, rtY;
    for (int i = 0; i < K; i++){
      cin >> ltX >> ltY >> rtX >> rtY;
      CheckArea(ltX,ltY, rtX,rtY);
    }

    // BFS를 통한 공간 확인
    for (int i = 0; i < M; i++) {
      for (int j = 0; j < N; j++) {
        if (arr[i][j] == 0) {
          int cnt = 0;
          int tmpX = 0, tmpY = 0, tmpXX = 0, tmpYY = 0;
          queue<pair<int,int>> Q;

          Q.push(make_pair(i,j));
          while (!Q.empty()) {
            tmpX = Q.front().first;
            tmpY = Q.front().second;
            Q.pop();

            for (int i = 0; i < 4; i++){
              tmpXX = tmpX + x[i];
              tmpYY = tmpY + y[i];
              if (tmpXX >= 0 && tmpXX < M && tmpYY >= 0 && tmpYY < N && arr[tmpXX][tmpYY] == 0) {
                Q.push(make_pair(tmpXX, tmpYY));
                arr[tmpXX][tmpYY] = 1;
                cnt++;
              }
            }
          }
          if(cnt == 0) ans.push_back(1);
          else ans.push_back(cnt);
          answer++;
        }
      }
    }
    sort(ans.begin(),ans.end());
    cout << answer << "\n";
    for (int i = 0; i < ans.size(); i++) cout << ans[i] << " ";
    return 0;
  }
  ```
