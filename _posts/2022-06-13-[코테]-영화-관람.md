## 문제

  현수네 반 N명의 학생들은 영화를 보기 위해 학교 강 당에 모였습니다. 강당의 좌석은 영화관처럼 계단형이 아니라 평평한 바닥에 의자가 일렬로 놓여진 상태입니다. 앉는 순서는 제일 앞자리부터 반 번호순(1번 부터)으로 앉습니다. 이때 만약 앞자리에 앉은 키가 큰 학생이 앉으면 그 학생보 다 앉은키가 작은 뒷자리 학생은 스크린이 보이지 않습니다. N명의 학생들의 앉은 키 정보가 주어지면 각 학생들의 최초 시야를 가려 영화관람을 불가능하게 하는 학생들을 찾고 싶습니다. 예를 들어 N=5이고 다음과 같이 1번 학생부터 앞자리에 차례대로 앉았다면 50  57  52  53  511번 학생(50)은 제일 앞자리이므로 최초 시청방해 학생이 없습니다. 2번 학생(57)은 시청 방해학생이 없습니다. 3번 학생(52)의 최초 시청 방해학생은 2번 학생입니다. 4번 학생(53)의 최초 시청 방해학생은 2번 학생입니다. 5번 학생(51)의 최초 시청 방해학생은 4번 학생입니다. N명의 앉은 키 정보가 주어지면 각 학생들의 최초 시청을 방해하는 학생들의 번호를 출력하는 프로그램을 장성하세요.

## 조건

  - 첫 줄에 한 줄에 앉은 학생수 N(3<=N<=100,000)이 주어진다.
  - 두 번째 줄에 N명의 앉은 키 정보(1부터 100,000이하)가 1번 학생부터 반 번호순으로 주어진다.
  - 첫 번째 줄에 1번 학생부터 N번 학생 순으로 자기 자신을 최초로 시청방해하는 학생의 번호를 출력한다. 시청방해 학생이 없을 경우 0을 출력한다.

## 풀이방식

  ### 1번 풀이방식

  - 맨 뒷자리에서 앞자리로 이동하며, 자신보다 큰 학생의 위치를 찾아 기록하는 방식. __(시간초과 발생)__

    ```cpp
    #include <iostream>
    #include <vector>

    int main(){
      freopen("input.txt", "rt", stdin);
      int n;
      cin >> n;
      vector<int> arr(n+1,0);
      vector<int> answer(n+1,0);
      
      // input
      for(int i = 1;i <= n;i++) cin >> arr[i];
      
      // Calculation
      for(int i = n;i >= 0;i--){
        int tmp = 0;
        for(int j = i - 1;j >= 0;j--){
          if(arr[i] < arr[j]){
            tmp = j;
            break;
          }
        }
        answer[i] = tmp;
      }
      
      for(int i = 1;i <= n;i++) cout << answer[i] << " ";
      
      return 0;
    }
    ```

  ### 2번 풀이방식
  
  - 본인이 어떤 학생의 방해학생인지를 확인하는 방법으로 __스택을__ 활용
  - 맨 뒤의 학생부터 앞의 학생으로 지나감
    - 스택이 비어있다면 학생을 넣고 다음 학생으로 넘김
    - 스택이 비어있지 않고
      1. 스택의 상단보다 학생의 키가 작다면 삽입
      2. 스택의 상단보다 학생의 키가 크다면 스택의 내용과 비교하여 큰 학생이 나올때까지 pop하고, answer에 갱신

    ```cpp
    #include <iostream>
    #include <vector>
    #include <stack>

    int main(){
      freopen("input.txt", "rt", stdin);
      int n;
      cin >> n;
      vector<int> arr(n+1,0), answer(n+1,0);
      stack<int> s;
      
      // input
      for(int i = 1;i <= n;i++) cin >> arr[i];
      
      // Calculation
      for(int i = n;i >= 1;i--){
        while(!s.empty() && arr[i] > arr[s.top()]){
          answer[s.top()] = i;
          s.pop();
        }
        s.push(i);
      }
      
      for(int i = 1;i <= n;i++) cout << answer[i] << " ";
      
      return 0;
    }
    ```

## 개선점
  - 스택, 큐의 특성을 잘 활용하자
